# 03 · 转化漏斗：从公域流量到付费学员

> 涨粉不等于赚钱。这一篇解决最关键的一步：**怎么把小红书上点赞收藏的陌生人，变成加你微信、信任你、最后付费的学员。**

---

## 一、完整转化漏斗（每一层都要设计）

```
① 看到笔记（公域曝光）         ← 02内容引擎负责
② 点赞/收藏/关注（产生兴趣）    ← 封面+干货质量
③ 加微信（进入私域）⭐关键跳转   ← 钩子设计（本篇核心）
④ 朋友圈/对话养信任            ← 持续价值+案例
⑤ 免费诊断/试听（体验专业）⭐     ← 诊断报告展示方法
⑥ 付费成单                    ← 成交话术
⑦ 出效果→转介绍               ← 飞轮（见04）
```

> 漏斗每层都会流失，所以**每层都要有钩子往下拉**。重点设计③加微信和⑤诊断转化。

## 二、合规引流：怎么把人从小红书引到微信

⚠️ 小红书严禁直接留微信号/二维码，轻则限流重则封号。用这些**安全做法**：

| 方法 | 怎么做 | 安全度 |
|------|--------|:---:|
| **资料包钩子** | 笔记结尾："整理了【AP生物高频考点图谱】，需要的扣1/评论'资料'" → 私信引导 | 高 |
| **私信承接** | 用户私信你 → 你回"主页简介有联系方式"或引导至公众号 | 高 |
| **公众号中转** | 简介/笔记引导关注公众号 → 公众号自动回复给微信/资料 | 高 |
| **评论区暗示** | "更多在我的同名公众号" | 中 |
| **谐音/拆字**（少用） | 微信号用文字描述 | 低（不推荐）|

**最稳路径**：**小红书 → 公众号 → 微信**。公众号是大陆唯一能合规承接+自动化的私域枢纽。

### 资料包钩子（最强引流诱饵）
你有 100 课弹药，随手就能做出高价值免费资料：
- 《AP生物8大单元高频考点图谱》
- 《AP生物7大统计公式+例题速查卡》
- 《FRQ六大题型答题模板》
- 《AP生物高频易错点TOP30》

> 用免费资料换"关注公众号/加微信"——用户拿到真东西，信任度立刻建立。

## 四、微信私域运营：把信任养厚

加到微信后，**别急着推销**。先做三件事养信任：

### 1. 朋友圈人设（每天 1–2 条，别刷屏）
| 内容类型 | 占比 | 示例 |
|----------|:---:|------|
| 专业输出 | 40% | "今天给学生讲FRQ评分点"（配教师版截图）|
| 学员日常/进步 | 25% | "学生这次月考生物92，FRQ满分"（配对比，隐私打码）|
| 教学理念/态度 | 20% | 3C方法、为什么这样教 |
| 真人温度 | 15% | 生活、读书、对教育的看法 |

> 朋友圈 = 你的"无声销售员"。家长加了你会先翻你朋友圈，**前 30 条决定第一印象**。

### 2. 一对一答疑（轻互动，重价值）
- 家长/学生问考点，认真答 → 直接展示专业
- 主动提供"发我看看孩子的卷子，我帮你看看问题在哪"

## 五、免费诊断：成交前的"专业体验"⭐

这是你最该用的转化抓手，因为它**直接展示你的方法论和体系**。

### 诊断流程
```
1. 家长发来孩子的成绩/一份FRQ答卷/最近卷子
2. 你用产品里的 rubric + 易错诊所，做一份《诊断报告》
3. 报告里指出：①薄弱单元 ②失分类型 ③提分路径
4. 给出"如果跟我学，我会怎么帮他"的方案
5. 自然过渡到试听/报名
```

### 《诊断报告》模板（半页，专业感拉满）
```
【XX同学 AP生物诊断报告】墨轩国际教育

一、当前定位
- 估分：约3.5分 / 薄弱单元：U3能量、U6基因表达
二、主要失分类型（按你的错题本分类法）
- FRQ：审题动词混淆（Explain答成Describe）→ 丢机制分
- 计算：水势/HW公式不熟 → 计算题失分
三、提分路径（3C穿透法）
- 第一步：U3/U6概念重建（C1）
- 第二步：FRQ六大题型模板训练（C3）
- 预计：6–8周可冲4.5–5分
四、建议
- 1对1 / 小班，每周X次，先从U3开始
```

> 一份这样的报告发过去，家长立刻感受到"这老师有体系、能看准病根"——**比任何广告都管用**。

## 六、成交话术（临门一脚）

### 处理常见疑虑

| 家长疑虑 | 回应要点 |
|----------|----------|
| "效果怎么样？" | 给案例+诊断报告，"我先帮孩子做个诊断，看准了再决定" |
| "多少钱？" | 先讲价值/方案再报价；按阶梯报（见04），给"先试听"台阶 |
| "再考虑下" | 不逼单，"理解，资料先用着，孩子卷子随时发我看"，持续养 |
| "别家更便宜" | 强调体系+方法+负责，"便宜的是课时，值钱的是提分确定性" |
| "能保5分吗？" | ❌绝不承诺保分（违规且失信）→ "保不了分，但我能保证方法和体系，努力到位结果不会差" |

### 试听课设计（成交转化率最高的一环）
- 用一节**真实正课**（如 U3.L05 光合或 FRQ 模板课）试听
- 试听中就用 3C 走一遍 → 让学生当场"听懂了之前不懂的"
- 试听后给"学习计划单"（M0.1 里的个性化路径）→ 顺势报名

## 七、转化漏斗数据追踪表

| 指标 | 起步目标 | 怎么算 |
|------|:---:|------|
| 笔记 → 加微信率 | 每千阅读加 3–5 人 | 加微信数 / 阅读量 |
| 加微信 → 诊断率 | 30%+ | 做诊断数 / 加微信数 |
| 诊断 → 试听率 | 50%+ | 试听数 / 诊断数 |
| 试听 → 成单率 | 40%+ | 成单数 / 试听数 |

> 哪一层转化低就重点优化哪层：加微信低→改钩子；诊断少→主动提供；试听不成单→优化试听体验和话术。

## 八、合规与口碑红线

- ❌ 不承诺保分/保 offer（虚假宣传，违规）
- ❌ 不伪造学员案例/截图（一旦穿帮 IP 崩塌）
- ❌ 不诋毁同行/机构
- ❌ 不在小红书硬留微信（封号）
- ✅ 真实案例需经学员/家长同意，隐私打码
- ✅ 先给价值、再谈成交——信任是慢功夫
