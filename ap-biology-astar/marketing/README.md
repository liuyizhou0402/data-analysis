## 一、先想清楚：你卖的到底是什么

你不是在卖"补习课时"，你卖的是 **"AP 生物从模棱两可到稳上 5 分/A★ 的确定性"**。

家长和学生真正焦虑的是：
- 孩子刷了题还是 3、4 分上不去 → **怕努力白费**
- FRQ 总丢分，不知道问题在哪 → **怕找不到病根**
- 机构老师换来换去、不负责 → **怕所托非人**

你的 100 课时产品 + 3C 方法论，正好对应"**有体系、有方法、能拆解病根、对齐官方考纲**"——这就是你内容里要反复传递的核心信任状。

> 🎯 **一句话定位**："专做 AP/A-Level 生物冲顶分的老师，用 3C 穿透法把生物讲成一张网、把考试拆成一套招。"

## 二、获客主线（零预算内容打法）

```
   小红书(主战场·拉新)
   ├─ 发"提分干货/易错点/答题模板"内容
   └─ 用你的100课产品做弹药库
        ↓ 引流
   微信私域(朋友圈+1对1·养信任)
   ├─ 持续输出+学员案例+答疑
        ↓ 转化
   免费诊断/试听课(成交临门一脚)
   └─ 用"诊断报告"展示专业度
        ↓ 成单
   种子学员(1对1/小班)
   └─ 做出口碑案例 → 反哺内容 → 转介绍
        ↺ 飞轮转起来
```

**为什么是小红书为主？**
中国大陆国际学校的**家长（尤其妈妈）和高中生本人**高度集中在小红书；搜索"AP生物 5分""AP bio FRQ""IGCSE 生物"等都有明确需求人群；零预算、纯内容也能起号。公众号做深度沉淀和转化承接，视频号/B站可作二次分发。

## 四、90 天冷启动作战地图

### 🟢 第 1 阶段（第 1–4 周）：搭台子 + 攒弹药
- 定位、起名、装修小红书账号（见 `01-xiaohongshu-cold-start.md`）
- 用 100 课产品提炼 **首批 30 条选题**（见 `02-content-engine.md`）
- 跑通"图文笔记"生产流程，先发 **10–15 条**测试
- 搭好私域入口（微信号、欢迎话术、资料包钩子）

**目标**：账号定位清晰，发够 15 条，跑出 1–2 条小爆款，加到第一批 5–10 个微信好友。

### 🟡 第 2 阶段（第 5–8 周）：起量 + 找种子学员
- 稳定日更/隔日更，主攻"易错点+答题模板+提分技巧"
- 上线**免费诊断**钩子（"发你孩子的 FRQ，我给一份诊断报告"）
- 用**种子学员策略**（首批限量优惠换案例，见 `04-seed-and-pricing.md`）拿下前 2–3 个学员
- 私域开始发学员进步/答疑日常

**目标**：粉丝过 500–1000，谈下 2–3 个种子学员，攒下第一批真实案例。

### 🔴 第 3 阶段（第 9–12 周）：转化 + 口碑飞轮
- 把种子学员的进步做成**案例笔记**（提分故事、FRQ 前后对比）
- 优化转化漏斗+话术（见 `03-conversion-funnel.md`）
- 推出正式定价与产品阶梯（1 对 1 / 小班 / 资料包）
- 启动**转介绍机制**（老带新）

**目标**：稳定接单、有真实口碑、内容-成交飞轮转起来，月收入开始爬坡。

## 五、核心原则（贴墙上）

1. **内容即资产**：你已有 100 课弹药库，别人抄不走你的体系——这是最深的护城河。
2. **真人 IP > 机构感**：家长信"一个负责的真老师"，多露专业、露态度、露温度。
3. **先给后取**：先免费给价值（干货、诊断、资料），信任够了成交是水到渠成。
4. **案例是王炸**：一个真实的"4→5 分"案例，胜过一百句自夸。
5. **慢就是快**：零预算靠复利，坚持 90 天发 60+ 条，账号会自己滚起来。
6. **合规第一**：只讲方法和原创内容，不承诺保分、不蹭官方真题版权、不碰虚假宣传。
