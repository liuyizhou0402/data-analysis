## 一、信号通路图题套路（模板 E）

**看到通路图，三步走**：
1. **读箭头**：→ 激活，⊣ 抑制
2. **定位**：被问的分子在通路哪一步、是激活还是被激活
3. **预测扰动**：某步被阻断/突变 → 顺箭头推下游 → 用 "because" 写因果

**必踩关键词**：ligand、receptor、conformational change、phosphorylation、second messenger、amplification、transcription。

---

## 二、反馈题套路（模板 D · CER）

- **Claim**：是正反馈还是负反馈
- **Evidence**：引用题干中"输出如何影响刺激"
- **Reasoning**：负反馈 = 输出**抵消**刺激→回设定点；正反馈 = 输出**加强**刺激→放大

## 三、专训题集 / Practice Set

**Q1.** *[Analyze Model｜4 分]*

Pathway: **Hormone → Receptor → Relay protein A → Relay protein B → Response enzyme**

(a) **Explain** how this pathway can amplify a small hormone signal into a large cellular response. *(2)*
(b) **Predict** the effect on the Response enzyme if Relay protein B is inhibited by a drug. *(1)*
(c) **Predict** the effect if a mutation makes Relay protein A constitutively (permanently) active. *(1)*

---

**Q2.** *[Conceptual Analysis — Feedback｜4 分]*

When body temperature rises above the set point, the hypothalamus triggers sweating and vasodilation, which cool the body back toward the set point.

(a) **Identify** the type of feedback described and **justify** your answer. *(1)*
(b) **Explain** how this mechanism maintains homeostasis. *(2)*
(c) **Contrast** this with what would happen in a positive feedback loop responding to the same temperature increase. *(1)*

---

**Q3.** *[Analyze Model — Disease｜4 分]*

Cholera toxin modifies a G protein so that it remains permanently active, continuously stimulating adenylyl cyclase to produce cAMP. High cAMP in intestinal cells causes massive water loss (diarrhea).

(a) **Explain** how the toxin's effect on the G protein leads to continuously high cAMP levels. *(2)*
(b) **Predict** what would happen to cAMP levels if a drug inhibited adenylyl cyclase in these cells. *(1)*
(c) **Explain** why this pathway normally would be turned off, and how the toxin prevents that. *(1)*

## 四、自评清单 / Self-Check

做完对照 `U4.L10-...-teacher` 或下方要点，检查是否：
- [ ] 通路题：是否说清"激活/抑制"方向 + 用 because 连因果
- [ ] 放大题：是否说"每一步激活多个下游分子"
- [ ] 反馈题：是否明确"抵消 vs 放大" + 引用题干证据
- [ ] 是否踩中关键术语

> **答案要点（简）**：
> Q1(a) 级联每步激活多个下游 → 指数放大；(b) Response 酶不被激活；(c) 即使无激素，B 和 Response 也被激活。
> Q2(a) 负反馈，因为响应抵消了温度升高；(b) 偏离设定点→触发响应→回到设定点；(c) 正反馈会让体温越升越高（危险）。
> Q3(a) G 蛋白卡在激活态→持续激活腺苷酸环化酶→cAMP 持续高；(b) cAMP 下降；(c) 正常 G 蛋白会自动水解 GTP 失活，毒素阻止了这一关闭。
