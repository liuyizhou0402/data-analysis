## 一、什么是操纵子 / What Is an Operon?

**Operon（操纵子）**：一组功能相关的基因，共用一个启动子（promoter），被统一调控，同时转录为一条 mRNA。

**组成元件（必须能识别）**：

| 元件 | 作用 |
|------|------|
| **Promoter 启动子** | RNA 聚合酶结合位点，转录从此开始 |
| **Operator 操纵基因** | 阻遏蛋白的结合位点，位于启动子和结构基因之间 |
| **Structural genes 结构基因** | 编码功能蛋白（酶）的基因，连续排列 |
| **Regulatory gene 调控基因** | 编码**阻遏蛋白（repressor）**，通常不在操纵子内 |
| **Repressor 阻遏蛋白** | 结合 operator → 阻止 RNA 聚合酶通过 → 基因关闭 |
| **Inducer / Corepressor** | 小分子，改变阻遏蛋白构象，决定其能否与 operator 结合 |

## 二、lac 操纵子（诱导型 / Inducible Operon）

> 代谢**分解**底物时用到的调控系统——底物来了才开。

**背景**：*E. coli* 携带可分解乳糖（lactose）的酶基因，但只有在乳糖存在时才需要表达。

### 无乳糖时（lac operon **关闭**）

1. 调控基因持续产生**活性阻遏蛋白（active repressor）**。
2. 阻遏蛋白结合 **operator**。
3. RNA 聚合酶被阻断，**无法通过 operator** → 结构基因**不转录**。
4. 细菌不合成乳糖酶 → **节约资源**。

### 有乳糖时（lac operon **开启**）

1. 乳糖（经异构化）产生**诱导剂（inducer）= 别乳糖（allolactose）**。
2. 诱导剂与阻遏蛋白结合 → **改变阻遏蛋白构象** → 阻遏蛋白**无法**结合 operator（失活）。
3. RNA 聚合酶自由通过 → 结构基因**转录** → 合成乳糖酶。
4. 乳糖被分解完 → 诱导剂消失 → 阻遏蛋白恢复活性 → 操纵子自动关闭。

> 💡 **逻辑**：底物（乳糖）= 诱导剂 = "钥匙"，打开基因的开关。底物来了就开，底物没了就关。

## 三、trp 操纵子（阻遏型 / Repressible Operon）

> 代谢**合成**产物时用到的调控系统——产物过多时关掉合成。

**背景**：*E. coli* 能自己合成色氨酸（tryptophan），但当外界色氨酸充足时，没必要再合成。

### 色氨酸稀少时（trp operon **开启**）

1. 调控基因产生**无活性阻遏蛋白（aporepressor）**。
2. 无活性阻遏蛋白**不能**结合 operator。
3. RNA 聚合酶自由转录 → 合成色氨酸合成酶 → 细菌自己生产色氨酸。

### 色氨酸充足时（trp operon **关闭**）

1. 色氨酸作为**辅阻遏物（corepressor）**。
2. 色氨酸 + 无活性阻遏蛋白 → **活性阻遏蛋白复合物（active repressor complex）**。
3. 活性阻遏蛋白结合 operator → 阻断转录。
4. 细菌停止合成色氨酸 → **负反馈调节，避免过度合成**。

> 💡 **逻辑**：产物（色氨酸）= 辅阻遏物 = "刹车"，产物积累了就踩刹车。

## 四、lac vs trp 对比表 / Comparison

| | **lac operon（诱导型）** | **trp operon（阻遏型）** |
|---|---|---|
| **类型** | Inducible（诱导开启） | Repressible（阻遏关闭） |
| **默认状态** | 关闭（repressor 有活性） | 开启（repressor 无活性） |
| **调控小分子** | **诱导剂** allolactose（使 repressor 失活） | **辅阻遏物** tryptophan（使 repressor 激活） |
| **小分子 + repressor** | Repressor 失活 → 基因**开** | Repressor 激活 → 基因**关** |
| **生物学逻辑** | 有底物 → 开启分解酶 | 产物过剩 → 关闭合成酶 |
| **属于** | 分解代谢（catabolism） | 合成代谢（anabolism） |

> ⚠️ **高频混淆点**：两者的 repressor 在默认状态下的"活性"是**相反**的！
> - lac：默认活性（有活性）→ 基因默认关
> - trp：默认无活性（aporepressor）→ 基因默认开

## 五、突变情境推理 / Mutation Analysis（FRQ 进阶题型）

> AP 考试最喜欢问"如果 X 元件突变了，会怎样？"这考的是对机制的**真理解**。

| 突变元件 | 效应 | 结果 |
|----------|------|------|
| **Operator 突变**（阻遏蛋白无法结合） | 阻遏蛋白永远无法结合 | 基因**持续转录**（constitutive expression），不受调控 |
| **Regulatory gene 突变**（不产生阻遏蛋白） | 没有阻遏蛋白 | 基因**持续转录**，不受调控 |
| **Repressor 突变**（无法结合 inducer/corepressor） | lac：repressor 永远有活性 → 基因**永远关闭**；trp：同理 | 反向于上面两种 |

## 七、考点提示 / Exam Tips

- 🎯 **Short FRQ（Q4 Conceptual Analysis）高频题型**：给一个情境（有/无乳糖、有/无色氨酸），让你预测基因是否转录 + 解释机制。
- 🎯 **突变题**：给操纵子某元件突变，预测在不同条件下的基因表达状态。
- 🧰 **答题模板 D（CER）**：
  - Claim：基因是否转录（开/关）
  - Evidence：哪种小分子存在/不存在
  - Reasoning：小分子如何改变阻遏蛋白构象 → 是否结合 operator → 是否阻断 RNA 聚合酶
- ⚠️ **必踩关键词**：repressor、operator、RNA polymerase、conformational change（构象改变）。
