# U3.L12 · Photosynthesis ↔ Respiration Integration 光合与呼吸整合（学生版）

> **本课你将学会 / By the end you can:**
> ① 解释光合作用与细胞呼吸如何构成物质能量闭环
> ② 对比两大过程的位置、输入输出、能量流向
> ③ 理解分子适应度（molecular fitness）的概念

---

## 📌 一句话本质 / In One Sentence
> **光合作用和细胞呼吸是一对"互逆搭档"——光合的产物（葡萄糖+O₂）正是呼吸的原料，呼吸的产物（CO₂+H₂O）正是光合的原料，能量在二者间循环流动。**

---

## 一、物质能量闭环 / The Cycle ⭐

```
        光能
         ↓
   ┌─── 光合作用 ───┐
   │  6CO₂ + 6H₂O   │ →  C₆H₁₂O₆ + 6O₂
   │  （叶绿体）     │
   └────────────────┘
         ↑↓（物质互为原料）
   ┌─── 细胞呼吸 ───┐
   │ C₆H₁₂O₆ + 6O₂  │ → 6CO₂ + 6H₂O + ATP
   │ （线粒体）      │
   └────────────────┘
         ↓
       ATP（供生命活动）+ 热能（散失）
```

- **物质**：在两过程间循环（CO₂、H₂O、O₂、葡萄糖反复利用）
- **能量**：单向流动——光能 → 化学能（葡萄糖）→ ATP → 最终以**热能**散失（第二定律，接 L04）

> ⚠️ **关键区分**：物质是**循环的**，能量是**单向流动、不循环**的（需持续输入光能）。这是 U8 生态系统能量流动的基础（接 U8.L02）。

---

## 二、两大过程全面对比 / Full Comparison ⭐

| | **光合作用** | **细胞呼吸** |
|---|---|---|
| 场所 | 叶绿体 | 线粒体 |
| 总反应 | 6CO₂+6H₂O→C₆H₁₂O₆+6O₂ | C₆H₁₂O₆+6O₂→6CO₂+6H₂O |
| 能量 | **储存**能量（吸能）| **释放**能量（放能）|
| 谁做 | 植物、藻类、蓝细菌 | 几乎所有生物 |
| 电子载体 | NADPH | NADH、FADH₂ |
| 化学渗透位置 | 类囊体膜 | 线粒体内膜 |
| 碳的去向 | 固定 CO₂ → 葡萄糖 | 葡萄糖 → 释放 CO₂ |

> 🔁 **化学渗透的共性**：两者都用"质子梯度 + ATP 合酶"产 ATP（机制相同，场所不同）——这是 C2 串联的高频考点。

---

## 三、相同的机制主题 / Shared Themes

1. **电子传递链 + 化学渗透**：光合（类囊体）和呼吸（内膜）都用
2. **氧化还原反应**：电子的转移贯穿两过程
3. **膜折叠增表面积**：类囊体 vs 嵴（接 U2.L01/L03）
4. **内共生起源**：叶绿体和线粒体都源于原核生物（接 U2.L09）

---

## 四、分子适应度 / Molecular Fitness

- **适应度（fitness）**在分子层面体现：高效的代谢酶/通路让生物获得更多能量 → 更可能存活繁殖
- 例：C4/CAM 植物的固碳适应（L07）提高了在干旱环境的适应度
- 代谢通路的进化保守性（如糖酵解几乎所有生物都有）反映其古老与重要（接 U7）

## 六、考点提示 / Exam Tips

- 🎯 **必考**：物质循环 vs 能量单向流动（别说"能量循环"）。
- 🎯 光合与呼吸的输入输出互为原料（追踪 CO₂/O₂/葡萄糖/H₂O）。
- 🎯 两者化学渗透机制相同（对比题）。
- ⚠️ 植物**同时**进行光合和呼吸！白天光合 > 呼吸（净放 O₂），夜间只呼吸。
