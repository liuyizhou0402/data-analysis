# Unit 3 · Cellular Energetics 细胞能量学

> 权重 12–16% ｜ 13 课时 ｜ CED 主题 3.1–3.7 ｜**三座大山之一**
> 大概念主线：**能量与物质——生命如何捕获、转化、释放能量（Energy & Matter）**

---

## 单元定位

Unit 3 是 AP 生物**最重、最难、FRQ 出现率最高**的单元之一。两大核心过程：
1. **光合作用（Photosynthesis）**：光能 → 葡萄糖（储能）
2. **细胞呼吸（Cellular Respiration）**：葡萄糖 → ATP（用能）

加上前面的**酶**（催化所有代谢反应）和 **ATP**（能量货币），构成完整的能量图景。

> 🎯 **本单元 FRQ 命脉**：
> - 酶动力学实验设计与数据分析（L03，已建）
> - 光合限制因子实验（L07）
> - 光合 ↔ 呼吸的物质能量追踪（L12）
> - ETC 与化学渗透机制（L10，最难）

## 本单元学习地图

```
        酶（催化一切代谢）+ ATP（能量货币）
                    |
      ┌─────────────┴──────────────┐
      ▼                            ▼
 光合作用（储能）              细胞呼吸（用能）
 光反应（类囊体）              糖酵解（细胞质）
   光→ATP+NADPH                葡萄糖→丙酮酸
 卡尔文循环（基质）            柠檬酸循环（线粒体基质）
   CO₂→G3P→葡萄糖              →NADH+FADH₂+CO₂
                              ETC+化学渗透（内膜）
                                →大量ATP
      └────────── 物质能量闭环 ──────────┘
       光合产物O₂/葡萄糖 = 呼吸的原料
       呼吸产物CO₂/H₂O = 光合的原料
```
