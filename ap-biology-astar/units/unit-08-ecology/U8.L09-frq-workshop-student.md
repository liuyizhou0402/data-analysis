## 📌 本课打法
> Unit 8 的 FRQ 多为**计算 + 图表分析**。三大公式必须熟：dN/dt、10% 法则、Simpson D。本课集中训练。

---

## 一、三大公式速查

| 主题 | 公式 |
|------|------|
| 指数增长 | dN/dt = r_max · N |
| 逻辑斯蒂增长 | dN/dt = r_max · N · (K−N)/K |
| 能量传递 | 下一级 ≈ 上一级 × 10% |
| Simpson 多样性 | D = 1 − Σ(n/N)² |

> 所有公式考试提供，重在**会用 + 写过程**。

---

## 二、图表题套路

- **增长曲线**：J 型(指数) vs S 型(逻辑斯蒂)；找 K（平台）、K/2（增长最快）
- **能量金字塔**：每级 ×10%；算总能量损失
- **描述趋势**：先说趋势 + 引数字（"随 X 增加，Y 从 a 升到 b"）

## 三、专训题集 / Practice Set

**Q1.** *[Population Growth｜4 分]*

A population of fish has r_max = 0.4 and carrying capacity K = 1500.

(a) When N = 300, **calculate** dN/dt using the logistic equation. Show work. *(2)*
(b) **Explain** what happens to dN/dt when N reaches 1500. *(1)*
(c) **Describe** how the curve would differ under exponential (unlimited) growth. *(1)*

---

**Q2.** *[Energy Flow｜4 分]*

A food chain: algae (15,000 kcal) → zooplankton → small fish → large fish.

(a) Using the 10% rule, **calculate** the energy available to the small fish. Show work. *(2)*
(b) **Explain** where most of the energy goes between trophic levels. *(1)*
(c) **Explain** why this ecosystem can support more algae (by mass) than large fish. *(1)*

---

**Q3.** *[Biodiversity｜4 分]*

A community has 3 species: A = 50, B = 30, C = 20 (total = 100).

(a) **Calculate** Simpson's diversity index D = 1 − Σ(n/N)². Show work. *(2)*
(b) **Predict** whether D would increase or decrease if the community changed to A = 90, B = 5, C = 5, and **explain**. *(1)*
(c) **Explain** how high diversity contributes to ecosystem stability. *(1)*

---

## 四、自评要点（简答）

> **Q1**: (a) dN/dt = 0.4 × 300 × (1500−300)/1500 = 0.4 × 300 × 0.8 = **96**；(b) dN/dt→0，种群稳定在 K；(c) 指数=J 型，无上限持续加速。
> **Q2**: (a) zooplankton = 1500，small fish = **150 kcal**（15000×0.1×0.1）；(b) ~90% 经呼吸以热散失 + 未摄食/未消化；(c) 能量逐级损失，低营养级可用能量多→可支撑更大生物量。
> **Q3**: (a) (50/100)²+(30/100)²+(20/100)² = 0.25+0.09+0.04 = 0.38；D = 1−0.38 = **0.62**；(b) **下降**，因为变得极不均匀（A 占绝大多数）；(c) 功能冗余→部分物种丧失时其他物种可补偿→更易恢复。
