## 单元定位

Unit 7 是 AP 生物**权重最高**的单元，也是把前面所有单元串起来的"大主线"——进化解释了生命的多样性与统一性。

> 🎯 **本单元 FRQ 命脉**：
> - **Hardy-Weinberg 计算**（统计核心，L05–L06）
> - **系统发育树 cladogram** 读/建（L09）
> - 自然选择机制论证、进化证据（L02、L07）

## 本单元学习地图

```
   变异（来源：突变 U6 + 有性重组 U5）
        ↓
   自然选择（达尔文）
   ├ 三要素：变异 + 遗传 + 差异繁殖
   ├ 适合度 fitness
   └ 选择类型：定向/稳定/分裂
        ↓
   群体遗传 + Hardy-Weinberg
   ├ 等位基因频率变化 = 进化
   ├ p+q=1; p²+2pq+q²=1
   └ 偏离 HW = 进化在发生
        ↓
   宏观进化
   ├ 证据（化石/解剖/分子）
   ├ 共同祖先 + 系统发育树
   ├ 物种形成（生殖隔离）
   └ 灭绝、生命起源
```
