## 📌 一句话本质 / In One Sentence
> **进化树（cladogram）用分支图展示物种的亲缘关系——共同祖先在分叉点，共有的衍生特征（衍征）决定谁和谁分在一组。**

---

## 一、系统发育树基础 / Reading a Cladogram ⭐

**关键符号**：
- **分支末端（tips）**：现存物种（taxa）
- **节点（node）**：共同祖先（分叉点）
- **根（root）**：所有类群的最早共同祖先
- **分支（branch）**：演化谱系

**读图要点**：
- 两个物种的**共同祖先节点越近（越靠右/越晚分叉）→ 亲缘关系越近**
- 共同祖先越早分叉（越靠根）→ 亲缘越远

> ⚠️ **易错点**：亲缘远近看**共同祖先的位置**，不是看末端排列的左右顺序。两个相邻排列的物种不一定最亲近！

---

## 二、共有衍征 / Shared Derived Characters ⭐

- **衍征（derived character）**：在某谱系中**新出现**的特征
- **共有衍征（synapomorphy）**：一组生物共享的衍生特征 → 表明它们有**共同祖先**
- **外群（outgroup）**：亲缘较远的参照类群，用来确定哪些特征是"原始"vs"衍生"

**建树逻辑**：
- 共享某衍征的物种被分在同一**分支（clade）**
- 越多共有衍征 → 越近的共同祖先

**例**（特征表）：
| 物种 | 脊椎 | 四足 | 羊膜卵 | 毛发 |
|------|:---:|:---:|:---:|:---:|
| 鲨鱼 | ✓ | | | |
| 蛙 | ✓ | ✓ | | |
| 蜥蜴 | ✓ | ✓ | ✓ | |
| 老鼠 | ✓ | ✓ | ✓ | ✓ |
→ 越往下共享的衍征越多，老鼠与蜥蜴共享"羊膜卵"，关系比与蛙近。

## 三、分子系统学 / Molecular Phylogenetics

- 用 **DNA/蛋白质序列**的相似度建树（接 L07）
- 序列差异越小 → 共同祖先越近
- **分子钟（molecular clock）**：某些中性突变以大致恒定速率积累 → 可估算分歧时间
- **Lab 3 BLAST**：把未知序列与数据库比对，找最相似序列 → 推断亲缘

## 五、考点提示 / Exam Tips

- 🎯 **FRQ 高频**：读 cladogram 判断亲缘远近（看共同祖先节点）；用特征表建树。
- 🎯 共有衍征、外群的作用。
- 🎯 分子数据建树（序列相似度）。
- ⚠️ 亲缘远近看**最近共同祖先节点**，不是末端的左右相邻！可旋转分支而不改变关系。
