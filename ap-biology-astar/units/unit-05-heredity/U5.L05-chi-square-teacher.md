## 📌 一句话本质

> 卡方判断：观测与期望的差异是随机误差，还是真的不同？χ²>临界值则拒绝 H₀。

## 教学关键点（防错）

1. **df = 类别数 − 1**，不是样本量减一！（最高频错误）
2. **χ² > 临界值 → 拒绝 H₀**（差异显著）。学生常记反方向。
3. **E 用比例×总数**：3:1 → 3/4 和 1/4 乘总数，不是凭感觉。
4. **结论必须双表述**：①拒绝/不拒绝 H₀ ②差异是否显著、模型是否成立。
5. 让学生养成**五步全写**的习惯——每步都有分。

---

## Part B 答案

**B1**（1:1 比例，70 A : 90 B，共 160）：
(a) E(A) = 160 × 1/2 = **80**；E(B) = 160 × 1/2 = **80**
(b) χ² = (70−80)²/80 + (90−80)²/80 = 100/80 + 100/80 = 1.25 + 1.25 = **2.5**
(c) df=1，临界值 3.84；χ²(2.5) < 3.84 → **不拒绝 H₀**；观测与 1:1 无显著差异，符合预期比例。

**B2**：9:3:3:1 有 **4 个类别** → df = 4 − 1 = **3**（临界值 7.81）。因为自由度 = 类别数减一。

## Part C · FRQ 评分细则（4 分）

### (a) 零假设 — 1 分
- ✅ H₀：观测的紫:白比例与期望的 3:1 比例**无显著差异**（差异由随机机会引起）
- ❌ 不能写成"紫:白=3:1 一定成立"——要写"无显著差异"

### (b) 算期望 + 算 χ² — 2 分

**得分点 1（期望值）**：
- E(紫) = 200 × 3/4 = **150**
- E(白) = 200 × 1/4 = **50**

**得分点 2（χ² 计算）**：
- 紫：(138−150)²/150 = 144/150 = 0.96
- 白：(62−50)²/50 = 144/50 = 2.88
- χ² = 0.96 + 2.88 = **3.84**
- ✅ 必须写出公式、代入、求和；过程分

### (c) 结论 — 1 分
- χ²（3.84）**等于**临界值（3.84）
- 惯例：χ² ≤ 临界值 → **不拒绝 H₀**（恰等于时通常视为不拒绝/边界，按 ≤ 处理）
- ✅ 结论：差异不显著，数据**支持** 3:1 遗传模型，偏差可由随机误差解释
- （注：此题 χ² 恰等于临界值，是边界情形；讲评时说明 ≤ 临界值即不拒绝。若学生答"接近临界值、勉强支持"亦可酌情给分）

**满分示范**：
> H₀: There is no significant difference between the observed flower color ratio and the expected 3:1 ratio.
> E(purple) = 200 × 3/4 = 150; E(white) = 200 × 1/4 = 50.
> χ² = (138−150)²/150 + (62−50)²/50 = 144/150 + 144/50 = 0.96 + 2.88 = **3.84**.
> With df = 1, the critical value at p = 0.05 is 3.84. Since χ² (3.84) is not greater than 3.84, we **do not reject** the null hypothesis. The observed data are consistent with the expected 3:1 ratio; deviations are attributable to random chance.

## 讲评重点（高频失分）

| 失分点 | 频率 | 纠正 |
|--------|:---:|------|
| df 用样本量算 | ★★★★★ | df = 类别数 − 1 |

| 结论只说数字不解读 | ★★★ | 必说"模型是否成立" |
