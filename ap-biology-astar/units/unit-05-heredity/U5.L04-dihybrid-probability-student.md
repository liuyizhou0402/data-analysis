## 📌 一句话本质 / In One Sentence
> **双杂交看似复杂，其实是两个独立单杂交的"相乘"——用概率乘法法则，比画 16 格 Punnett 方格快得多。**

---

## 一、双杂交 / Dihybrid Cross ⭐

**例**：黄圆（YyRr）× 黄圆（YyRr），黄(Y)对绿(y)显性，圆(R)对皱(r)显性。

**经典结果（F2 表型比）**：**9 黄圆 : 3 黄皱 : 3 绿圆 : 1 绿皱**

> 这个 9:3:3:1 来自两个独立 3:1 的组合（自由组合定律，接 L03）。

---

## 二、概率法（比画方格快）/ Probability Method ⭐

> 双杂交不用画 16 格！把它拆成两个单杂交，再相乘。

**乘法法则（AND，同时发生）**：两个独立事件**都**发生的概率 = 各自概率**相乘**。

**例**：YyRr × YyRr，求黄圆（Y_R_）后代概率：
- 黄（Y_）的概率 = 3/4（来自 Yy×Yy）
- 圆（R_）的概率 = 3/4（来自 Rr×Rr）
- 黄圆 = 3/4 × 3/4 = **9/16** ✓

**求绿皱（yyrr）**：1/4 × 1/4 = **1/16** ✓

**加法法则（OR，二选一）**：两个互斥事件**任一**发生的概率 = 各自概率**相加**。

**例**：求"黄圆 或 绿皱"的概率 = 9/16 + 1/16 = 10/16 = 5/8

---

## 三、解题流程 / Strategy ⭐

> 遇到多基因杂交，**拆成单基因 + 相乘**：

1. 把每个基因当独立单杂交，算出各表型概率（如 3/4、1/4）
2. 求"同时满足多个性状" → 各概率**相乘**（乘法法则）
3. 求"满足这个或那个" → 概率**相加**（加法法则）

> 💡 这比画 4×4=16 格甚至 8×8=64 格的 Punnett 方格高效得多，三基因以上必用概率法。

---

## 四、配子种类计算 / Gamete Types

- 杂合基因座（heterozygous loci）数 = n → 配子种类 = **2ⁿ**
- YyRr（2 个杂合）→ 2² = **4 种配子**（YR, Yr, yR, yr）
- AaBbCc（3 个杂合）→ 2³ = 8 种配子
