---

## 📌 一句话本质 / In One Sentence
> **基因在染色体上，所以染色体的行为决定基因的遗传——X 染色体上的基因有特殊的伴性遗传模式，而染色体不分离会导致数目异常。**

---

## 一、伴性遗传 / Sex-Linked Inheritance ⭐

- 人类性染色体：女性 **XX**，男性 **XY**
- **X 连锁基因（X-linked）**：位于 X 染色体上的基因（Y 上很少有对应基因）
- 经典例：**红绿色盲、血友病**（X 连锁隐性）

**X 连锁隐性遗传的特点**：
- **男性更易表现**！男性只有 1 条 X（半合子 hemizygous），1 个隐性等位基因就表现
- 女性需 2 个隐性等位基因（X^a X^a）才表现 → 较少见
- 男性的 X 来自母亲 → 母亲是携带者（X^A X^a）可把性状传给儿子
- 父亲的 X 连锁性状不会传给儿子（儿子的 X 来自母亲，Y 来自父亲）

> 💡 **记忆**：X 连锁隐性 → "传男不传女"的表现倾向；母亲携带者 → 儿子 50% 患病。

---

## 二、不分离 / Nondisjunction ⭐

- **定义**：减数分裂时**同源染色体（减一）或姐妹染色单体（减二）未能正常分离**
- 结果：配子染色体数异常（n+1 或 n−1）
- 受精后 → **非整倍体（aneuploidy）**

**经典例**：
- **唐氏综合征（Down syndrome）**：21 号染色体三体（trisomy 21，2n+1 = 47）
- 性染色体异常：XXY（克氏综合征）、XO（特纳综合征）

> 🔗 串联 L01：不分离是减数分裂出错；理解减一 vs 减二不分离的区别。

## 三、系谱图分析 / Pedigree Analysis ⭐（FRQ 高频）

**符号约定**：
- □ = 男性，○ = 女性
- 实心（■●）= 表现该性状（受影响 affected）
- 空心 = 不表现
- 水平线连接 = 婚配；竖线向下 = 后代

**判断遗传模式的技巧**：

| 线索 | 提示 |
|------|------|
| 两个未受影响的亲代生出受影响后代 | 性状是**隐性** |
| 两个受影响的亲代生出未受影响后代 | 性状是**显性** |
| 男性患者远多于女性 | 可能是 **X 连锁隐性** |
| 受影响个体每代都有、不跳代 | 可能是**显性** |
| 父亲患病但儿子都不患（女儿可能携带）| 支持 **X 连锁** |

**分析步骤**：
1. 先判显性 or 隐性（看"无×无→有"=隐性）
2. 再判常染色体 or 伴性（看性别比例、父子传递）
3. 写出关键个体的基因型，验证

## 五、考点提示 / Exam Tips

- 🎯 **必考**：X 连锁隐性为什么男性多发；系谱图判断遗传模式。
- 🎯 不分离导致非整倍体（唐氏=21三体）。
- 🎯 系谱分析两步：先显/隐，再常/伴性。
- ⚠️ X 连锁隐性：父亲不会把 X 连锁性状传给儿子（儿子 X 来自母）。

(A) recessive  (B) dominant  (C) only found in males  (D) caused by nondisjunction
