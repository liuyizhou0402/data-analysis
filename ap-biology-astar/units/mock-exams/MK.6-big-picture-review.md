# MK.6 · 八单元串讲：一张大概念图（教师版 + 学生用）

> **本课目标**：用 4 条贯穿全书的大概念（Big Ideas），把 8 个单元连成一张网，做最后的整体梳理。

## 📌 本课打法
> AP 生物不是 8 个孤立单元，而是 **4 条大概念**贯穿始终。看懂这 4 条线，跨单元综合题就不怕了。

## 二、五条贯穿全书的主线 / Five Throughlines ⭐

### 主线 1：结构决定功能（Structure → Function）
```
U1 蛋白质/分子结构 → U2 膜/细胞器 → U3 酶活性位点
→ U4 受体 → U6 限制酶 → U7 同源结构
```

### 主线 2：能量与物质（Energy & Matter）
```
U1 ATP/化学键 → U3 光合↔呼吸（化学渗透）
→ U2 主动运输耗ATP → U8 能量流(10%)/物质循环
```
> 🔑 能量**单向流动**，物质**循环**——贯穿 U3 和 U8。

### 主线 3：信息流（Information Flow）
```
U6 中心法则(DNA→RNA→蛋白) → U4 细胞信号转导
→ U5 遗传信息传递 → U7 突变与进化
```

### 主线 4：稳态与调控（Homeostasis & Regulation）
```
U2 膜运输维持内环境 → U4 反馈机制/细胞周期检查点
→ U6 基因表达调控 → U8 种群密度调节
```
> 🔑 负反馈无处不在：U4 血糖/体温、U6 操纵子、U8 密度依赖。

### 主线 5：进化统一一切（Evolution Unifies）
```
U5 变异来源(减数分裂) + U6 突变 → U7 自然选择
→ U2 内共生 → U6 通用密码 → U8 协同进化
```

## 三、跨单元高频综合考点 / Cross-Unit Integration ⭐

> AP 大题常跨单元。这些"连接点"必须熟：

| 连接 | 内容 |
|------|------|
| U1↔U3 | 蛋白质结构→酶；ATP 核苷酸→能量 |
| U2↔U3 | 线粒体/叶绿体结构→化学渗透；内共生 |
| U3↔U8 | 光合呼吸→生态系统能量流 |
| U4↔U6 | 信号转导→基因表达调控；癌症 |
| U5↔U7 | 减数分裂变异→自然选择原料；HW |
| U6↔U7 | 突变→进化；通用密码→共同祖先 |
| U4↔U8 | 反馈调节→种群密度调节 |

## 六、课堂用法（给老师）

1. 让学生**自己画**一张"8 单元大概念图"（白纸），老师补充。
2. 出 2–3 道**跨单元综合题**现场练（如"光合-呼吸-生态能量流"）。
3. 把 7 个统计公式集中默写一遍。
4. 强调：大题得高分靠"**把单元连起来**"。
