# A-Level 生物 A* 冲刺课程体系（Cambridge CIE 9700）

> 一套面向国际学校学生的 **100 课时** 完整教学产品：从零基础到 A*。
> 中英双语 · 对标 Cambridge CIE 9700（2025–2027 考纲）· 老师教案 + 学生讲义 + 配套练习 + 答案解析 全套。

---

## 这套产品是什么 / What this is

这不是把市面资料拼凑的"题集"，而是一套**原创、成体系、可直接交付教学**的课程产品，定位为授课老师的**个人 IP 教材**。

- **对标考纲**：Cambridge International AS & A Level Biology **9700**（2025、2026、2027 考试适用）。
- **覆盖范围**：AS（Topic 1–11）+ A2（Topic 12–19），共 **19 个主题**，拆解为 **100 个 1 小时课时**。
- **语言**：中英双语。**讲解、思路、记忆方法用中文**；**专业术语、必背定义、答题模板用英文**（因为考试全英文作答）。
- **四件套**：每个主题都配齐
  1. **教师教案**（Teacher Lesson Plan）— 教学目标、分课时授课流程、时间分配、板书重点、易错点、提问设计。
  2. **学生讲义**（Student Handout）— 学生能直接看懂的知识精讲、图示说明、必背清单、A* 拓展。
  3. **配套练习**（Practice）— 按 Paper 1/2/4 题型设计，含选择题、结构题、应用分析题。
  4. **答案与解析**（Answers & Mark Scheme）— 对标 Cambridge 评分点（mark points），讲清"为什么这么写才能得分"。

## 怎么用这套教材 / How to use

**作为老师（你）：**
1. 先读 `00-curriculum-plan/100-hour-master-plan.md`，掌握整体节奏与每节课目标。
2. 上课前看对应主题的 `teacher-lesson-plan.md`，按流程与时间分配授课。
3. 把 `student-handout.md` 发给学生作为讲义；课后布置 `practice.md`。
4. 用 `practice-answers.md` 对标 Cambridge 评分点批改，重点讲"踩分点"。

**作为学生：**
1. 课前预习 `student-handout.md` 的"本课目标"和"必背清单"。
2. 课后独立完成 `practice.md`，再对 `practice-answers.md`，把丢分点抄进错题本。
