# A* 提分逻辑与教学法 / The A* Strategy
## 这套教材凭什么让学生"轻松上 A*"

> 这是整套课程的**方法论内核**，也是你区别于其他老师的**个人 IP**。
> 它回答一个问题：为什么很多学生"背得很熟却只能拿 B"，而我们的学生能稳 A*？

---

## 一、核心诊断：A* 学生 vs B 学生差在哪

| 维度 | B 学生 | A* 学生 |
|---|---|---|
| 知识 | 记住了 | 记住 **+ 能用英文精准表达** |
| 数据题 | 描述现象 | **描述 + 解释 + 量化（引用数据）** |
| 陌生情境 | 慌、答非所问 | 把课本原理**迁移**到新材料 |
| 答题 | 答得多但散 | 紧扣 **command word** 与**踩分点** |
| 实验题 | 凭感觉 | 有**变量/控制/评价**的固定框架 |

**结论**：A* 不靠"学更多"，靠"在 **AO2（应用与分析）** 和 **答题规范** 上把会的分全拿到"。本课程所有设计都围绕这两点。

---

## 二、五大教学法支柱 / Five pillars

### 支柱 1：术语英文化、定义"官方措辞化"
- 考试全英文作答，**关键定义必须用英文背、且贴近 mark scheme 措辞**。
- 每课讲义设 **"Must-know definitions"** 模块：给出官方风格英文定义 + 中文解释。
- 教学法：中文讲透**原理**（学生懂），英文锁死**表达**（考试拿分）。

### 支柱 2：用 command words 训练"精准作答"
- Cambridge 题干动词（**describe / explain / compare / suggest / calculate…**）决定答法。
- 学生丢分常因"答错类型"：题目要 *explain*（讲原因）却只 *describe*（讲现象）。
- 每套练习的答案都标注：**这道题的 command word 是什么、对应要写什么**。
- 详见 `01-standards-and-templates/command-words-and-mark-scheme-guide.md`。

### 支柱 3："踩分点"意识（Mark-point thinking）
- Cambridge 评分是**按点给分（marking points）**，不是按篇幅。
- 训练学生：**一句话 = 一个采分点**，写满题目分值对应的点数即可，不啰嗦。
- 答案解析里把每个 mark point 拆开标注 ✓，让学生看见"分是怎么给的"。

### 支柱 4：数据题三步法（Describe–Explain–Quote）
针对 AO2 数据/图表题（最大权重、最易丢分），固定框架：
1. **Describe 趋势**：整体走向（上升/下降/峰值/平台）。
2. **Quote 数据**：引用具体数值与单位（含起止点、变化量）。
3. **Explain 原因**：用学过的生物学原理解释为什么。
> 口诀：**"说趋势、引数据、讲原理"**。本课程每个数据题答案都按此拆解。

### 支柱 5：实验题万能框架（Plan–Control–Evaluate）
针对 P3/P5：
- **Plan**：自变量、因变量、如何测量、范围与重复。
- **Control**：控制变量（列具体的、可操作的）。
- **Evaluate**：误差来源、可靠性、如何改进。
> 把"实验题"从玄学变成"套框架填空"。

## 三、每节课的统一节奏 / Lesson rhythm（建议）

每个 60 分钟课时，按"**5–15–20–15–5**"推进：

| 时间 | 环节 | 目的 |
|---|---|---|
| 0–5′ | **回顾 + 目标** Recap & objectives | 衔接上节、明确本课要拿下什么 |
| 5–20′ | **精讲新知** Teach（中文讲透原理） | 建立理解 |
| 20–40′ | **示范 + 当堂练** Model & practice | 老师先示范一道标准答法，再当堂练 |
| 40–55′ | **真题题型训练** Exam-style Q | 用 P1/P2/P4 真题型即时应用 |
| 55–60′ | **总结 + 必背清单** Wrap & must-know | 锁定英文定义与采分点 |

---

## 四、贯穿全程的三条暗线 / Three threads

1. **回扣 AS**：A2 教学中持续连接 AS 知识，为 Paper 4 综合题（synoptic）铺路。
2. **结构—功能关系**：生物学的核心思维（structure relates to function）——每讲一个结构都问"它的结构如何支持功能"。
3. **错题资产化**：每个学生建错题本，按"知识漏洞 / 表达不准 / 审题错"三类归档，定期复盘。

---

## 五、AS / A2 阶段重点差异 / What changes from AS to A2

| | AS | A2 |
|---|---|---|
| 认知层级 | 以理解、应用为主 | 大量**综合（synoptic）+ 评价** |
| 题型 | 短结构题 | 长结构题 + **15 分长答题** + 实验设计 |
| 关键提分点 | 术语精准、基础数据题 | 跨主题连接、统计检验、长答题结构化 |

---

## 六、给学生的"A* 习惯清单" / A* habits

- [ ] 每个定义都能**默写英文**（不是只看懂）。
- [ ] 看到题先**圈 command word**，再决定怎么答。
- [ ] 数据题永远**引用具体数值**。
- [ ] 答题**一点一分**，写够分值对应点数。
- [ ] 实验题先想**变量与控制**。
- [ ] 每周整理错题，按三类归档。
- [ ] A2 阶段每周**回看一个 AS 主题**。

---

> 这套方法论是课程的灵魂。后续每个主题的教案与答案解析，都是这六大支柱的**具体落地**。
