## 1. 减数分裂与遗传变异 / Meiosis & Variation

减数分裂产生 **4 个单倍体（haploid, n）配子**，染色体数目减半。

**遗传变异的三大来源**：
| 来源 | 发生时机 | 机制 |
|---|---|---|
| **交叉互换 Crossing over** | MI 前期 | 同源染色体的非姐妹染色单体交换片段 → 新等位组合 |
| **独立分配 Independent assortment** | MI 中期 | 同源染色体对随机朝向两极排列 → 2ⁿ 种组合 |
| **随机受精 Random fertilisation** | 受精时 | 任意配子结合 → 增加组合多样性 |

> n 对染色体可产生 **2ⁿ** 种配子（仅独立分配，未计交叉互换）。人类 n=23 → 2²³ ≈ 800 万种。

## 3. 单因子杂交 / Monohybrid Cross

**规范遗传图解格式**（必须每步写全）：
```
亲本表型(Parental phenotypes)：   高茎    ×    矮茎
亲本基因型(Genotypes)：           Tt     ×    tt
配子(Gametes)：               (T) (t)   ×  (t) (t)   ← 圈出配子！

Punnett 方格：
           T          t
      ┌─────────┬─────────┐
   t  │   Tt    │   tt    │
      ├─────────┼─────────┤
   t  │   Tt    │   tt    │
      └─────────┴─────────┘

子代基因型：  Tt : tt = 1 : 1
子代表型：    高茎 : 矮茎 = 1 : 1
```

**经典比例**：
- 纯合显 × 纯合隐 → F1 全显（Tt）
- F1 × F1 → F2 **3 显 : 1 隐**

**测交（test cross）**：未知基因型 × 隐性纯合（tt）→ 由子代判断未知个体是纯合还是杂合。

## 4. 双因子杂交 / Dihybrid Cross

两对独立遗传的基因，F1 自交 → F2 比例 **9 : 3 : 3 : 1**。

```
F1 自交：  RrYy   ×   RrYy
配子(各4种)： (RY)(Ry)(rY)(ry)
→ 4×4 Punnett 方格 → 16 格

F2 表型比：
9 圆黄 : 3 圆绿 : 3 皱黄 : 1 皱绿
```

## 5. 共显性、复等位基因 / Codominance & Multiple Alleles

### 5.1 共显性（Codominance）
杂合时**两个等位基因都表达**（不是中间型）。
- 例：牛毛色，红(C^R)与白(C^W) → C^R C^W = **沙毛（roan，红白混杂）**

> ⚠️ 区分：**不完全显性（incomplete dominance）**是中间型（如红×白→粉）；**共显性**是两者都显现。

### 5.2 复等位基因 + 共显性：ABO 血型
- 三个等位基因：**Iᴬ, Iᴮ, i**（i 为隐性）
- Iᴬ 和 Iᴮ **共显性**；都对 i 显性

| 基因型 | 血型 |
|---|---|
| Iᴬ Iᴬ / Iᴬ i | A 型 |
| Iᴮ Iᴮ / Iᴮ i | B 型 |
| **Iᴬ Iᴮ** | **AB 型**（共显性）|
| i i | O 型 |

## 6. 伴性遗传 / Sex Linkage

性染色体：**XX（雌）/ XY（雄）**。X 连锁基因写在 X 上。

**例：红绿色盲（X 连锁隐性）**，用 Xᴮ（正常）/ Xᵇ（色盲）：

| 基因型 | 表型 |
|---|---|
| Xᴮ Xᴮ | 正常女性 |
| **Xᴮ Xᵇ** | **携带者女性**（正常但携带）|
| Xᵇ Xᵇ | 色盲女性 |
| Xᴮ Y | 正常男性 |
| **Xᵇ Y** | **色盲男性**（半合子）|

> 🎯 男性只有一个 X（半合子 hemizygous）→ 单个隐性等位即表现 → **隐性 X 连锁病在男性更常见**。

## 7. 上位效应 / Epistasis

一个基因座的等位基因**影响另一个基因座**的表达。

| 类型 | F2 比例 |
|---|---|
| 普通双因子（无上位）| 9:3:3:1 |
| **隐性上位**（recessive epistasis）| **9:3:4** |
| **显性上位**（dominant epistasis）| **12:3:1** 或 13:3 |

> 上位 9:3:4 中，"4" = 两类表型因上位基因隐性纯合而合并。

## 8. 卡方检验 / Chi-squared Test

用于判断观察值与预期值的差异是否**仅由随机误差引起**。

$$\chi^2 = \sum \frac{(O-E)^2}{E}$$

**步骤**：
1. **虚无假设**：观察值与预期比例**无显著差异**
2. 按理论比例算每类**预期值 E**
3. 代入公式算 χ²
4. **自由度 df** = 类别数 − 1
5. 查表，看 **p = 0.05** 对应的临界值
6. 结论：
   - **χ² < 临界值** → 差异不显著 → **接受**虚无假设（差异由随机引起）
   - **χ² > 临界值** → 差异显著 → **拒绝**虚无假设（有其他因素）

**示例计算表**：
| 类别 | O | E | (O−E) | (O−E)² | (O−E)²/E |
|---|---|---|---|---|---|
| 圆黄 | 315 | 312.75 | 2.25 | 5.06 | 0.016 |
| 圆绿 | 108 | 104.25 | 3.75 | 14.06 | 0.135 |
| 皱黄 | 101 | 104.25 | −3.25 | 10.56 | 0.101 |
| 皱绿 | 32 | 34.75 | −2.75 | 7.56 | 0.218 |
| | | | | **χ²=** | **0.47** |

df = 4−1 = 3；p=0.05 临界值 = 7.82。χ²(0.47) < 7.82 → 接受虚无假设。

## 9. 突变 / Mutations

### 9.1 基因突变（gene mutation）
| 类型 | 描述 | 后果 |
|---|---|---|
| 替换 substitution | 一个碱基被替换 | 可能沉默/错义/无义 |
| 插入 insertion | 插入碱基 | 移码（若非3倍数）|
| 缺失 deletion | 缺失碱基 | 移码（若非3倍数）|

**镰刀型贫血**：HBB 基因 **GAG→GTG**（替换）→ 谷氨酸→缬氨酸 → 血红蛋白异常聚合 → 红细胞镰刀形。

### 9.2 染色体突变
- 数目：**不分离（non-disjunction）** → 唐氏综合征（21 三体 trisomy 21）
- 结构：缺失、重复、倒位、易位
