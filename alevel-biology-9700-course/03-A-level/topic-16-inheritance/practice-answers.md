### B2
**(a) Genetic diagram for carrier × normal man. [4]**
- ✓ 1. Parents: **XᴴXʰ (carrier woman) × XᴴY (normal man)**. **[1]**
- ✓ 2. Gametes: woman **(Xᴴ)(Xʰ)**; man **(Xᴴ)(Y)** — circled. **[1]**
- ✓ 3. Offspring genotypes: XᴴXᴴ, XᴴXʰ, XᴴY, XʰY. **[1]**
- ✓ 4. Phenotypes: normal daughter, carrier daughter, normal son, **haemophiliac son**. **[1]**

**(b) Probability daughter has haemophilia. [2]** — *Explain*
- ✓ 1. Probability = **0** (no daughter can be XʰXʰ). **[1]**
- ✓ 2. Daughters receive Xᴴ from the father, so cannot be homozygous recessive. **[1]**

**(c) Why haemophilia more common in males. [2]** — *Explain*
- ✓ 1. Males have only **one X chromosome (XʰY)** — a single recessive allele is expressed (hemizygous). **[1]**
- ✓ 2. Females need **two copies (XʰXʰ)** to be affected, which is much rarer. **[1]**

### B3
**(a) Define epistasis. [1]**
- ✓ The interaction in which **one gene (locus) affects/masks the expression of another gene** at a different locus. **[1]**

**(b) Why 9:7 not 9:3:3:1. [3]** — *Explain*
- ✓ 1. Both dominant alleles (A_ and B_) are required for colour to develop. **[1]**
- ✓ 2. The 9 (A_B_) are coloured; the 3 (A_bb) + 3 (aaB_) + 1 (aabb) all lack a required allele → **white**. **[1]**
- ✓ 3. So 3 + 3 + 1 = **7 white**, giving 9 coloured : 7 white. **[1]**

## Part C — 应用/数据题

### C1
**(a) Expected numbers. [2]** — *Calculate*
- ✓ 1. Total = 160; 160 ÷ 16 = 10 per part. **[1]**
- ✓ 2. E = Round yellow **90**, Round green **30**, Wrinkled yellow **30**, Wrinkled green **10**. **[1]**

**(b) Chi-squared test. [3]** — *Calculate（show working）*
- ✓ 1. Set up (O−E)²/E for each:
  - Round yellow: (88−90)²/90 = 4/90 = 0.044
  - Round green: (32−30)²/30 = 4/30 = 0.133
  - Wrinkled yellow: (28−30)²/30 = 4/30 = 0.133
  - Wrinkled green: (12−10)²/10 = 4/10 = 0.400 **[1]**
- ✓ 2. Sum: χ² = 0.044 + 0.133 + 0.133 + 0.400. **[1]**
- ✓ 3. χ² = **0.71** (≈0.71). **[1]**

**(c) Conclusion. [2]** — *Explain*
- ✓ 1. χ² (0.71) is **less than** the critical value (7.82). **[1]**
- ✓ 2. **Accept the null hypothesis** — the difference between observed and expected is due to chance; the data fit the 9:3:3:1 ratio. **[1]**

### C2
**(a) Dominant or recessive? [2]** — *State + reason*
- ✓ 1. **Recessive**. **[1]**
- ✓ 2. Two unaffected parents have an **affected child** → the allele must be recessive (parents are carriers). **[1]**

**(b) Autosomal vs sex-linked. [2]** — *Explain*
- ✓ 1. The condition affects **both sexes roughly equally** / an affected female has an unaffected father. **[1]**
- ✓ 2. This is consistent with **autosomal** inheritance (if X-linked recessive, an affected daughter would need an affected father). **[1]**

**(c) Probability child affected. [2]** — *Calculate*
- ✓ 1. Both parents heterozygous (Aa × Aa) → offspring 1 AA : 2 Aa : 1 aa. **[1]**
- ✓ 2. Probability affected (aa) = **1/4 (25%)**. **[1]**

## 老师讲评建议 / Marking tips
- 遗传图解题的"规范分"最易丢：**配子必须圈、必须写表型比**——每年都有学生因格式丢 1-2 分。
- C1 卡方题：强调"先算 E（总数÷比例份数）→ 列表算 (O-E)²/E → 求和 → 比临界值"。结论必须说"接受/拒绝虚无假设"。
- B2(b) 女儿患血友病概率为 0：因为父亲提供 Xᴴ——这是 X 连锁题的经典陷阱。
- B3 上位 9:7：要解释"两个显性都需要"才是有色——这是 epistasis 题的核心逻辑。
- Q10 血型：AB × O 的子代只能 A 或 B，绝不可能 AB/O——务必反复强调。
