## 课程安排 / Lesson Breakdown

### 第 1 课（60 min）—— 分类系统 Classification

**5 min 导入**：地球上数百万物种，如何系统命名和归类？

**20 min 讲授分类阶元**：
**三域系统（three domains）**（基于 rRNA 序列）：
- **细菌域（Bacteria）**
- **古菌域（Archaea）**（极端环境，膜脂/细胞壁不同于细菌）
- **真核域（Eukarya）**

**分类阶元（hierarchy）**（从大到小）：
```
域 Domain → 界 Kingdom → 门 Phylum → 纲 Class → 目 Order
→ 科 Family → 属 Genus → 种 Species
```
记忆口诀：Dear King Philip Came Over For Good Soup

**双名法（binomial nomenclature）**：
- 属名（首字母大写）+ 种加词（小写），斜体
- 例：*Homo sapiens*、*Panthera leo*

**15 min 讲授五界/分类依据**：
- 原核生物界、原生生物界、真菌界、植物界、动物界（传统五界）
- 现代分类依据：分子证据（DNA/RNA 序列）、形态、生化

**15 min 活动**：学生将给定生物分类到正确阶元；分析 rRNA 树状图。

**5 min 总结**

---

### 第 2 课（60 min）—— 生物多样性的层次 Levels of Biodiversity

**5 min 导入**：生物多样性不只是"物种数量"。

**25 min 讲授三个层次**：
| 层次 | 定义 |
|---|---|
| **遗传多样性（genetic diversity）** | 一个物种内等位基因的多样性 |
| **物种多样性（species diversity）** | 一个生态系统中物种的数量（丰富度）和均匀度 |
| **生态系统多样性（ecosystem diversity）** | 一个区域内不同生境/生态系统的多样性 |

**物种多样性的两个组分**：
- **物种丰富度（species richness）**：物种的**数量**
- **物种均匀度（evenness）**：各物种个体数的**相对均衡程度**

> 高多样性 = 物种多 + 分布均匀。两个生态系统物种数相同，但均匀度不同 → 多样性不同。

**25 min 活动**：
- 对比两个群落数据，讨论丰富度与均匀度
- 引入需要量化指标的需求 → Simpson 指数

**5 min 总结**

---

### 第 3 课（60 min）—— Simpson 多样性指数

**5 min 导入**：如何用一个数字表达多样性？

**30 min 讲授 Simpson 指数**：
CIE 9700 使用的公式：
$$D = 1 - \sum \left(\frac{n}{N}\right)^2$$
- n = 每个物种的个体数
- N = 所有物种的总个体数
- D 范围 0–1，**D 越接近 1 多样性越高**

**计算步骤**：
1. 列出每个物种的 n
2. 算 N（总数）
3. 对每个物种算 (n/N)²
4. 求和
5. D = 1 − 求和

**指数的意义**：
- D 高 = 物种多且均匀 → 生态系统更稳定、抗干扰能力强
- D 低 = 少数物种主导 → 脆弱

**20 min 活动**：完整做 1-2 道 Simpson 计算题（含表格）

**5 min 总结**

---

### 第 4 课（60 min）—— 取样方法 Sampling Methods

**5 min 导入**：不可能数清所有个体，如何估算？取样。

**30 min 讲授取样方法**：
**随机取样（random sampling）**：
- 样方（quadrat）：用随机数生成坐标，避免偏差
- 测量：物种频度（frequency）、密度（density）、百分比覆盖度（% cover）

**系统取样（systematic sampling）**：
- 样带（transect）：沿环境梯度（如从海岸到内陆）规律取样
- 用于研究分布与环境因子的关系（zonation）

**动物取样**：
- 标志重捕法（mark-release-recapture）：
$$\text{种群估算} = \frac{\text{第一次标记数} \times \text{第二次捕获数}}{\text{第二次中带标记数}}$$
（Lincoln 指数）
- 假设：标记不影响生存/行为；充分混合；无大量出生/死亡/迁移

**20 min 活动 + 计算**：标志重捕法计算题；样方数据处理

**5 min 总结**

---

### 第 5 课（60 min）—— 保护生物多样性与综合复习

**25 min 讲授保护（conservation）**：
**为什么维持生物多样性？**
- 生态价值：维持生态系统稳定、营养循环、授粉
- 经济价值：药物来源、农作物野生近缘种（基因库）、生态旅游
- 伦理价值：物种的内在价值
- 气候调节

**保护方法**：
| 方法 | 说明 |
|---|---|
| **就地保护（in situ）** | 自然保护区、国家公园（保护栖息地）|
| **迁地保护（ex situ）** | 动物园、植物园、种子库（seed bank）|
| **国际合作** | CITES（濒危物种贸易公约）、IUCN 红色名录 |
| **可持续管理** | 可持续渔业/林业、配额制 |

**种子库（seed bank）**：低温干燥保存种子，维持遗传多样性，可用于复种。

**20 min P4/P5 综合题精练**：Simpson 计算 + 取样设计 + 保护论述

**15 min 易错纠正**：
- ⚠️ Simpson 指数 D 越**高**多样性越**高**（用 1−Σ 公式时）
- ⚠️ 随机取样用随机数（避免主观偏差），不是"随便走走"
- ⚠️ 标志重捕假设要记牢（不影响行为、充分混合等）

## 评分考点 / Key Exam Points
1. 三域：细菌、古菌、真核（基于 rRNA）
2. 分类阶元 8 级；双名法格式（属种、斜体）
3. 多样性三层次；物种多样性 = 丰富度 + 均匀度
4. Simpson D = 1−Σ(n/N)²，越接近 1 越高
5. 标志重捕：种群 = (M₁×n₂)/m₂，记假设

---

## 常见误区 / Common Misconceptions
- "生物多样性就是物种数量"——还要看**均匀度**和遗传/生态系统层次
- "Simpson 指数越小多样性越高"——用 1−Σ 公式时**越大越高**（注意公式版本）
- "随机取样 = 随意选地点"——必须用**随机数**确定坐标
- "迁地保护就是建保护区"——迁地是**离开**原栖息地（动物园/种子库）；就地才是保护区
