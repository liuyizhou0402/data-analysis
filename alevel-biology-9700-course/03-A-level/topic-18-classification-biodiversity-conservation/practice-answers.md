- ✓ 1. Domain, Kingdom, Phylum, Class... **[1]**

**(c) Two rules for binomial name. [2]**
- ✓ 1. Genus name starts with a **capital letter**; species name is **lower case**. **[1]**
- ✓ 2. The whole name is written in *italics* (or **underlined** if handwritten). **[1]**

### B2
**(a) Distinguish three types of diversity. [3]** — *Distinguish*
- ✓ 1. **Genetic diversity** = variety of alleles within a species. **[1]**
- ✓ 2. **Species diversity** = number and evenness of species in a community/ecosystem. **[1]**
- ✓ 3. **Ecosystem diversity** = variety of different habitats/ecosystems in an area. **[1]**

**(b) Why species diversity depends on richness + evenness. [2]** — *Explain*
- ✓ 1. Richness = number of species; evenness = how equal their abundances are. **[1]**
- ✓ 2. Two communities with the same number of species differ in diversity if one is **dominated by a single species** (low evenness = lower diversity). **[1]**

**(c) Why high biodiversity → more stable ecosystem. [2]** — *Explain*
- ✓ 1. More species means more **complex food webs / more interactions**. **[1]**
- ✓ 2. If one species is lost, others can compensate → ecosystem more **resilient to disturbance**. **[1]**

### B3
**(a) Three reasons conservation is important. [3]** — *Explain*
任三条（各 1 分）：
- ✓ **Ecological**: maintains ecosystem stability / nutrient cycling / pollination. **[1]**
- ✓ **Economic / medical**: source of medicines, crop wild relatives (gene pool for breeding), ecotourism. **[1]**
- ✓ **Ethical**: organisms have intrinsic value / responsibility to future generations. **[1]**

**(b) In situ vs ex situ + examples. [4]** — *Distinguish*
- ✓ 1. In situ = conservation **in the natural habitat**. **[1]**
- ✓ 2. Example: nature reserve / national park. **[1]**
- ✓ 3. Ex situ = conservation **outside the natural habitat**. **[1]**
- ✓ 4. Example: zoo / botanic garden / seed bank. **[1]**

**(c) How seed bank conserves biodiversity. [2]** — *Explain*
- ✓ 1. Seeds are stored in **cool, dry conditions** to keep them viable for long periods. **[1]**
- ✓ 2. Preserves the **genetic diversity** of plant species; seeds can be grown to restore populations. **[1]**

## Part C / D — 应用与实验数据题

### C1
**(a) Calculate Simpson's Index. [4]** — *Calculate（show working）*
- ✓ 1. N = 12 + 8 + 15 + 5 = **40**. **[1]**
- ✓ 2. (n/N)²: Mayfly (12/40)²=0.09; Caddis (8/40)²=0.04; Shrimp (15/40)²=0.1406; Beetle (5/40)²=0.0156. **[1]**
- ✓ 3. Σ(n/N)² = 0.09 + 0.04 + 0.1406 + 0.0156 = **0.286**. **[1]**
- ✓ 4. D = 1 − 0.286 = **0.71**. **[1]**

**(b) Compare two streams. [3]** — *Compare + Suggest（引用数据）*
- ✓ 1. The clean stream (D = 0.71) has **much higher biodiversity** than the polluted stream (D = 0.21). **[1]**
- ✓ 2. Pollution has **reduced biodiversity** — fewer species and/or dominance by a few tolerant species. **[1]**
- ✓ 3. Pollution-sensitive species (e.g. mayfly nymphs) are lost, lowering richness and evenness. **[1]**

### C2
**(a) Estimate snail population. [2]** — *Calculate*
- ✓ 1. N = (M₁ × n₂) / m₂ = (60 × 75) / 20. **[1]**
- ✓ 2. = 4500 / 20 = **225 snails**. **[1]**

**(b) Two assumptions. [2]**
任两条：
- ✓ Marking does **not affect survival or behaviour** of the snails. **[1]**
- ✓ Marked snails **mix randomly** back into the population. **[1]**
- ✓ No significant **births, deaths, immigration or emigration** between captures; marks don't wear off. **[1]**

**(c) Why bright paint → overestimate. [2]** — *Suggest*
- ✓ 1. Bright paint may make marked snails **more visible to predators** → more likely to be eaten. **[1]**
- ✓ 2. Fewer marked snails recaptured (m₂ smaller) → calculated N is **larger** (overestimate). **[1]**

## 老师讲评建议 / Marking tips
- C1 Simpson 计算：最易丢分在**先算 N**和**逐项平方**。强调展示每一步，结尾 D = 1 − Σ。
- C2 标志重捕：公式 N = (M₁×n₂)/m₂ 要记牢；假设题是常考送分点。
- Q5/Q10 + C1(b)：D 越大多样性越高（CIE 用 1−Σ 版本）——务必让学生记准方向，避免与其他教材的 Simpson 版本混淆。
- B3(b) 就地/迁地：用"in = 原地（保护区）；ex = 离开（动物园/种子库）"帮助记忆。
