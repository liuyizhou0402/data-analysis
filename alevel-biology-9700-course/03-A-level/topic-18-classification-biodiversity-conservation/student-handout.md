## 1. 分类系统 / Classification

### 1.1 三域系统（Three Domains）
基于 **rRNA 序列**比较：
| 域 | 特征 |
|---|---|
| **Bacteria 细菌** | 原核，肽聚糖细胞壁 |
| **Archaea 古菌** | 原核，但膜脂/细胞壁/rRNA 不同；常居极端环境 |
| **Eukarya 真核** | 有膜包细胞核与细胞器（原生生物/真菌/植物/动物）|

### 1.2 分类阶元（Taxonomic Hierarchy）
```
域 Domain → 界 Kingdom → 门 Phylum → 纲 Class
→ 目 Order → 科 Family → 属 Genus → 种 Species
```
记忆：**D**ear **K**ing **P**hilip **C**ame **O**ver **F**or **G**ood **S**oup

### 1.3 双名法（Binomial Nomenclature）
- **属名**（首字母大写）+ **种加词**（小写），整体**斜体**
- 例：*Homo sapiens*，*Panthera leo*，*Escherichia coli*

### 1.4 分类依据（现代）
- **分子证据**：DNA/RNA 序列、蛋白质（亲缘越近越相似）
- 形态、解剖、生化特征

## 2. 生物多样性的层次 / Levels of Biodiversity

| 层次 | 定义 |
|---|---|
| **遗传多样性（genetic）** | 一个物种内**等位基因**的多样性 |
| **物种多样性（species）** | 一个生态系统中物种的**数量与均匀度** |
| **生态系统多样性（ecosystem）** | 一个区域内不同**生境/生态系统**的多样性 |

**物种多样性 = 两个组分**：
- **物种丰富度（richness）**：物种**数量**
- **物种均匀度（evenness）**：各物种个体数的**均衡程度**

> 🎯 两个群落物种数相同，但若一个由单一物种主导、另一个分布均匀 → 后者多样性更高。

## 3. Simpson 多样性指数 / Simpson's Index

CIE 9700 使用：
$$D = 1 - \sum \left(\frac{n}{N}\right)^2$$
- **n** = 每个物种的个体数
- **N** = 所有物种的总个体数
- **D** 范围 0–1，**越接近 1 多样性越高**

### 计算步骤
```
① 列出每个物种个体数 n
② 算总数 N
③ 每个物种算 (n/N)²
④ 求和 Σ(n/N)²
⑤ D = 1 − Σ
```

### 示例
| 物种 | n | n/N | (n/N)² |
|---|---|---|---|
| A | 10 | 0.40 | 0.16 |
| B | 9 | 0.36 | 0.13 |
| C | 6 | 0.24 | 0.058 |
| **总 N = 25** | | | Σ = 0.348 |

D = 1 − 0.348 = **0.65**

**指数意义**：
- D 高 → 物种多且均匀 → 生态系统更**稳定**、抗干扰能力强
- D 低 → 少数物种主导 → 生态系统**脆弱**

## 4. 取样方法 / Sampling Methods

### 4.1 随机取样（Random Sampling）—— 植物/固着生物
- **样方（quadrat）**：用**随机数**生成网格坐标，放置样方
- 避免主观偏差（不能"挑好看的地方"）
- 测量指标：
  - 频度（frequency）：物种出现的样方比例
  - 密度（density）：单位面积个体数
  - 百分比覆盖度（% cover）：物种覆盖样方的面积比例

### 4.2 系统取样（Systematic Sampling）—— 研究分布梯度
- **样带（transect）**：沿环境梯度（如海岸→内陆）规律取样
- 研究**带状分布（zonation）**与环境因子的关系
- 类型：线样带（line transect）、带样带（belt transect）

### 4.3 标志重捕法（Mark-Release-Recapture）—— 活动动物
$$\text{种群估算 } N = \frac{M_1 \times n_2}{m_2}$$
- M₁ = 第一次捕获并标记的数量
- n₂ = 第二次捕获总数
- m₂ = 第二次捕获中带标记的数量

**假设条件**（必背）：
1. 标记**不影响**生存或行为（不增加被捕食风险）
2. 标记动物充分**混合**回种群
3. 取样期间无显著**出生/死亡/迁入/迁出**
4. 标记不脱落

## 5. 保护生物多样性 / Conservation

### 5.1 为什么维持生物多样性？
| 价值 | 说明 |
|---|---|
| 生态 | 维持生态系统稳定、营养循环、授粉 |
| 经济 | 药物来源、作物野生近缘种（基因库）、生态旅游 |
| 农业 | 遗传多样性提供育种原材料（抗病基因等）|
| 伦理 | 物种有内在价值，人类有责任保护 |
| 气候 | 森林等调节气候、固碳 |

### 5.2 保护方法
| 方法 | 类型 | 例子 |
|---|---|---|
| **就地保护（in situ）** | 在原栖息地 | 自然保护区、国家公园 |
| **迁地保护（ex situ）** | 离开原栖息地 | 动物园、植物园、**种子库** |
| **国际合作** | 法规/名录 | **CITES**（濒危物种贸易公约）、IUCN 红色名录 |
| **可持续管理** | 资源利用 | 可持续渔业（配额）、林业（择伐补种）|

**种子库（seed bank）**：低温干燥保存种子 → 维持遗传多样性 → 可用于复育与研究。

> ⚠️ 就地 vs 迁地：就地 = 在原地（保护区）；迁地 = 离开原地（动物园/种子库）。

## A* 拓展 / Stretch to A*
- **Topic 17 连接**：物种多样性是进化的产物；保护遗传多样性 = 保护进化潜力（应对环境变化）。
