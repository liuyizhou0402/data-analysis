## 1. 变异的来源 / Sources of Variation

| 类型 | 来源 | 可遗传？ |
|---|---|---|
| **遗传变异** | 突变、减数分裂（交叉互换/独立分配）、随机受精 | ✅ 可遗传 |
| **环境变异** | 营养、温度、光照等 | ❌ 不可遗传 |

- **不连续变异**（discontinuous）：单基因控制，明显类别（如血型）
- **连续变异**（continuous）：多基因（polygenic）+ 环境影响，连续范围（如身高）

> 进化的**原材料**是可遗传的变异（最终源于突变）。

---

## 2. 自然选择 / Natural Selection

**核心机制（达尔文）**：
```
① 种群内存在遗传变异（来自突变等）
② 后代数量超过环境承载量 → 生存斗争（争夺有限资源）
③ 某些变异个体更适应环境（有选择优势）→ 选择压力作用
④ 适者生存并繁殖更多（differential survival & reproduction）
   → 将有利等位基因传给后代
⑤ 有利等位基因频率随世代增加 → 种群进化
```

**关键术语**：
- **选择压力（selection pressure）**：环境中影响生存的因素（捕食、竞争、疾病）
- **适合度（fitness）**：个体存活并成功繁殖的能力（不等于"最强壮"）
- **等位基因频率（allele frequency）**：某等位基因在基因库中的比例

> ⚠️ 变异是**随机**产生的；选择是**非随机**的。生物不会"为了需要"而产生变异（拉马克错误）。

## 3. 选择的三种类型 / Three Types of Selection

```
稳定选择 Stabilising      定向选择 Directional     分裂选择 Disruptive
   ___                       ___                    _     _
  /   \      →   /\         /   \     →    /\      / \   / \  →  /\  /\
 /     \       /  \        /     \        /  \    /   \_/   \   _/  \/  \_
偏好中间，         偏好一端，            偏好两端，
淘汰极端           平均值移动            淘汰中间
```

| 类型 | 作用 | 例子 |
|---|---|---|
| **稳定选择 Stabilising** | 偏好**中间**表型，淘汰极端 → 减少变异 | 人类出生体重（过大/过小风险高）|
| **定向选择 Directional** | 偏好**一个极端** → 表型均值移动 | 抗生素抗性、长颈鹿颈长 |
| **分裂选择 Disruptive** | 偏好**两个极端**，淘汰中间 → 可能导致物种形成 | 鸟喙大小（适应不同食物）|

## 4. Hardy-Weinberg 原理

用于计算种群的等位基因频率和基因型频率。

**等位基因频率**：
$$p + q = 1$$
（p = 显性等位频率，q = 隐性等位频率）

**基因型频率**：
$$p^2 + 2pq + q^2 = 1$$

| 项 | 含义 |
|---|---|
| **p²** | 显性纯合子（如 AA）频率 |
| **2pq** | 杂合子（如 Aa）频率 |
| **q²** | 隐性纯合子（如 aa）频率 = **隐性表型频率** |

**五个平衡假设**：
1. 大种群（无遗传漂变 genetic drift）
2. 无突变
3. 无迁移（无基因流）
4. 随机交配
5. 无自然选择

**计算套路**（最常考）：
```
已知隐性表型频率（= q²）
    ↓ 开平方
q = √(q²)
    ↓
p = 1 − q
    ↓
携带者频率 = 2pq；显性纯合 = p²
```

**示例**：某隐性遗传病发病率 1/10000。
- q² = 1/10000 = 0.0001
- q = √0.0001 = 0.01
- p = 1 − 0.01 = 0.99
- 携带者频率 2pq = 2 × 0.99 × 0.01 = **0.0198 ≈ 2%**

## 5. 物种形成 / Speciation

**物种（species）**：能相互交配并产生**可育后代**的群体。

物种形成需要**生殖隔离（reproductive isolation）**——两群体不再能产生可育后代。

### 5.1 异域物种形成（Allopatric）
```
地理屏障（山脉/海洋/河流）分隔种群
    ↓
两群体面临不同选择压力 + 独立突变 + 遗传漂变
    ↓
基因库逐渐分化
    ↓
再次相遇时已无法交配产生可育后代 → 新物种
```
例：达尔文雀（Galápagos finches）在不同岛屿分化。

### 5.2 同域物种形成（Sympatric）
- 无地理隔离，在**同一区域**内发生
- 机制：生殖隔离（交配时间不同、行为差异、**多倍体 polyploidy**）
- **多倍体**在植物中常见 → 瞬间产生生殖隔离

### 5.3 隔离机制
| 类型 | 例子 |
|---|---|
| 合子前（prezygotic）| 时间隔离、行为隔离、机械隔离、配子隔离 |
| 合子后（postzygotic）| 杂种不活、**杂种不育**（如骡子 = 马 × 驴）|

## 6. 人工选择 / Artificial Selection

人为选择具有理想性状的个体进行繁殖。

| | 自然选择 | 人工选择 |
|---|---|---|
| 选择者 | **环境**（选择压力）| **人类** |
| 目标 | 适应环境/繁殖成功 | 满足人类需求 |
| 例子 | 抗药性、保护色 | 高产作物、奶牛、犬种 |

**风险**：过度选择 → 基因库变窄 → 近交衰退（inbreeding depression）、抗病力下降。

---

## 7. 进化的证据 / Evidence for Evolution

- **可观察的进化**：抗生素抗性（细菌）、杀虫剂抗性（昆虫）——定向选择实例
- **分子证据**：DNA/蛋白质序列相似性（亲缘越近越相似）
- **化石记录**：过渡形态
- **同源结构**（homologous structures）：共同祖先

## 必背定义 / Must-know Definitions

| 概念 | 英文定义 | 中文 |
|---|---|---|
| Natural selection | *The process by which organisms better adapted to their environment survive and reproduce, passing on advantageous alleles.* | 自然选择 |
| Allele frequency | *The proportion of a particular allele among all alleles of that gene in a population.* | 等位基因频率 |
| Speciation | *The formation of a new species through reproductive isolation of populations.* | 物种形成 |
| Species | *A group of organisms that can interbreed to produce fertile offspring.* | 物种 |
