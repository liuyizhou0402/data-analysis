## 1. 神经元结构 / Neurone Structure

| 结构 | 功能 |
|---|---|
| 细胞体（cell body）| 含核与细胞器 |
| 树突（dendrites）| 接收信号 |
| 轴突（axon）| 传导冲动 |
| **髓鞘（myelin sheath）** | 施万细胞包裹，绝缘 → 加快传导 |
| **郎飞结（nodes of Ranvier）** | 髓鞘间隙，动作电位在此产生 |

**神经元类型**：感觉（sensory）、中间（relay/intermediate）、运动（motor）。

---

## 2. 静息电位 / Resting Potential（约 -70 mV）

膜内相对膜外为**负**（极化 polarised）。

**形成机制**：
```
① Na⁺/K⁺ 泵（主动运输）：每次泵 3 Na⁺ 出 + 2 K⁺ 进 → 膜外正电荷多
② 膜对 K⁺ 通透性高（K⁺ 漏出通道多）→ K⁺ 外流 → 内部更负
③ 大的有机阴离子留在膜内
→ 膜内净负电（-70 mV）
```

> ⚠️ 静息≠静止：Na⁺/K⁺ 泵**持续耗 ATP**维持离子梯度。

---

## 3. 动作电位 / Action Potential

```
电位 (mV)
+40 |        /\  ← ② 去极化峰值（Na⁺ 内流）
    |       /  \
  0 |      /    \
-55 |---- /阈值--\------ ③ 复极化（K⁺ 外流）
-70 |___/        \__/← ④ 超极化
    |  ①静息        ⑤ 恢复静息
         时间 →
```

| 阶段 | 离子运动 |
|---|---|
| ① 静息 | 极化 -70 mV，电压门控通道关 |
| ② 去极化（depolarisation）| 达阈值 → **电压门控 Na⁺ 通道开** → Na⁺ 涌入 → +40 mV |
| ③ 复极化（repolarisation）| Na⁺ 通道关，**K⁺ 通道开** → K⁺ 外流 → 回负 |
| ④ 超极化（hyperpolarisation）| K⁺ 过度外流 → 短暂低于 -70 mV |
| ⑤ 恢复 | Na⁺/K⁺ 泵恢复离子分布 |

### 3.1 全或无定律（All-or-Nothing）
- 刺激达阈值 → 产生**完整、固定幅度**的动作电位
- 刺激未达阈值 → 不产生
- **更强的刺激 → 更高的动作电位频率**（不是更大幅度）

### 3.2 不应期（Refractory Period）
- Na⁺ 通道失活期间，不能产生新动作电位
- 意义：①保证**单向传导** ②限制冲动频率

## 4. 冲动传导 / Conduction

### 4.1 局部电流
去极化区域引起相邻区域去极化 → 动作电位沿轴突传播。不应期确保**只向前传**。

### 4.2 跳跃式传导（Saltatory Conduction）
- 有髓鞘神经元：髓鞘绝缘，离子无法通过
- 动作电位**只在郎飞结**产生
- 冲动"跳跃"于结之间 → **传导速度大大提高**

**影响传导速度因素**：
| 因素 | 速度 |
|---|---|
| 有髓鞘 | 快（跳跃式）|
| 轴突直径大 | 快（电阻小）|
| 温度高 | 快（至酶变性前）|

## 5. 突触传递 / Synaptic Transmission

**化学突触**（以乙酰胆碱 ACh 为例）：
```
① 动作电位到达突触前膜
② 电压门控 Ca²⁺ 通道开 → Ca²⁺ 涌入突触前神经元
③ Ca²⁺ 触发突触小泡与前膜融合
④ 神经递质（ACh）经胞吐释放入突触间隙（synaptic cleft）
⑤ ACh 扩散，结合突触后膜的受体（配体门控 Na⁺ 通道）
⑥ Na⁺ 通道开 → 后膜去极化 → 可能引发新动作电位
⑦ 乙酰胆碱酯酶（AChE）分解 ACh → 终止信号，产物回收再利用
```

**突触的意义**：
- **单向传导**：递质只由前膜释放、受体只在后膜 → 信号只能前→后
- **整合（summation）**：多个兴奋性输入累加才能达阈值；抑制性突触可对抗
- **可塑性**：学习与记忆的基础

> 🔗 应用：有机磷农药/神经毒气抑制 AChE → ACh 堆积 → 持续刺激 → 肌肉痉挛。

## 6. 肌肉收缩 / Muscle Contraction

### 6.1 骨骼肌结构（横纹肌）
```
肌肉 → 肌纤维(muscle fibre) → 肌原纤维(myofibril) → 肌节(sarcomere)
```

**肌节（sarcomere）结构**：
```
  Z线 |---- I带 ----|---- A带 ----|---- I带 ----| Z线
      |  (细肌丝)   |(粗+细重叠)  |             |
              H区（仅粗肌丝）在 A 带中央
```
- **粗肌丝**：肌球蛋白（myosin）
- **细肌丝**：肌动蛋白（actin）+ 原肌球蛋白（tropomyosin）+ 肌钙蛋白（troponin）

### 6.2 滑动丝学说（Sliding Filament Theory）
```
① 神经冲动 → 神经肌肉接头 → ACh → 肌膜去极化
② 动作电位沿 T 小管 → 肌质网（sarcoplasmic reticulum）释放 Ca²⁺
③ Ca²⁺ 结合肌钙蛋白(troponin) → 原肌球蛋白移位 → 暴露 actin 上结合位点
④ 肌球蛋白头部结合 actin → 形成横桥(cross-bridge)
⑤ 横桥摆动(power stroke) → 细肌丝向肌节中央滑动（ADP+Pi 释放）
⑥ ATP 结合肌球蛋白头 → 横桥脱离；ATP 水解使头部复位 → 循环重复
⑦ 肌节缩短
```

**收缩时肌节变化**（重要考点）：
| 结构 | 收缩时变化 |
|---|---|
| 肌节长度 | **缩短** |
| I 带 | **变窄** |
| H 区 | **变窄** |
| **A 带** | **不变**（= 粗肌丝长度）|
| 肌丝本身长度 | **不变**（只是滑动重叠增加）|

### 6.3 能量供应
- ATP 直接供能（横桥摆动 + 脱离）
- **磷酸肌酸（creatine phosphate）**：快速再生 ATP（ADP + 磷酸肌酸 → ATP + 肌酸）
- 持续运动：有氧呼吸 / 无氧（乳酸）

## 7. 神经控制 vs 激素控制 / Nervous vs Hormonal

| | 神经控制 | 激素控制 |
|---|---|---|
| 信号 | 电信号 + 化学突触 | 化学（激素，血液运输）|
| 速度 | **快**（毫秒）| 慢（秒~天）|
| 持续时间 | 短暂 | 持久 |
| 作用范围 | 精确、局部 | 广泛 |
| 例子 | 反射、肌肉运动 | 血糖、生长、代谢 |

---

## 8. 植物的协调 / Coordination in Plants

**生长素（auxin / IAA）**：促进细胞伸长。

**向光性（phototropism）**：
```
单侧光照射
    ↓
生长素从向光侧重新分布到背光侧
    ↓
背光侧生长素浓度高 → 细胞伸长更快
    ↓
茎向光弯曲（正向光性）
```

**向地性（gravitropism）**：根正向地性（向下），茎负向地性（向上）。

## 必背定义 / Must-know Definitions

| 概念 | 英文定义 | 中文 |
|---|---|---|
| Resting potential | *The potential difference across a neurone membrane when not transmitting an impulse (about −70 mV).* | 静息电位 |
| Action potential | *A rapid, transient reversal of membrane potential caused by Na⁺ influx, propagated along an axon.* | 动作电位 |
| Synapse | *A junction between two neurones where a chemical neurotransmitter transmits the signal.* | 突触 |
| Sliding filament theory | *The model in which muscle contraction occurs as actin filaments slide over myosin filaments, shortening the sarcomere.* | 滑动丝学说 |
