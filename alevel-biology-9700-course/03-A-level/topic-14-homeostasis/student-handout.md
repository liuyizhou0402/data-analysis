## 1. 稳态与负反馈 / Homeostasis & Negative Feedback

**稳态（homeostasis）**：维持内环境（温度、血糖、水势、pH）在狭窄范围内稳定，以保证酶活性和细胞功能。

**负反馈回路（通用）**：
```
变量偏离设定点
    ↓
感受器 (receptor) 检测变化
    ↓
控制中心 (control centre)，如下丘脑
    ↓
效应器 (effector) 产生反应
    ↓
变量恢复设定点 → 反馈关闭反应
```

| | 负反馈 | 正反馈 |
|---|---|---|
| 效果 | 使变量**回到**设定点（稳定）| 使变量**进一步偏离**（放大）|
| 例子 | 体温、血糖、水分调节 | 分娩催产素、神经冲动去极化 |

**激素 vs 神经信号**：
- 激素（内分泌）：血液传递，**慢、持久、广泛**
- 神经：电信号，**快、短暂、精确**

## 2. 肾脏与排泄 / Kidney & Excretion

### 2.1 肾单位（nephron）结构

```
入球小动脉 → 肾小球(glomerulus) → 鲍曼囊(Bowman's capsule)
                                       ↓
近端小管(PCT) → 髓袢(loop of Henle) → 远端小管(DCT) → 集合管 → 尿
```

### 2.2 超滤（Ultrafiltration）

发生在肾小球-鲍曼囊：
- 入球小动脉**比出球小动脉粗** → 肾小球内**高静水压**
- 血浆经三层屏障过滤：**毛细血管内皮孔 → 基底膜 → 足细胞（podocytes）裂隙**
- **小分子**（水、葡萄糖、尿素、无机离子）滤入鲍曼囊 → 滤液
- **大分子**（血浆蛋白、血细胞）**留在血液**

> ⚠️ 超滤是**被动**的，由血压（静水压）驱动，**不耗 ATP**。

### 2.3 选择性重吸收（PCT）

近端小管（PCT）重吸收：**全部葡萄糖和氨基酸 + 大部分水和离子**。

机制（葡萄糖重吸收为例）：
```
① Na⁺/K⁺ 泵（基底膜侧）主动泵 Na⁺ 出细胞 → 建立低 Na⁺
② 管腔 Na⁺ 顺梯度进入 + 葡萄糖协同运输（共转运 co-transport）
③ 葡萄糖经易化扩散进入血液
```

**PCT 适应特征**：
- **微绒毛**（microvilli）→ 大表面积
- 大量**线粒体** → 供主动运输 ATP
- 紧密排列、富含转运蛋白

### 2.4 髓袢与尿浓缩（逆流倍增）

```
降支（descending limb）：透水、不透盐 → 水渗出 → 滤液浓缩
升支（ascending limb）：不透水、主动泵 Na⁺/Cl⁻ 出 → 髓质间隙高渗
        ↓
髓质形成渗透梯度（越深越高渗）
        ↓
集合管穿过高渗髓质 → 水可被重吸收（由 ADH 控制）
```

## 3. 渗透压调节（ADH）/ Osmoregulation

**ADH（抗利尿激素，antidiuretic hormone）**调节集合管对水的通透性。

**脱水（血液水势低）时**：
```
下丘脑渗透感受器检测水势↓
    ↓
垂体后叶释放 ADH↑
    ↓
ADH → 集合管细胞插入更多水通道蛋白（aquaporins）
    ↓
更多水被重吸收 → 尿少而浓 → 血液水势恢复
```

**多饮水（水势高）时**：ADH↓ → 集合管水通道减少 → 重吸收水减少 → **尿多而稀**。

> 🎯 ADH = "抗利尿" = **保水**（促进重吸收）。记反了就全错。

## 4. 血糖调节 / Blood Glucose Regulation

**胰岛（islets of Langerhans）**两种细胞：
- **β 细胞** → 分泌**胰岛素（insulin）** → **降**血糖
- **α 细胞** → 分泌**胰高血糖素（glucagon）** → **升**血糖

### 4.1 血糖升高（餐后）
```
β 细胞检测高血糖 → 分泌胰岛素
胰岛素作用于肝细胞和肌细胞：
  - 增加细胞摄取葡萄糖（插入 GLUT4 转运蛋白）
  - 促进糖原合成（glycogenesis）
  - 促进葡萄糖氧化（呼吸）
→ 血糖下降到正常
```

### 4.2 血糖降低（空腹/运动）
```
α 细胞检测低血糖 → 分泌胰高血糖素
胰高血糖素作用于肝细胞：
  - 糖原分解为葡萄糖（glycogenolysis）
  - 糖异生：由非糖物质合成葡萄糖（gluconeogenesis）
→ 血糖上升到正常
```

### 4.3 第二信使机制（cAMP）
胰高血糖素/肾上腺素结合细胞膜受体 → 激活腺苷酸环化酶 → ATP → **cAMP（第二信使）** → 激活蛋白激酶级联 → 糖原分解。

### 4.4 糖尿病（Diabetes mellitus）

| | I 型（Type 1）| II 型（Type 2）|
|---|---|---|
| 病因 | β 细胞被自身免疫破坏 → **无胰岛素** | 靶细胞**胰岛素抵抗**（受体不敏感）|
| 典型发病 | 儿童/青少年 | 成年/肥胖相关 |
| 治疗 | **注射胰岛素** | 饮食控制、运动、降糖药 |

## 5. 植物气孔调节 / Stomatal Control

### 5.1 保卫细胞开闭机制
```
气孔张开：
  保卫细胞主动泵入 K⁺ → 水势↓ → 渗透吸水 → 膨胀
  → 细胞壁不均匀加厚（内壁厚）使细胞向外弯 → 气孔开

气孔关闭：
  K⁺ 流出 → 水势↑ → 失水 → 保卫细胞松弛 → 气孔关
```

### 5.2 ABA（脱落酸）信号——干旱响应
```
缺水 → 根/叶产生 ABA（化学信号）
    ↓
ABA 结合保卫细胞受体 → 触发 K⁺ 和阴离子外流
    ↓
保卫细胞失水松弛 → 气孔关闭
    ↓
减少蒸腾失水（保护植物）
```

> 🔗 联系 Topic 7：气孔关闭减少蒸腾，是植物应对水分胁迫的稳态机制。

## 必背定义 / Must-know Definitions

| 概念 | 英文定义 | 中文 |
|---|---|---|
| Homeostasis | *The maintenance of a constant internal environment within narrow limits.* | 稳态 |
| Negative feedback | *A control mechanism in which a change in a variable triggers a response that returns the variable toward its set point.* | 负反馈 |
| Ultrafiltration | *Filtration of blood plasma under high hydrostatic pressure, separating small molecules from large ones (blood cells/proteins).* | 超滤 |
| Osmoregulation | *The control of water and solute balance to maintain a constant water potential of body fluids.* | 渗透调节 |
