## 课时 2.1（Lesson 6）— 水与无机离子 / Water & Inorganic Ions（60 min）

### 教学目标 / Learning objectives
学生在本课后能够：
- [ ] **Describe** 水分子的偶极结构，以及氢键的形成
- [ ] **Explain** 水的各项性质（良好溶剂、高比热容、内聚力/cohesion、高汽化潜热）如何支持生命
- [ ] **State** Ca²⁺、Na⁺、Cl⁻、Mg²⁺、Fe²⁺、PO₄³⁻ 各自在生物体内的作用

### 必备术语 / Key terms
| English | 中文 |
|---|---|
| dipole / polar molecule | 偶极分子 / 极性分子 |
| hydrogen bond | 氢键 |
| solvent / solute | 溶剂 / 溶质 |
| specific heat capacity | 比热容 |
| cohesion | 内聚力 |
| latent heat of vaporisation | 汽化潜热 |
| inorganic ion | 无机离子 |

### 授课流程 / Lesson flow
| 时间 | 环节 | 老师做什么 | 学生做什么 |
|---|---|---|---|
| 0–5′ | 导入 | 展示"水在生命中有哪些作用"思维导图空框，提问"如果地球没有水，最先消失的生命过程是什么？" | 自由讨论，填空框 |
| 5–20′ | 精讲 | 讲水的偶极结构（O 偏负、H 偏正）→ 氢键形成；画 H₂O 分子图，标明 δ＋/δ－；强调氢键是**分子间键**，比共价键弱、可断裂 | 画偶极图、标注 δ＋/δ－ |
| 20–38′ | 精讲 | 逐项讲 4 大性质：①溶剂（离子和极性分子可溶）→ 代谢意义；②高比热容（需大量热才升温）→ 体温稳定；③内聚力（分子间氢键，蒸腾拉力 cohesion–tension）→ 植物水运输；④高汽化潜热（蒸发散热效率高）→ 体温调节 | 填写"性质-原因-意义"三栏表 |
| 38–50′ | 精讲+练 | 讲六种无机离子（Ca²⁺→骨骼/肌肉收缩；Na⁺→动作电位/渗透；Cl⁻→渗透/胃酸；Mg²⁺→叶绿素的镁核；Fe²⁺→血红蛋白血红素；PO₄³⁻→ATP/核酸/磷脂）；学生当场做小配对 | 快速配对练习 |
| 50–60′ | 总结 | 锁定"性质→机制→意义"逻辑链，默写 4 大性质英文 | 默写 |

### 板书/重点 / Board work
- **水分子偶极图**（标 δ＋/δ－，画氢键虚线）。
- **性质-机制-意义三栏表**：
  | 性质 | 化学机制 | 生命意义（举例） |
  |---|---|---|
  | 良好溶剂 | 极性分子与离子可水合 | 溶解离子/代谢物，参与化学反应 |
  | 高比热容 | 打断大量氢键需多热量 | 体温稳定（血液/组织液缓冲温度变化） |
  | 内聚力 cohesion | 分子间氢键 | 蒸腾流中连续水柱（cohesion–tension） |
  | 高汽化潜热 | 蒸发时吸收大量热 | 排汗/蒸腾降温 |
- **无机离子速查**（见上文）。

### 易错点 / Common errors
- ❌ 说氢键"很强" → ✅ 氢键**弱于共价键**，但大量氢键累积可提供可观的结合力（如 DNA 双链、蛋白质二级结构）。
- ❌ 混淆"cohesion（水分子间）"与"adhesion（水与管壁间）" → ✅ 蒸腾拉力靠 cohesion 把水柱拉上去，adhesion 是辅助爬壁。
- ❌ 说 Mg²⁺ 在"线粒体中" → ✅ Mg²⁺ 是**叶绿素的中心离子**（在叶绿体中）；Fe²⁺ 才与血红蛋白相关。

### 提问设计 / Key questions
- "为什么汗液蒸发比加热水散热更有效？"（引出汽化潜热）
- "如果植物茎被切断，水柱会断裂吗？为什么水柱不容易断？"（内聚力的应用）

- `practice.md` Part A 第 1–2 题、Part B 第 1 题。

## 课时 2.2（Lesson 7）— 碳水化合物 I：单糖与二糖 / Carbohydrates I（60 min）

### 教学目标
- [ ] **Draw / Recognise** α-葡萄糖与 β-葡萄糖的结构（环状，标注 OH 方向）
- [ ] **State** 核糖（ribose）与脱氧核糖（deoxyribose）的结构差异
- [ ] **Describe** 缩合反应（condensation）形成糖苷键（glycosidic bond）；水解反应（hydrolysis）打断糖苷键
- [ ] **Describe** 还原糖 Benedict's 检测与非还原糖的区分（先水解再测）

### 必备术语
| English | 中文 |
|---|---|
| monosaccharide / disaccharide | 单糖 / 二糖 |
| α-glucose / β-glucose | α-葡萄糖 / β-葡萄糖 |
| glycosidic bond | 糖苷键 |
| condensation reaction | 缩合反应 |
| hydrolysis | 水解反应 |
| reducing sugar / non-reducing sugar | 还原糖 / 非还原糖 |
| Benedict's reagent | 本尼迪特试剂 |

### 授课流程
| 时间 | 环节 | 内容 |
|---|---|---|
| 0–5′ | 回顾 | 快问：水的 4 大性质、Fe²⁺ 的作用 |
| 5–22′ | 精讲 | 单糖：葡萄糖（六碳，C₆H₁₂O₆）、核糖（五碳）、脱氧核糖（五碳，C₂'位无 OH）；**α/β 葡萄糖关键区别**：C₁ 上 OH 的方向——α 在下（下方）、β 在上（上方），这决定了聚合后形成哪种多糖 |
| 22–38′ | 精讲 | 缩合反应：2 个单糖脱去 1 分子水 → 形成糖苷键（1,4-糖苷键为例）；水解反应：加水打断糖苷键；常见二糖：麦芽糖（α-葡萄糖+α-葡萄糖）、蔗糖（葡萄糖+果糖，**非还原糖**）、乳糖（葡萄糖+半乳糖） |
| 38–52′ | 实验演示/讨论 | **还原糖 Benedict's 检测**：蓝色→绿→黄→砖红（加热，砖红色沉淀）；**非还原糖**（蔗糖）：先加稀盐酸水解，再中和，再做 Benedict's，才出砖红；碘液：检测**淀粉**→蓝黑色；Biuret：检测蛋白质→紫色；乳化试验：脂质→白色乳浊 |
| 52–60′ | 总结 | 补完检测试剂表，背还原糖定义英文 |

### 板书/重点
- **α vs β 葡萄糖环状结构**（重点标注 C₁ 的 OH 方向）。
- **缩合/水解可逆反应方程式**：单糖 + 单糖 ⇌ 二糖 + H₂O
- **食物检测速查表**（见课时 2.6 汇总）。
- 缩合-水解反应的对称性图解（中间写 + H₂O / - H₂O）。

### 易错点 / Common errors
- ❌ 把 α/β 葡萄糖混淆——只记"α 是储能用的（形成淀粉），β 是结构用的（形成纤维素）"，而不理解 OH 方向 → ✅ α-glucose：C₁ 上的 −OH 在**环平面下方**；β-glucose：C₁ 上的 −OH 在**环平面上方**。
- ❌ 说"蔗糖是还原糖" → ✅ 蔗糖是**非还原糖**（糖苷键封闭了活性端），不与 Benedict's 直接反应。
- ❌ 非还原糖检测不"中和"酸就加 Benedict's → ✅ 酸性条件下 Benedict's 不工作，必须**先用 NaHCO₃ 中和**。

### 提问设计
- "α-葡萄糖和 β-葡萄糖的分子式相同，但聚合成多糖后性质截然不同，为什么？"（引出 C₁ OH 方向决定糖苷键取向，从而决定链的形状）

- `practice.md` Part A 第 3–4 题、Part D 第 1 题（检测实验专项）。

## 课时 2.3（Lesson 8）— 碳水化合物 II：多糖 / Polysaccharides（60 min）

### 教学目标
- [ ] **Compare** 淀粉（amylose + amylopectin）、糖原（glycogen）、纤维素（cellulose）的结构
- [ ] **Explain** 每种多糖的结构如何适应其功能（structure–function，本主题最重要考点）
- [ ] 理解 α1,4 / α1,6 / β1,4 糖苷键的位置与分支的意义

### 必备术语
| English | 中文 |
|---|---|
| amylose / amylopectin | 直链淀粉 / 支链淀粉 |
| glycogen | 糖原 |
| cellulose | 纤维素 |
| α-1,4-glycosidic bond | α-1,4-糖苷键 |
| α-1,6-glycosidic bond | α-1,6-糖苷键（分支点） |
| β-1,4-glycosidic bond | β-1,4-糖苷键 |
| microfibril | 微纤维 |
| hydrogen bond | 氢键（纤维素链间） |

### 授课流程
| 时间 | 环节 | 内容 |
|---|---|---|
| 0–5′ | 回顾 | α/β 葡萄糖区别快答；缩合反应产物 |
| 5–25′ | 精讲 | **淀粉**：amylose（α1,4 直链→螺旋状，紧凑，储能）+ amylopectin（α1,4 主链 + α1,6 分支，多端同时水解→快速释放）；功能：植物储能，不溶于水（减少渗透压）；遇碘变**蓝黑**（螺旋包碘分子） |
| 25–40′ | 精讲 | **糖原**：与 amylopectin 相似但**分支更多更密**，溶解度更高；功能：动物（肝/肌肉）储能，多分支→更多游离端→更快速释放葡萄糖（应急能源） |
| 40–55′ | 精讲 | **纤维素**：β1,4 连接→每隔一个葡萄糖翻转→直线长链→相邻链间大量**氢键**→平行束 = 微纤维（microfibril）→高抗张强度；功能：植物细胞壁结构支撑，水不能轻易溶解（坚韧）；不被大多数动物消化（无 β-葡萄糖苷酶） |
| 55–60′ | 总结 | 三者对比表（结构特征 vs 功能），强调"**结构决定功能**"逻辑 |

### 板书/重点
- **三种多糖对比表**（考试高频，必须熟练）：

| | 淀粉（amylose/amylopectin） | 糖原 Glycogen | 纤维素 Cellulose |
|---|---|---|---|
| 单体 | α-葡萄糖 | α-葡萄糖 | β-葡萄糖 |
| 糖苷键 | α1,4（直链）+ α1,6（分支） | α1,4 + α1,6 | β1,4 |
| 分支 | amylopectin 有，amylose 无 | 多（更密） | 无 |
| 链形状 | 螺旋（amylose）/ 树状 | 高度分支树状 | 直线长链 |
| 链间作用 | — | — | 大量氢键 → 微纤维 |
| 功能 | 植物储能 | 动物（肝/肌肉）储能 | 细胞壁结构支撑 |
| 溶解性 | 不溶（减少渗透压） | 相对不溶 | 不溶，坚韧 |

### 易错点 / Common errors
- ❌ 说"糖原就是淀粉" → ✅ 糖原分支**更多**，且是动物（非植物）的储能多糖。
- ❌ 说纤维素链间是"共价键" → ✅ 纤维素**链间**是**氢键**（链内是β1,4 共价糖苷键），氢键形成微纤维提供强度。
- ❌ 描述分支功能只说"有分支" → ✅ 要说"分支提供**更多游离端（non-reducing ends）**，可**同时被多个酶水解**，快速释放葡萄糖"。
- ❌ 说淀粉遇碘变红 → ✅ 变**蓝黑**色（amylose 螺旋包裹碘分子）。

### 提问设计
- "一个运动员在百米冲刺瞬间，肌肉细胞需要大量快速释放葡萄糖，糖原的哪个结构特点满足这一需求？"（多分支→多游离端→快速水解）
- "为什么人吃木头（纤维素）不消化？"（无 β-糖苷酶，引申反刍动物/白蚁共生菌）

## 课时 2.4（Lesson 9）— 脂质 / Lipids（60 min）

### 教学目标
- [ ] **Describe** 甘油三酯的形成（甘油 + 3 脂肪酸，酯键，缩合）
- [ ] **Distinguish** 饱和与不饱和脂肪酸（双键位置与数量）
- [ ] **Describe** 磷脂的结构（亲水头+疏水尾）及双层膜形成的原理
- [ ] **Explain** 脂质作为储能/隔热/保护功能的化学依据
- [ ] **Describe** 脂质的乳化试验（emulsion test）操作与结果

### 必备术语
| English | 中文 |
|---|---|
| triglyceride | 甘油三酯 |
| glycerol | 甘油 |
| fatty acid | 脂肪酸 |
| ester bond | 酯键 |
| saturated / unsaturated fatty acid | 饱和 / 不饱和脂肪酸 |
| phospholipid | 磷脂 |
| hydrophilic / hydrophobic | 亲水性 / 疏水性 |
| emulsion test | 乳化试验 |

### 授课流程
| 时间 | 环节 | 内容 |
|---|---|---|
| 0–5′ | 回顾 | 多糖三者对比（淀粉/糖原/纤维素）快问 |
| 5–18′ | 精讲 | **甘油三酯**：甘油（3 个—OH）+ 3 脂肪酸（各含—COOH）→ 3 次缩合 → 3 个**酯键（ester bond）**+ 3H₂O；脂肪酸：碳链长，饱和（无双键，直链，固态）vs 不饱和（有 C=C 双键，弯链，液态）；每克脂肪含能量 > 每克糖（更多 C−H 键）|
| 18–35′ | 精讲 | **磷脂**：甘油 + 2 脂肪酸 + 磷酸基团（取代第 3 个脂肪酸）；磷酸头（极性，**亲水**）+ 2 脂肪酸尾（非极性，**疏水**）；在水中自发形成**双层（bilayer）**：疏水尾相向、亲水头朝水——此即细胞膜磷脂双层的化学基础（引出 Topic 4） |
| 35–50′ | 精讲+实验讨论 | **脂质功能**：①储能（单位质量能量高，不溶水减少渗透压）②隔热（皮下脂肪绝热层）③机械保护（内脏周围脂肪垫）④激素前体（胆固醇→固醇类激素）；**乳化试验**：样品 + 乙醇（溶解脂质）→ 取澄清液加等体积水 → 白色乳浊液（阳性）；阴性：保持透明 |
| 50–60′ | 总结 | 类比：甘油三酯 vs 磷脂（结构差异表），锁定酯键英文 |

### 板书/重点
- **甘油三酯形成示意**（标注 3 个酯键、脱去 3H₂O）。
- **磷脂结构图**（标注亲水头 phosphate、疏水尾 fatty acids）→ 双层膜自组装。
- **甘油三酯 vs 磷脂对比**：
  | | 甘油三酯 Triglyceride | 磷脂 Phospholipid |
  |---|---|---|
  | 甘油 | 1 分子 | 1 分子 |
  | 脂肪酸 | 3 条 | 2 条 |
  | 第 3 基团 | 脂肪酸 | 磷酸基团 |
  | 极性 | 非极性（整体疏水） | 两亲性（amphipathic） |
  | 主要功能 | 储能、隔热 | 形成膜双层 |

### 易错点 / Common errors
- ❌ 混淆"酯键（ester bond）"与"糖苷键（glycosidic bond）" → ✅ 脂质用酯键（—COOH + —OH），糖类用糖苷键（—OH + —OH，脱水）。
- ❌ 说磷脂"是脂肪" → ✅ 磷脂和脂肪都属于脂质，但磷脂有**两亲性**，脂肪（甘油三酯）整体疏水。
- ❌ 乳化试验漏关键步骤 → ✅ 必须：①先用**乙醇**溶解脂质，②取**澄清层**，③加等体积**水**，④观察白色乳浊液。

### 提问设计
- "鸟类迁徙前储存大量脂肪而不是糖原，从化学上分析，脂肪的哪些特性使其更适合作为长途飞行的能源储备？"（单位质量能量高、质量轻、不溶水无渗透压）

## 课时 2.5（Lesson 10）— 蛋白质 I：结构层次 / Protein Structure（60 min）

### 教学目标

### 必备术语
| English | 中文 |
|---|---|
| amino acid | 氨基酸 |
| amino group (−NH₂) | 氨基 |
| carboxyl group (−COOH) | 羧基 |
| R group (variable/side chain) | R 基（侧链）|
| peptide bond | 肽键 |
| polypeptide chain | 多肽链 |
| primary structure | 一级结构 |
| secondary structure | 二级结构 |
| α-helix | α 螺旋 |
| β-pleated sheet (β-strand) | β 折叠 |
| tertiary structure | 三级结构 |
| disulfide bond | 二硫键 |
| quaternary structure | 四级结构 |

### 授课流程
| 时间 | 环节 | 内容 |
|---|---|---|
| 0–5′ | 回顾 | 磷脂结构快答；乳化试验步骤 |
| 5–20′ | 精讲 | **氨基酸通用结构**（必画）：中心碳连 −NH₂、−COOH、−H、R 基；20 种天然氨基酸 R 基不同；**肽键形成**：一个氨基酸的 −COOH 与另一个的 −NH₂ 缩合，脱水，形成 −CO−NH− 肽键；n 个氨基酸 → (n-1) 个肽键 |
| 20–38′ | 精讲 | **一、二级结构**：①一级结构 = 氨基酸序列（共价肽键）；②二级结构：**α-helix**（氢键在同一链的 −C=O 与 −N−H 间形成，螺旋）；**β-pleated sheet**（氢键在相邻平行链之间）——共同点：**氢键维持** |
| 38–52′ | 精讲 | **三、四级结构**：③三级结构：多肽链进一步折叠成特定三维形状；维持键：**氢键、离子键（ionic bond）、二硫键（disulfide bond，−S−S−，两个半胱氨酸 Cys）、疏水相互作用（hydrophobic interactions）**；④四级结构：两条以上**多肽亚基**（subunit）通过非共价键（主要氢键 + 离子键）聚合，如血红蛋白 4 亚基 |
| 52–60′ | 总结 | 默写四级结构层次与对应键类型 |

### 板书/重点
- **氨基酸通用结构图**（必画，标四个基团）。
- **肽键形成图**（标−CO−NH−，注脱水方向）。
- **四级结构与键的类型汇总**：
  | 结构层次 | 内容 | 维持的化学键 |
  |---|---|---|
  | 一级 Primary | 氨基酸排列顺序 | **肽键（共价）** |
  | 二级 Secondary | α 螺旋 / β 折叠 | **氢键** |
  | 三级 Tertiary | 三维折叠形状 | 氢键、离子键、**二硫键（共价）**、疏水作用 |
  | 四级 Quaternary | 多亚基聚合 | 氢键、离子键、疏水作用 |

### 易错点 / Common errors
- ❌ 说"所有蛋白质都有四级结构" → ✅ 只有**两条以上多肽亚基**才有四级结构（单链蛋白无四级结构）。
- ❌ 混淆"二硫键在哪一级" → ✅ 二硫键（−S−S−）维持**三级结构**（连接同一链内远端或不同链的侧链间）；**不是**维持一级或二级结构的。
- ❌ 说"R 基相同的氨基酸也是不同种" → ✅ R 基完全相同则是同一种氨基酸；20 种氨基酸 R 基各不相同。
- ❌ 遗漏"疏水相互作用（hydrophobic interactions）"只写三种键 → ✅ 三级结构的维持力有**四类**（氢键/离子键/二硫键/疏水作用）。

### 提问设计

### 板书/重点
- **球状 vs 纤维状对比表**（必考成对比较）：
  | | 球状 Globular | 纤维状 Fibrous |
  |---|---|---|
  | 形状 | 紧密折叠成球形 | 平行长链 |
  | 溶解性 | 可溶（极性 R 基朝外） | 不溶 |
  | 三级结构 | 高度折叠、精确 | 相对规则重复 |
  | 功能 | 酶、激素、运输（代谢性） | 结构支撑（机械性） |
  | 实例 | 血红蛋白、胰岛素 | 胶原蛋白、角蛋白 |

- **食物检测完整汇总表**（本主题最重要实验表）：
  | 检测物 | 试剂 | 操作 | 阳性结果 | 阴性结果 |
  |---|---|---|---|---|
  | 还原糖 | Benedict's 试剂 | **加热（水浴约 80 °C，5 min）** | 砖红色沉淀（brick-red precipitate）| 蓝色 |
  | 非还原糖 | ①稀HCl水解 ②NaHCO₃中和 ③Benedict's | 加热 | 砖红色 | 蓝色 |
  | 淀粉 | 碘液（碘/碘化钾溶液）| 直接滴加，无需加热 | 蓝黑色 | 橙黄/棕色 |
  | 蛋白质 | Biuret（NaOH + 稀 CuSO₄）| 先加 NaOH，再加 CuSO₄ | 紫色 | 蓝色 |
  | 脂质 | 乳化试验（乙醇→水）| 溶于乙醇后加水 | 白色乳浊液 | 透明 |

### 易错点 / Common errors
- ❌ 描述血红蛋白说"含铁" → ✅ 精确说"血红素中含 **Fe²⁺（二价铁）**，每个 Fe²⁺ 结合 1 分子 O₂"。
- ❌ 说胶原是球状蛋白 → ✅ 胶原是**纤维状**（fibrous）蛋白，具三股螺旋结构。
- ❌ Biuret 试验加错顺序（先加 CuSO₄）→ ✅ 必须**先加 NaOH 建立碱性环境**，再加稀 CuSO₄；顺序错则结果不稳定。
- ❌ 还原糖检测说"颜色变化"不具体 → ✅ 标准答法：**蓝色 → 绿色 → 黄色 → 砖红色（brick-red precipitate）**，颜色深浅与还原糖浓度正相关。
