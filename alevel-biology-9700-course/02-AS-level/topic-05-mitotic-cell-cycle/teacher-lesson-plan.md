## 课时 5.1 — 染色体结构与细胞周期 / Chromosomes & Cell Cycle（60 min）
> 对应 Lesson 21

### 教学目标 / Learning objectives
学生在本课后能够：
- [ ] **Describe** 染色体结构：DNA→组蛋白→染色质→染色体；姐妹染色单体在着丝粒相连
- [ ] **State** 同源染色体、二倍体、单倍体的含义
- [ ] **Outline** 细胞周期各阶段（G1、S、G2、M）及各期特征
- [ ] **Explain** S 期 DNA 复制后染色单体数量翻倍

### 必备术语 / Key terms
| English | 中文 |
|---|---|
| chromosome | 染色体 |
| chromatin | 染色质 |
| histone | 组蛋白 |
| sister chromatids | 姐妹染色单体 |
| centromere | 着丝粒 |
| homologous chromosomes | 同源染色体 |
| diploid / haploid | 二倍体 / 单倍体 |
| cell cycle | 细胞周期 |
| interphase | 间期 |
| G1 phase (Gap 1) | 第一生长期 |
| S phase (Synthesis) | DNA 合成期 |
| G2 phase (Gap 2) | 第二生长期 |
| mitosis (M phase) | 有丝分裂期 |
| cytokinesis | 胞质分裂 |

### 授课流程 / Lesson flow
| 时间 | 环节 | 老师做什么 | 学生做什么 |
|---|---|---|---|
| 0–5′ | 导入 | 展示一张洋葱根尖染色显微图，问"这些深色条状物是什么？从哪里来？" | 讨论、猜测 |
| 5–20′ | 精讲（染色体结构） | 从 DNA 双螺旋开始，逐层讲：① DNA 缠绕 histone 形成核小体（nucleosome）→②核小体串连成染色质（chromatin）→③染色质螺旋压缩成染色体。强调：间期（interphase）DNA 呈松散染色质形态（不可见），M 期高度螺旋凝缩才可见 | 画层次图（DNA→nucleosome→chromatin→chromosome） |
| 20–30′ | 精讲（染色单体与着丝粒） | 讲"复制之后"：同一条染色体复制产生**两条姐妹染色单体（sister chromatids）**在**着丝粒（centromere）**相连 → 此时仍算"1 条染色体、2 条染色单体"；区分 homologous chromosomes（来自父/母各一条，含同一基因位点）；二倍体 2n vs 单倍体 n | 完成染色体结构标注图 |
| 30–45′ | 精讲（细胞周期） | 画圆形细胞周期图，标注：G1（生长合成蛋白，时间最长）→S（DNA 复制，精确一次）→G2（继续生长，为 M 期准备）→M（有丝分裂+胞质分裂，时间最短）。强调：间期占约 90% 时间。演示如何由"有丝分裂指数低"推断"大多数细胞处于间期" | 在图上标注各期比例特征 |
| 45–55′ | 真题型练习 | 放一道"下图为细胞周期图，标出 S 期 / 用 compare 说出间期与分裂期区别"类题；学生作答 | 限时作答 |
| 55–60′ | 总结 | 锁定：染色体 = 1 DNA 分子；复制后 1 染色体 = 2 姐妹染色单体 + 着丝粒；细胞周期各期顺序与特征 | 默写细胞周期 4 阶段英文名 |

### 板书/重点 / Board work
- **染色体结构层次图**：
  ```
  DNA 双螺旋
    ↓ 缠绕 histone
  核小体 nucleosome
    ↓ 串连
  染色质 chromatin（间期状态）
    ↓ 螺旋压缩
  染色体 chromosome（M 期可见）
  ```
- **复制后染色体示意**：
  | 名称 | 数量规则 |
  |---|---|
  | 染色体（chromosome）| 着丝粒分裂前保持不变 |
  | 姐妹染色单体（sister chromatids）| S 期复制后 = 染色体数 × 2 |
  | DNA 分子数 | = 染色单体数（或复制前 = 染色体数）|

- **细胞周期时间比例**：G1 最长 → S → G2 → M 最短（M 约占 10%，间期约 90%）。

### 易错点 / Common errors
- ❌ 认为"分裂期（M 期）= 全部细胞周期" → ✅ 间期（G1+S+G2）占绝大部分，M 期只是一小部分。
- ❌ 混淆"染色体数"与"染色单体数"→ ✅ S 期后：1 条染色体 = 2 条姐妹染色单体，**染色体数不变，染色单体数翻倍**。
- ❌ 把 centromere（着丝粒）写成 centriole（中心粒）→ ✅ 完全不同：着丝粒是染色体上连接两条染色单体的位置；中心粒是形成纺锤体的细胞器。
- ❌ 说"G1 期 DNA 复制" → ✅ DNA 复制发生在 **S 期**（Synthesis）。

### 提问设计 / Key questions
- "S 期之前一个体细胞有 46 条染色体（人），S 期后染色体数是多少？染色单体数是多少？"（引导出：46 条染色体，92 条染色单体，92 个 DNA 分子）
- "为什么在显微镜下观察细胞分裂时，大多数细胞看起来处于间期？"（联系细胞周期时间比例与 mitotic index）

- `practice.md` Part A 第 1–3 题；Part B 第 1 题（细胞周期结构题）。

## 课时 5.2 — 有丝分裂各期 / Stages of Mitosis（60 min）
> 对应 Lesson 22

### 教学目标 / Learning objectives
学生在本课后能够：
- [ ] **Describe** 有丝分裂前期、中期、后期、末期的主要事件
- [ ] **Identify** 显微图中各分裂期的染色体特征
- [ ] **Calculate** mitotic index 并**Explain** 高 mitotic index 的意义
- [ ] **Sketch** 细胞周期中染色体/DNA 含量变化曲线的大致形状

### 必备术语 / Key terms
| English | 中文 |
|---|---|
| prophase | 前期 |
| metaphase | 中期 |
| anaphase | 后期 |
| telophase | 末期 |
| spindle (spindle fibres) | 纺锤体（纺锤丝） |
| centriole | 中心粒（动物细胞） |
| equator / metaphase plate | 赤道板 |
| spindle fibre / microtubule | 纺锤丝/微管 |
| nuclear envelope | 核膜 |
| mitotic index | 有丝分裂指数 |

### 授课流程 / Lesson flow
| 时间 | 环节 | 老师做什么 | 学生做什么 |
|---|---|---|---|
| 0–5′ | 回顾 | 快问：细胞周期各期英文名、S 期后染色单体数变化 | 口答 |
| 5–22′ | 精讲（PMAT 各期） | 按前→中→后→末顺序逐一讲解，每期给出"染色体状态 + 关键事件"，配显微图/示意图：**Prophase**（染色体凝缩可见；纺锤体开始形成；中心粒向两极移动（动物）；核膜开始解体）→**Metaphase**（染色体排列在赤道板；着丝粒连于纺锤丝；此期染色体最清晰，最适合计数）→**Anaphase**（着丝粒分裂；姐妹染色单体被拉向两极；两极各有一套完整染色体；注意：此时染色体数目**暂时加倍**）→**Telophase**（染色体解螺旋到染色质；核膜重建；纺锤体消失；随后胞质分裂 cytokinesis） | 在 PMAT 表中填事件 |
| 22–35′ | 识图练习 | 给一组 4 张有丝分裂各期显微图（乱序），要求学生辨认并说明判断依据 | 配对识别，说判断理由 |
| 35–48′ | 精讲 + 练习（mitotic index） | 讲公式：**Mitotic index = 处于有丝分裂期（M 期）的细胞数 / 细胞总数**。讲意义：指数越高 → 细胞分裂越旺盛（如胚胎组织、肿瘤）。示范一道计算题 → 学生独立做 1 道数据题 | 计算并解释 |
| 48–57′ | 精讲（DNA/染色体含量变化） | 画简化坐标图（横轴：细胞周期各期；纵轴：DNA 分子数或染色体数），重点标出：S 期 DNA 量加倍；anaphase 着丝粒分裂时染色体数目加倍（姐妹染色单体变成独立染色体）；M 期末（胞质分裂后）DNA/染色体量回到 2n | 描绘草图 |
| 57–60′ | 总结 | 锁定 mitotic index 公式 + PMAT 各期关键词 | 默写公式 |

### 板书/重点 / Board work

**PMAT 事件对照表**：
| 时期 | 主要事件（英文关键词） |
|---|---|
| Prophase 前期 | Chromosomes **condense** (become visible); spindle begins to form; **nuclear envelope breaks down** |
| Metaphase 中期 | Chromosomes align at **equator** (metaphase plate); centromeres attached to spindle fibres |
| Anaphase 后期 | **Centromeres split**; sister chromatids pulled to **opposite poles** by spindle fibres |
| Telophase 末期 | Chromosomes decondense back to chromatin; **nuclear envelope re-forms**; cytokinesis follows |

**Mitotic Index 公式**：
$$\text{Mitotic index} = \frac{\text{Number of cells in mitosis}}{\text{Total number of cells}}$$

**染色体 / DNA 含量关键转折点**：
- S 期结束：DNA 量 × 2；染色单体数 × 2；染色体数**不变**
- Anaphase（着丝粒分裂）：染色体数暂 × 2；染色单体数归 0
- 胞质分裂后：两个子细胞，每个染色体数 = 2n

### 易错点 / Common errors
- ❌ 混淆 prophase 与 anaphase 时染色体行为 → ✅ **Prophase**：染色体**凝缩显现**；**Anaphase**：着丝粒**分裂**、染色单体**拉向两极**。
- ❌ 认为"后期（anaphase）染色体数目不变" → ✅ 后期着丝粒分裂后，每条染色单体成为独立染色体，**染色体数暂时加倍为 4n**（随后胞质分裂各回 2n）。
- ❌ mitotic index 写成百分比而忘记它是比值/小数 → ✅ 可以是小数（如 0.15）或百分比（15%），**注意题目要求单位**。
- ❌ 描述 metaphase 时说"染色体移向赤道板" → ✅ 是"染色体**排列**在赤道板（align at the equator）"；是**anaphase**时才有"移向两极"。

### 提问设计 / Key questions
- "在一张根尖显微图中，你数到 200 个细胞，其中 30 个处于分裂期，mitotic index 是多少？该组织分裂活跃吗？"
- "如果实验药物阻断了纺锤体形成，细胞会停在哪一期？为什么？"（→ metaphase，因为纺锤丝缺失 → 着丝粒无法被拉开 → anaphase 无法进行）

| mutation | 基因突变 |
| proto-oncogene / tumour suppressor gene | 原癌基因 / 抑癌基因 |
| carcinogen | 致癌物 |

### 授课流程 / Lesson flow
| 时间 | 环节 | 老师做什么 | 学生做什么 |
|---|---|---|---|
| 0–5′ | 回顾 | 抽一道 mitotic index 计算，PMAT 快问 | 口答/板演 |
| 5–20′ | 精讲（有丝分裂意义） | ①**产生遗传上相同（genetically identical）的子细胞**：原因是 S 期 DNA 精确半保留复制 + M 期染色体精确均等分配，每个子细胞获得完整 2n 套。②**意义**：**生长（growth）**——多细胞生物体积增大；**修复/替换（repair and replacement）**——皮肤脱落、受伤愈合；**无性生殖（asexual reproduction）**——植物走茎、分裂生殖等。引导学生思考"为什么必须是遗传相同" | 记笔记，举无性生殖实例 |
| 20–38′ | 精讲（癌症） | 讲癌症发生链：**正常 DNA → 致癌物（carcinogen）/辐射/病毒诱发 → 突变（mutation）→ 原癌基因激活/抑癌基因失活 → 细胞周期检查点失效 → 不受控分裂（uncontrolled cell division）→ 肿瘤（tumour）**。区分**良性（benign）**：局限、有包膜、不转移；**恶性（malignant）**：侵入周围组织，癌细胞进入血/淋巴，扩散到远处形成新肿瘤（**metastasis 转移**）。常见致癌物举例：紫外线、石棉、黄曲霉毒素、烟草烟雾 | 画因果链图 |
| 38–48′ | 简介（端粒与干细胞） | **端粒（telomere）**：染色体末端重复 DNA 序列，保护染色体末端不被降解，每次 DNA 复制后缩短，与细胞衰老有关；癌细胞中端粒酶（telomerase）活跃 → 端粒不缩短 → 无限分裂。**干细胞（stem cell）**：具有自我更新能力、可分化为多种细胞类型的未分化细胞；受精卵/早期胚胎（全能干细胞）vs 组织干细胞（多能）。联系：骨髓干细胞持续有丝分裂提供血细胞 | 做笔记 |
| 48–57′ | 实验介绍（根尖压片） | 讲述根尖压片（garlic / onion）观察细胞分裂的步骤：①取 1–2 cm 幼根尖；②盐酸处理（60 °C）使细胞分离（maceration）；③水洗；④醋酸洋红（aceto-carmine）或龙胆紫（crystal violet）染色（使染色体着色）；⑤置于载玻片，盖盖玻片后用拇指轻压（squashing）使细胞分开成单层；⑥高倍镜观察，识别各分裂期。强调安全：盐酸腐蚀性，需佩戴护目镜 | 记实验步骤，提问 |
| 57–60′ | 总结 + 本主题整合 | 重申三大意义、癌症关键词；分发 practice.md，说明 Part D 考根尖压片 | 列出错题待补 |

### 板书/重点 / Board work
**癌症发生因果链**：
```
正常细胞
  ↓ 致癌物 / 辐射 / 致癌病毒 (carcinogen / radiation / oncogenic virus)
基因突变 (mutation)
  ↓ 原癌基因激活 (proto-oncogene → oncogene)
    + 抑癌基因失活 (tumour suppressor gene inactivated)
细胞周期失控 (loss of cell cycle control)
  ↓
不受控分裂 (uncontrolled cell division)
  ↓
肿瘤 (tumour)
  ↓ (若恶性)
转移 (metastasis) → 扩散至其他器官
```

**良性 vs 恶性肿瘤**：
| | 良性 Benign | 恶性 Malignant |
|---|---|---|
| 生长 | 缓慢，局限 | 快速，侵袭周围 |
| 转移 | 无 | 有（metastasis） |
| 包膜 | 有 | 通常无 |
| 危险性 | 低（取决于位置） | 高 |

### 易错点 / Common errors
- ❌ 说有丝分裂"产生遗传多样性" → ✅ 有丝分裂产生**遗传上相同（genetically identical）**的细胞；遗传多样性靠**减数分裂**（Topic 16）。
- ❌ 只说"突变导致癌症" → ✅ 要说**细胞周期失控**（uncontrolled cell division），突变是原因，结果是**不受控分裂**形成**肿瘤**。
- ❌ 混淆 proto-oncogene（原癌基因，正常时控制细胞生长，突变后成为促癌基因 oncogene）与 tumour suppressor gene（正常时抑制细胞分裂，突变后失去刹车功能）→ ✅ 癌症需二者同时异常。
- ❌ 描述 metastasis 仅说"扩散" → ✅ 应说：**癌细胞通过血液或淋巴系统扩散（enter blood / lymph）到身体其他部位，形成新肿瘤（secondary tumour）**。

### 提问设计 / Key questions
- "为什么用盐酸处理根尖有助于观察染色体？"（→ 酸水解使细胞间质分解，细胞分离为单层，有助于压片展开染色体）
- "科学家发现某类癌症细胞中端粒酶活性极高，这与癌症的哪个特征有关？"（→ 端粒不缩短 → 细胞不衰老 → 持续无限分裂）
