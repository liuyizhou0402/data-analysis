## 1. 染色体结构 / Chromosome Structure

### 1.1 从 DNA 到染色体的层次结构

生命的遗传信息存在 DNA 中，但 DNA 要塞进一个微小细胞核里，需要经过多级压缩：

```
① DNA 双螺旋
    ↓ DNA 缠绕在 histone（组蛋白）蛋白八聚体上
② 核小体 nucleosome（DNA + histone = 基本包装单位）
    ↓ 核小体串连 + 进一步折叠螺旋
③ 染色质 chromatin（间期状态：松散，光镜下不可见）
    ↓ 高度螺旋压缩（M 期发生）
④ 染色体 chromosome（高度凝缩，光镜下清晰可见）
```

> 关键理解：在**间期（interphase）**，DNA 以松散**染色质**形态存在，为转录提供方便；进入**M 期**时才高度凝缩为染色体。

### 1.2 染色体、姐妹染色单体与着丝粒

- **DNA 复制（S 期）之后**：原来一条染色体变成由**着丝粒（centromere）**相连的**两条姐妹染色单体（sister chromatids）**。
- **注意**：此时仍然算**1 条染色体**（2 条染色单体，用 1 个着丝粒相连）。

**计数规则**（S 期复制后，M 期着丝粒分裂前）：
| 指标 | 数量 |
|---|---|
| 染色体数（chromosome number） | 2n（不变） |
| 姐妹染色单体数（chromatid number） | 4n（翻倍） |
| DNA 分子数 | 4n（= 染色单体数） |

> ⚠️ 关键转折：**Anaphase（后期）着丝粒分裂**后，每条染色单体成为独立染色体 → 染色体数**暂时加倍为 4n**，染色单体数归 0（不再有"姐妹关系"）。

### 1.3 同源染色体与二倍体/单倍体
- **同源染色体（homologous chromosomes）**：形态大小相同，携带控制同一性状的等位基因（一条来自父本，一条来自母本）。
- **二倍体（diploid，2n）**：细胞含两套染色体（体细胞）。
- **单倍体（haploid，n）**：细胞含一套染色体（配子，如精/卵）。

## 2. 细胞周期 / The Cell Cycle

细胞周期 = 从一次细胞分裂结束到下一次分裂结束之间的全过程。

### 2.1 各期特征

| 阶段 | 英文全称 | 主要事件 | 相对时长 |
|---|---|---|---|
| G1 期 | Gap 1 / First growth phase | 细胞生长，合成蛋白质和细胞器 | **最长**（约 40% 周期） |
| S 期 | Synthesis phase | **DNA 半保留复制**（精确一次）；组蛋白合成 | 中等 |
| G2 期 | Gap 2 / Second growth phase | 继续生长；为 M 期准备（复制中心粒等） | 较短 |
| M 期 | Mitosis phase | **有丝分裂 + 胞质分裂** | **最短**（约 10%） |

> 🎯 **重要规律**：**间期（Interphase = G1 + S + G2）**约占细胞周期的 **90%**，M 期仅约 10%。因此在显微图中，**绝大多数细胞处于间期**（interphase 细胞核膜完整，看不到清晰染色体）。这就是为什么 mitotic index 通常远小于 1。

### 2.2 细胞周期示意图

```
              G1（最长）
           /              \
  胞质分裂              S（DNA 复制）
  cytokinesis             |
           \              /
        M 期    ←    G2（较短）
     （最短，约10%）
```

## 3. 有丝分裂各期 / Stages of Mitosis（PMAT）

> 记忆口诀：**P**rophase（前）→ **M**etaphase（中）→ **A**naphase（后）→ **T**elophase（末）= **PMAT**（也叫 IPMAT，含间期 Interphase）

### 3.1 前期 Prophase

**关键事件**：
1. 染色体**凝缩（condense）**，在光镜下可见，每条染色体由两条姐妹染色单体组成
2. **纺锤体（spindle）**开始形成（由微管/纺锤丝组成）
3. 动物细胞：**中心粒（centrioles）**向细胞两极移动，组织纺锤体
4. **核膜（nuclear envelope）解体（breaks down）**

> 识图特征：染色体可见，散乱分布于细胞中；核膜已不清晰。

### 3.2 中期 Metaphase

**关键事件**：
1. 染色体排列在细胞**赤道板（equator / metaphase plate）**
2. 每条染色体的**着丝粒与纺锤丝（spindle fibres）**相连
3. 此期染色体**最短最粗，最清晰**——最适合计数染色体

> 识图特征：染色体整齐排列在中央一行（侧面观）或散开放射状（顶面观）。

### 3.3 后期 Anaphase

**关键事件**：
1. **着丝粒分裂（centromeres split）**——这是后期的标志！
2. 姐妹染色单体被纺锤丝**拉向细胞两极（pulled to opposite poles）**
3. 每条原姐妹染色单体现在成为一条**独立染色体**
4. 此时细胞内染色体数目**暂时为 4n**（每极 2n）

> 识图特征：细胞明显拉长；两组染色体向两端运动，呈 V 字或 J 字形。

### 3.4 末期 Telophase

**关键事件**：
1. 染色体**解螺旋（decondense）**回到染色质状态，逐渐消失
2. **核膜重建（nuclear envelope re-forms）**在每极染色体周围
3. 纺锤体消失
4. 随后**胞质分裂（cytokinesis）**：动物细胞——细胞膜向内收缩（cleavage furrow）；植物细胞——形成新细胞板（cell plate）→ 新细胞壁

> 识图特征：两端各出现一个核，胞质开始缢裂。

### 3.5 PMAT 一眼识别速查表

| 时期 | 识图一眼看到 |
|---|---|
| Prophase | 染色体可见，散乱，**无核膜**，无赤道排列 |
| Metaphase | 染色体整齐排在**中央**（赤道板） |
| Anaphase | 两组染色体**向两极移动**，细胞被拉长 |
| Telophase | 两端各一核，染色体**消失/模糊**，胞质开始缢裂 |

## 4. 有丝分裂指数 / Mitotic Index

### 4.1 公式

$$\text{Mitotic index} = \frac{\text{Number of cells in mitosis (M phase)}}{\text{Total number of cells observed}}$$

$$\text{有丝分裂指数} = \frac{\text{处于有丝分裂期（M 期）的细胞数}}{\text{观察到的细胞总数}}$$

### 4.2 计算示范

**例题**：在一张洋葱根尖显微图中，共观察到 400 个细胞，其中 60 个处于 M 期（可见清晰染色体）。计算 mitotic index 并说明该组织的分裂活性。

**解题步骤**：
1. 识别各量：处于 M 期的细胞数 = 60；细胞总数 = 400
2. 代入公式：Mitotic index = 60 ÷ 400 = **0.15**（或 15%）
3. 解释：Mitotic index 为 0.15，说明约 15% 的细胞处于分裂状态，**该组织分裂较活跃**（如根尖分生区）。

### 4.3 Mitotic Index 的意义

- **Mitotic index 越高** → 细胞分裂越旺盛（如：胚胎组织、骨髓、皮肤基底层）
- **肿瘤组织**通常有**极高的 mitotic index**（不受控的频繁分裂）
- **成熟组织**（如肌肉、神经）mitotic index 极低或接近 0

## 5. 有丝分裂的意义 / Significance of Mitosis

有丝分裂产生**遗传上相同（genetically identical）**的子细胞，原因是：
1. S 期进行了**精确的 DNA 半保留复制**
2. M 期染色体**精确均等分配**到两个子细胞

**三大意义**：
| 意义 | 英文 | 举例 |
|---|---|---|
| **生长** | Growth | 多细胞生物由受精卵发育：细胞数目增多 |
| **修复与替换** | Repair and replacement | 皮肤细胞脱落再生、受伤后细胞修复 |
| **无性生殖** | Asexual reproduction | 草莓走茎、细菌分裂、珊瑚出芽 |

> 🎯 切记：有丝分裂产生的子细胞**遗传上相同**。**减数分裂**（meiosis，Topic 16）才产生遗传多样性的配子——两者绝对不能混淆！

## 6. 癌症与细胞周期失控 / Cancer & Uncontrolled Cell Division

### 6.1 癌症的形成机制

```
正常细胞
  ↓ 致癌物 / 辐射 / 致癌病毒（carcinogen / ionising radiation / oncogenic virus）
基因突变（mutation in DNA）
  ↓ 原癌基因（proto-oncogene）激活 → 成为癌基因（oncogene）
    + 抑癌基因（tumour suppressor gene）失活（如 p53 基因突变）
细胞周期检查点（cell cycle checkpoints）失效
  ↓
不受控细胞分裂（uncontrolled cell division）
  ↓
肿瘤（tumour）
```

**关键概念**：
- **Proto-oncogene（原癌基因）**：正常状态下控制细胞正常生长与分裂；突变后变为 **oncogene（癌基因）**，持续促进细胞分裂（类比"油门失控踩到底"）。
- **Tumour suppressor gene（抑癌基因）**：正常状态下抑制细胞过度分裂；突变后失去抑制功能（类比"刹车失灵"）。
- 癌症通常需要**多个基因同时突变**才发生。

**常见致癌物（carcinogens）**：
- 化学致癌物：烟草烟雾（多环芳烃等）、黄曲霉毒素（aflatoxin）、石棉
- 辐射：紫外线（UV）→ 皮肤癌；X 射线；γ 射线
- 致癌病毒（oncogenic viruses）：人乳头瘤病毒（HPV）→ 宫颈癌

### 6.2 良性肿瘤 vs 恶性肿瘤

| 特征 | 良性 Benign | 恶性 Malignant（癌症） |
|---|---|---|
| 生长速度 | 慢 | 快 |
| 包膜 | 有，边界清晰 | 通常无 |
| 侵袭性 | 局限，不侵入周围 | 侵入周围组织 |
| 转移（metastasis） | **无** | **有**（进入血液/淋巴→扩散） |
| 危险性 | 低（取决于位置） | 高 |

**转移（Metastasis）**的机制：恶性肿瘤细胞脱落 → 进入**血管或淋巴管** → 随血流/淋巴流到远处 → 在新部位增殖形成**继发肿瘤（secondary tumour）**。

## 7. 端粒与干细胞 / Telomeres & Stem Cells

### 7.1 端粒 Telomere

- **位置**：染色体**末端**的重复 DNA 序列（如 TTAGGG 重复数千次）
- **功能**：保护染色体末端不被降解或融合，稳定染色体
- **与衰老的关系**：每次 DNA 复制，端粒略有缩短 → 端粒过短 → 细胞停止分裂（细胞衰老/凋亡）
- **与癌症的关系**：癌细胞高度表达**端粒酶（telomerase）** → 端粒持续维持长度 → 细胞可无限分裂（immortalisation）

### 7.2 干细胞 Stem Cell

- **定义**：具有**自我更新（self-renewal）**能力和分化为多种类型细胞潜能的**未分化细胞**
- **全能干细胞（totipotent）**：可分化为生物体**所有**类型细胞（受精卵及早期胚胎）
- **多能干细胞（pluripotent）**：可分化为多种但非全部细胞类型（胚胎干细胞）
- **组织干细胞（multipotent）**：分化能力受限（如骨髓造血干细胞 → 各类血细胞）
- **意义**：干细胞靠有丝分裂维持自身数量，同时产生分化子细胞——是组织修复与再生的基础

## 8. 实验技能 / Practical: 根尖压片观察有丝分裂

### 8.1 材料与原理

- **材料**：大蒜（Allium sativum）或洋葱（Allium cepa）根尖（幼嫩、生长中的根尖分生区）
- **原理**：根尖分生区细胞持续有丝分裂，固定后可在各分裂期"定格"，用染料使染色体着色后在光镜下观察

### 8.2 实验步骤

| 步骤 | 操作 | 目的 |
|---|---|---|
| ① 切取根尖 | 切取约 1 cm 的根尖（含分生区） | 获取含大量分裂细胞的材料 |
| ② 盐酸处理 | 浸入 1 mol/L HCl，60 °C 水浴 5–8 min | **软化（macerate）**细胞壁；破坏细胞间果胶使细胞分离 |
| ③ 水洗 | 蒸馏水漂洗数次 | 去除盐酸 |
| ④ 染色 | 浸入**醋酸洋红（aceto-carmine）**或**龙胆紫（crystal violet）**数分钟 | 使**染色体着深色**（染料与 DNA/蛋白质结合） |
| ⑤ 压片 | 置于载玻片，盖盖玻片，用**大拇指垂直按压（squash）** | 使细胞分散成**单细胞层**，便于观察 |
| ⑥ 观察 | 先低倍镜找分生区（细胞密集），再换高倍镜 | 辨认各分裂期染色体特征 |

> ⚠️ **安全**：盐酸有腐蚀性，需佩戴护目镜和手套。

### 8.3 观察预期
- 大多数细胞处于**间期**（核膜完整，无清晰染色体，核内染色质均匀着色）
- 少数细胞可见各 PMAT 期特征
- Mitotic index = 处于 M 期细胞数 ÷ 总细胞数

### 8.4 实验局限性
- 细胞是**死亡固定**后的，不能观察实时分裂过程
- 压片过程可能使染色体受损或细胞重叠
- 盐酸处理时间过长/过短均影响效果

## 必背定义 / Must-know definitions（英文背诵）

| 概念 | 英文定义（贴官方措辞） | 中文 |
|---|---|---|
| Chromosome | *A thread-like structure composed of DNA and histone proteins; visible during cell division.* | 染色体 |
| Sister chromatids | *Two identical copies of a chromosome joined at the centromere, formed after DNA replication.* | 姐妹染色单体 |
| Mitosis | *A type of nuclear division that produces two genetically identical nuclei, each containing the same number of chromosomes as the parent cell.* | 有丝分裂 |
| Mitotic index | *The ratio of the number of cells in mitosis to the total number of cells observed.* | 有丝分裂指数 |
| Tumour | *A mass of cells resulting from uncontrolled cell division.* | 肿瘤 |
| Metastasis | *The spread of cancer cells from the original tumour to other parts of the body via the blood or lymph.* | 转移 |
| Carcinogen | *Any agent (chemical, radiation or virus) that can cause cancer by inducing mutations.* | 致癌物 |
| Telomere | *Repetitive DNA sequences at the ends of chromosomes that protect chromosomal ends from degradation.* | 端粒 |
| Stem cell | *An undifferentiated cell that is capable of self-renewal and differentiating into specialised cell types.* | 干细胞 |

## A* 拓展 / Stretch to A*（Synoptic 联系）

- **连接 Topic 2（DNA 结构）**：DNA 复制发生在 S 期，用 DNA 聚合酶、需解旋酶（Topic 6）；histone 是球形蛋白，DNA 缠绕其外——理解染色体压缩的结构基础。
- **连接 Topic 6（DNA 复制）**：S 期的"精确 DNA 复制"保证子细胞遗传相同——半保留复制的意义在有丝分裂中得到体现。
- **连接 Topic 1（中心粒 centriole）**：动物细胞中心粒在 G2 期复制，在 M 期组织纺锤体——与 Topic 1 细胞器功能联系。
- **连接 Topic 16（减数分裂）**：有丝分裂（保持 2n，产生遗传相同细胞）vs 减数分裂（产生 n 的配子，产生遗传多样性）——synoptic 经典比较题。
- **癌症治疗 synoptic**：化疗药物（如秋水仙碱，colchicine）靶向纺锤体 → 阻断 metaphase 进行 anaphase；放疗利用 DNA 损伤杀死快速分裂细胞——均与有丝分裂机制相关。
- **端粒酶与干细胞**：干细胞和生殖细胞有较高端粒酶活性，维持端粒长度从而维持增殖能力——解释为什么干细胞能长期自我更新。
