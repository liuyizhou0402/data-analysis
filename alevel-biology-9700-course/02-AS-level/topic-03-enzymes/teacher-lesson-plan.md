## 课时 3.1 — 酶的本质与作用机制 / Enzyme Mechanism（60 min）
> 对应 Lesson 12（100-Hour Master Plan）

### 教学目标 / Learning objectives
学生在本课后能够：
- [ ] **Define** enzyme 的官方定义（biological catalyst，降低 activation energy，不被消耗）
- [ ] **Explain** 酶为何能降低活化能，以及这对反应速率的意义
- [ ] **Compare** lock-and-key 模型与 induced-fit 模型（成对说清区别）
- [ ] **Describe** enzyme-substrate complex 的形成过程

### 必备术语 / Key terms
| English | 中文 |
|---|---|
| enzyme | 酶 |
| biological catalyst | 生物催化剂 |
| activation energy | 活化能 |
| active site | 活性位点 |
| substrate | 底物 |
| product | 产物 |
| enzyme-substrate complex (ESC) | 酶-底物复合物 |
| enzyme-product complex | 酶-产物复合物 |
| lock-and-key model | 锁钥模型 |
| induced-fit model | 诱导契合模型 |
| specificity | 专一性 |
| globular protein | 球状蛋白 |
| complementary shape | 互补形状 |

### 授课流程 / Lesson flow
| 时间 | 环节 | 老师做什么 | 学生做什么 |
|---|---|---|---|
| 0–5′ | 导入 | 展示淀粉→麦芽糖（唾液淀粉酶）与体外高温加热同样反应速率比较——"为什么 37 °C 身体里反应能发生？"引出"降低活化能" | 思考猜测 |
| 5–20′ | 精讲 | ①酶=球状蛋白+活性位点（active site，三维形状由蛋白质三级结构决定）；②活化能概念（能量障碍，好比"翻山"：酶降低山峰高度，而非改变反应起/终点能量）；③专一性来源（每种酶活性位点形状只与特定底物互补） | 记笔记，画活化能曲线（有/无酶对比） |
| 20–38′ | 精讲+对比 | 分步讲两个模型：**Lock-and-key**（活性位点与底物形状完全互补，形状不变，像钥匙插锁）→ **Induced-fit**（底物结合时活性位点形状**改变**以更好契合底物，形成更稳定的 ESC；更好解释为什么活性位点大小与底物略有差异但仍结合）；强调诱导契合是当代更被接受的模型 | 填写两模型对比表 |
| 38–52′ | 真题型练习 | 出"描述酶作用机制"[4]、"compare lock-and-key and induced-fit"[3] 型题，学生限时作答 | 限时 10 分钟 |
| 52–60′ | 总结 | 锁定定义英文措辞；板书"ESC 形成 → 活化能降低 → 产物生成 → 酶释放、循环利用" | 默写 enzyme 定义 |

### 板书/重点 / Board work
- **活化能曲线**：横轴时间/反应进程，纵轴能量；无酶曲线峰值高，有酶峰值低；反应物与产物能量不变。
- **两模型对比表**：
  | | Lock-and-key | Induced-fit |
  |---|---|---|
  | 活性位点形状 | 固定不变 | 随底物结合而**改变/弯曲** |
  | 底物与活性位点 | 原本就完全互补 | 结合后诱导形状互补 |
  | 关键词 | rigid, pre-formed | flexible, changes shape, moulds around substrate |
- **ESC 形成流程**：底物 → 进入活性位点（互补）→ ESC 形成 → 活化能降低 → 键断裂/形成 → 产物离开 → 活性位点游离，可再利用。

### 易错点 / Common errors
- ❌ "酶被消耗掉" → ✅ 酶在反应后**释放**、形状**恢复**，可被**重复利用（not used up）**。
- ❌ "induced-fit 说活性位点和底物不互补" → ✅ induced-fit 是说**底物结合时活性位点形状发生改变**（而非两者原来形状不对应），改变后形成更紧密的 ESC。
- ❌ 说酶"提供能量给反应" → ✅ 酶**降低活化能（lower activation energy）**，不给能量。
- ❌ 笼统说"酶与底物形状相同" → ✅ 应说"active site and substrate have **complementary shapes**"（互补，不是相同）。

### 提问设计 / Key questions
- "如果把酶的 active site 氨基酸序列改变，会发生什么？"（引出蛋白质结构与专一性的关系→ Topic 2 synoptic）
- "为什么每种酶只能催化一种或一类反应？"（强化 complementary shape + specificity）

### 作业 / Homework
- `practice.md` Part A 第 1–3 题；Part B 第 1 题（ESC 与作用机制）。

## 课时 3.2 — 影响酶活性的因素 / Factors Affecting Enzyme Activity（60 min）
> 对应 Lesson 13

### 教学目标 / Learning objectives
- [ ] **Describe and explain** 温度对酶活性的影响，区分"变性"前后的机制
- [ ] **Describe and explain** pH 对酶活性的影响
- [ ] **Describe and explain** 底物浓度对反应速率的影响（含平台期的解释）
- [ ] **Describe and explain** 酶浓度对反应速率的影响
- [ ] **Sketch** 四种因素各自对反应速率的图像，并标注关键特征

### 必备术语 / Key terms
| English | 中文 |
|---|---|
| optimum temperature / pH | 最适温度/pH |
| denaturation | 变性 |
| active site (shape changed) | 活性位点（形状改变） |
| hydrogen bond | 氢键 |
| kinetic energy | 动能 |
| rate of reaction | 反应速率 |
| substrate concentration | 底物浓度 |
| enzyme concentration | 酶浓度 |
| plateau (maximum rate) | 平台（最大速率） |
| limiting factor | 限制因素 |
| collision frequency | 碰撞频率 |

### 授课流程 / Lesson flow
| 时间 | 环节 | 老师做什么 | 学生做什么 |
|---|---|---|---|
| 0–5′ | 回顾 | 快问：enzyme 定义、ESC 形成步骤、induced-fit 关键词 | 快答 |
| 5–22′ | 精讲：温度 | 分两段讲——①最适温度以下：温度↑→分子动能↑→底物与酶碰撞频率↑→有效碰撞↑→速率↑；②超过最适温度：过多热能→氢键（及其他弱键）断裂→**active site 形状永久改变（denaturation）**→底物不能结合→速率骤降至 0；强调 denaturation 是**不可逆**的。画 bell 型曲线，标 optimum | 画温度-速率曲线并标注 |
| 22–35′ | 精讲：pH | pH 偏离最适 → 影响活性位点附近氨基酸的电荷（改变离子键/氢键）→ active site 形状改变 → 底物不能结合；极端 pH → denaturation；画 bell 型（或偏态）pH-速率曲线 | 记笔记 |
| 35–48′ | 精讲：底物浓度 & 酶浓度 | 底物浓度↑：更多底物与活性位点碰撞→速率↑；但酶数量固定→所有活性位点饱和（saturated）→平台（Vmax）→酶成为限制因素；画"先升后平"曲线。酶浓度↑（底物充足时）：更多活性位点→速率线性↑（无平台，只要底物充足）；画线性上升曲线 | 画两条曲线 |
| 48–57′ | 图像分析练 | 给学生不同条件的酶速率图，要求"描述趋势+引数据+解释原因"三步 | 限时完成 2 图 |
| 57–60′ | 总结 | 锁定四个因素的解释关键词（动能/变性/饱和/限制因素） | 默写三个关键词 |

### 板书/重点 / Board work
- **温度曲线**（bell 型）：标 optimum temperature，左侧标"↑ kinetic energy, ↑ collision frequency"，右侧标"denaturation, active site changes shape permanently"。
- **底物浓度曲线**（先升后平）：标 plateau/Vmax，注"all active sites saturated / enzyme is limiting factor"。
- **四个因素解释框架**：
  | 因素 | 速率↑ 机制 | 速率↓/平台 机制 |
  |---|---|---|
  | 温度 | 动能↑，碰撞↑ | 变性，active site 改变（不可逆） |
  | pH | 最适 pH 维持活性位点形状 | 偏离→键断裂→active site 改变 |
  | 底物浓度 | 更多有效碰撞 | 所有酶饱和→平台（酶限制） |
  | 酶浓度 | 更多活性位点 | 底物耗尽时速率不再↑ |

### 易错点 / Common errors
- ❌ 说"低温也使酶变性" → ✅ 低温只是**降低活性（活性降低/分子动能低/碰撞少）**，并不变性；升温后活性可**恢复**。变性只发生在**过高温度/极端 pH**（不可逆）。
- ❌ 底物浓度图"解释平台"时只写"底物足够了" → ✅ 平台的原因是"**all enzyme active sites are occupied/saturated**，enzyme is the **limiting factor**"。
- ❌ "温度升高速率一直升" → ✅ 超过最适温度**变性**，速率骤降。
- ❌ pH 影响只说"变性" → ✅ 先说键（ionic bonds / hydrogen bonds）断裂→active site 形状改变→底物不能结合，极端时才完全变性；分层解释更得分。

### 提问设计 / Key questions
- "37 °C 的酶放入 4 °C 冰水中，活性会如何变化？取出复温后呢？"（辨析低温 vs 高温变性，关键：低温可逆，高温不可逆）
- "画出：温度 30 °C vs 40 °C 时，底物浓度与速率的曲线（同一图）。"（图像综合：Vmax 在 40 °C 更高，但若 40 °C 超过最适则更低）

### 作业 / Homework
- `practice.md` Part A 第 4–6 题；Part B 第 2 题；Part C 第 1 题（图像分析）。

## 课时 3.3 — 抑制剂 / Inhibition（60 min）
> 对应 Lesson 14

### 教学目标 / Learning objectives
- [ ] **Describe** 竞争性抑制（competitive inhibition）的机制
- [ ] **Describe** 非竞争性抑制（non-competitive inhibition）的机制
- [ ] **Compare** 两类抑制剂（成对：结合位点、能否被底物浓度克服、对 Vmax 和 Km 的影响）
- [ ] **Explain** Vmax 与 Km 的概念及意义
- [ ] **Interpret** 含抑制剂的速率-底物浓度图像

### 必备术语 / Key terms
| English | 中文 |
|---|---|
| inhibitor | 抑制剂 |
| competitive inhibitor | 竞争性抑制剂 |
| non-competitive inhibitor | 非竞争性抑制剂 |
| allosteric site | 别构位点 |
| Vmax (maximum rate) | 最大反应速率 |
| Km (Michaelis constant) | 米氏常数 |
| affinity | 亲和力 |
| reversible / irreversible inhibition | 可逆/不可逆抑制 |
| structural analogue | 结构类似物 |

### 授课流程 / Lesson flow
| 时间 | 环节 | 老师做什么 | 学生做什么 |
|---|---|---|---|
| 0–5′ | 回顾 | 快问：变性定义；底物浓度平台原因；pH 影响机制 | 快答 |
| 5–18′ | 精讲：竞争性抑制 | 结构与底物类似（structural analogue）→ 与底物**争夺 active site**→ 占据活性位点后底物无法结合→速率降低；加大底物浓度→底物"抢回"活性位点→可克服（overcome by increasing [S]）；效果：**Vmax 不变，表观 Km 升高**（需更多底物才达到 ½Vmax）；举例：磺胺药物（sulphonamide）竞争性抑制细菌 PABA | 填对比表"结合位点"栏 |
| 18–35′ | 精讲：非竞争性抑制 | 结合酶的**别构位点（allosteric site，非活性位点）**→ 导致 active site 形状改变→底物无法与 active site 结合或 ESC 不能反应→速率降低；底物浓度再高**也无法克服**（cannot be overcome）；效果：**Vmax 降低，Km 不变**（亲和力不变，只是最大速率下降）；举例：重金属离子（如汞、铅）、有机磷农药 | 填对比表其余栏 |
| 35–50′ | 图像解读 | 画"速率-底物浓度"图，分别在同一图上标出：无抑制剂、竞争性（Vmax 不变、曲线右移/Km 增大）、非竞争性（Vmax 降低、Km 不变）；逐条分析三条曲线差异 | 解读三条曲线，回答提问 |
| 50–57′ | Vmax 与 Km 讲解 | Km 定义（达到½Vmax 时的底物浓度）；Km 越小→对底物亲和力越高；强调 Km 是酶的固有特性（与酶的本质有关）；竞争性抑制使**表观 Km 升高**（非真正 Km 改变，因为可被克服）；非竞争性抑制不改变 Km | 在自测清单勾选 |
| 57–60′ | 总结 | 锁定两类抑制剂的成对表达方式 | 当场默写成对对比两条 |

### 板书/重点 / Board work
- **两类抑制剂对比表（必背成对）**：
  | | 竞争性 Competitive | 非竞争性 Non-competitive |
  |---|---|---|
  | 结合位点 | **Active site**（活性位点） | **Allosteric site**（别构位点） |
  | 结构 | 类似底物（structural analogue） | 不类似底物 |
  | 能被高底物浓度克服？ | **是（Yes, overcome）** | **否（No, cannot be overcome）** |
  | Vmax | **不变（unchanged）** | **降低（decreased）** |
  | Km（表观）| **升高（increased）** | **不变（unchanged）** |
- **图像**：三条"速率-[S]"曲线；竞争性曲线在高 [S] 与无抑制剂曲线趋于相同 Vmax；非竞争性曲线 Vmax 更低但最初斜率（Km 位置）与无抑制剂相同。

### 易错点 / Common errors
- ❌ "竞争性抑制剂改变 Vmax" → ✅ 竞争性抑制剂**不改变 Vmax**（高底物可克服），只使表观 Km 升高。
- ❌ "非竞争性抑制剂结合在活性位点" → ✅ 非竞争性结合**别构位点（allosteric site）**，**不与底物争夺活性位点**。
- ❌ 判断抑制类型时只看"Vmax" → ✅ 同时看 Vmax 和 Km 两个指标才能完整判断。
- ❌ 说竞争性抑制是"永久"的 → ✅ 竞争性抑制通常是**可逆的**（reversible），底物浓度高时可被克服。

### 提问设计 / Key questions
- "从图中如何判断是竞争性还是非竞争性抑制？"（训练图像解读：看 Vmax 是否变化，看曲线右移还是峰值下降）
- "如果某药物是酶的非竞争性抑制剂，给患者增大酶的底物能否提高酶活性？"（迁移到临床，强化"不可克服"概念）

### 作业 / Homework
- `practice.md` Part A 第 7–9 题；Part B 第 3 题；Part C 第 2 题（图像分析：判断抑制类型）。

### 授课流程 / Lesson flow

| 35–52′ | 综合练习 | 学生完成 `practice.md` Part D 实验题 + Part C 第 3 题（固定化酶应用分析） | 限时作答 |
| 52–57′ | 讲评 | 对照 `practice-answers.md` 逐题讲评；重点：D 题中"控制变量"与"如何测量速率"的规范 | 对答案，标错误类型 |
| 57–60′ | 总结 | 本主题 A\* 关键四点快速复盘；布置本主题整体练习 | 整理自测清单 |

### 板书/重点 / Board work
- **固定化酶优点对比表**：
  | 优点 | 具体原因 |
  |---|---|
  | 可重复使用 | Enzyme not lost / washed away with product |
  | 产物纯净 | No enzyme contamination in product |
  | 稳定性更高 | More resistant to changes in temperature / pH |
  | 连续生产 | Substrate can flow continuously over/through the immobilised enzyme |
- **实验：测反应速率关键点**：①明确因变量测量方式（O₂ 体积 or 碘液褪色时间）②列出控制变量（至少 2 个具体、可操作的）③说明如何计算速率（体积/时间 或 1/时间）。

### 易错点 / Common errors
- ❌ 说固定化酶"不会失活" → ✅ 固定化酶**更稳定**，但并非永不失活；长时间使用后仍会降解。
- ❌ 实验中"控制变量"只写"其他条件不变" → ✅ 必须**具体列出**（如"same volume and concentration of H₂O₂"，"same volume and concentration of enzyme"，"same temperature"）。
- ❌ 说固定化酶"可重复使用因为是催化剂" → ✅ 所有酶本身都是催化剂；固定化酶的优点是**不随产物流失**（not washed away），因此可反复使用。

### 提问设计 / Key questions
- "工厂生产无乳糖奶，为什么要用固定化乳糖酶而非直接把酶加入牛奶？"（综合优点：重复使用、产物无酶污染）
- "设计一个实验，研究 pH 对 catalase 活性的影响。说出你的自变量、因变量、控制变量，以及如何测定速率。"（实验设计框架训练）
