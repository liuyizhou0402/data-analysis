## 1. 气体交换表面的五大特征 / Five Features of a Gas Exchange Surface

所有高效气体交换表面都具备（以肺泡为例）：

| 特征 | 肺泡如何实现 | 功能意义 |
|---|---|---|
| **Large surface area（大表面积）** | ~7000 万肺泡，总面积约 70 m² | 扩散总速率 ∝ 面积 |
| **Thin barrier（薄障碍）** | 肺泡壁 + 毛细血管壁各一层细胞（约 0.5 μm）| 扩散距离短→速率快（Fick 定律）|
| **Moist surface（湿润）** | 表面活性物质 + 薄层水膜 | 气体溶解后才能扩散 |
| **Maintained concentration gradient（维持浓度梯度）** | 血液循环（移走 O₂，带来 CO₂）| 维持高扩散驱动力 |
| **Ventilation（通气）** | 呼吸运动不断更换肺泡气体 | 防止肺泡气体与血液浓度平衡 |

**Fick 定律（Fick's Law）**：扩散速率 ∝ 表面积 × 浓度差 / 扩散距离

## 2. 肺泡结构 / Alveolar Structure

```
空气 → 气管 → 支气管 → 细支气管 → 肺泡管 → 肺泡（alveolus）
```

肺泡壁由**鳞状上皮（squamous epithelium）**构成（极薄，利于扩散）。

**II 型肺泡细胞**分泌**表面活性物质（surfactant）**：
- 降低表面张力，防止肺泡在呼气时塌陷
- 早产儿缺乏 surfactant → 呼吸窘迫综合征

气体交换路径（O₂ 方向）：
```
肺泡腔 → 肺泡壁（一层鳞状上皮）→ 毛细血管壁（一层内皮）→ 血浆 → 红细胞 → 与 Hb 结合
```

## 3. 肺通气机制 / Breathing Mechanism

### 3.1 吸气（Inspiration）——主动过程

```
膈肌收缩（变平向下）+ 肋间外肌收缩（肋骨向上向外）
    ↓
胸腔容积↑
    ↓
肺容积↑（肺有弹性随之扩张）
    ↓
肺内气压↓（低于大气压）
    ↓
空气顺压力梯度流入肺
```

### 3.2 呼气（Expiration）——静息时为被动过程

```
膈肌和肋间外肌舒张
    ↓
肺弹性回缩（elastic recoil）→ 胸腔容积↓
    ↓
肺内气压↑（高于大气压）
    ↓
空气排出
```

> ⚠️ 静息呼气**不需要**肋间内肌收缩——是弹性回缩驱动的被动过程。用力呼气才需要肋间内肌和腹肌主动收缩。

### 3.3 肺容积参数

| 参数 | 含义 | 典型值 |
|---|---|---|
| **Tidal volume (TV)** | 每次平静呼吸吸入/呼出量 | ~0.5 L |
| **Vital capacity (VC)** | 最大吸气后能呼出的最大量 | ~4.5 L |
| **Residual volume (RV)** | 最大呼气后肺内剩余量 | ~1.5 L |
| **Ventilation rate** | 每分钟通气量 = TV × 呼吸频率 | ~6 L/min |

## 4. 鱼鳃与逆流交换 / Fish Gills & Countercurrent Exchange

### 4.1 鳃结构
- 鳃弓（gill arch）→ 鳃丝（gill filaments）→ 鳃片（lamellae，次级薄片）
- 鳃片内有密集毛细血管网

### 4.2 逆流交换原理（核心！）

血流方向与水流方向**相反**（逆流）：

```
水流方向 ————————————————→ 高 pO₂（40%）→ 低 pO₂（5%）
血流方向 ←———————————————— 低 pO₂（15%）← 高 pO₂（35%）
                                          ↑
                          在整个鳃长度上，血液 pO₂ 始终低于水 pO₂
                          → 全程维持扩散梯度 → O₂ 持续扩散进入血液
```

**效率对比**：
- 同向流（parallel）：O₂ 提取效率约 **50%**（最终达到平衡）
- 逆流（countercurrent）：O₂ 提取效率约 **80–90%**

> 🎯 关键：逆流保证"在鳃片出口端，血液 pO₂ 仍低于水 pO₂"——扩散梯度始终存在。

## 5. 昆虫气管系统 / Insect Tracheal System

气体交换路径：
```
外界空气 → 气门（spiracle，体侧小孔）→ 气管（trachea，有几丁质支环）
→ 气管小管（tracheoles，终端湿润）→ 组织细胞
```

**特点**：
- O₂ 和 CO₂ **直接扩散**到/来自细胞——不经过血液（blood doesn't carry O₂）
- 大型昆虫通过腹部伸缩产生通风（pumping）
- 体型受限：气管过长则扩散效率不足 → 昆虫不能像哺乳动物那么大

> ⚠️ 昆虫血液（血淋巴 haemolymph）运输**营养物质和激素**，**不运输 O₂**。

## 6. 叶片气体交换 / Gas Exchange in Leaves

```
外界 CO₂ → 气孔（stomata）→ 亚气孔腔（sub-stomatal air space）
→ 叶肉细胞壁（湿润）→ 溶解→扩散进叶肉细胞 → 叶绿体
```

- 气孔白天开 → CO₂ 进入（光合），O₂ 释放
- 气孔夜间关 → 呼吸作用 CO₂ 部分积聚
- 叶肉组织（海绵叶肉 spongy mesophyll）有大量**细胞间隙**→ 增大气体扩散路径

## 7. 螺旋仪（Spirometer）

用于测量肺通气量和呼吸速率。

**关键操作**：
- 加**CO₂ 吸收剂（soda lime / NaOH）**：吸收呼出 CO₂ → 气钟只记录 O₂ 消耗 → 基线缓慢下降
- 不加吸收剂：CO₂ 与 O₂ 体积变化部分相消 → 基线不明显下降

**从迹图读取**：
- TV = 潮气量（相邻峰谷差）
- VC = 深吸后最大呼出量
- 呼吸速率 = 每分钟波峰数

## 必背定义 / Must-know Definitions

| 概念 | 英文定义 | 中文 |
|---|---|---|
| Gas exchange surface | *A surface across which gases diffuse between the organism and its environment.* | 气体交换面 |
| Countercurrent exchange | *A mechanism where two fluids flow in opposite directions, maintaining a concentration gradient along the entire length of the exchange surface.* | 逆流交换 |
| Ventilation | *The movement of a medium (air/water) over the gas exchange surface to maintain a concentration gradient.* | 通气 |
| Tidal volume | *The volume of air inhaled or exhaled in one normal breath.* | 潮气量 |

## A* 拓展 / Stretch to A*
- **Fick's Law 量化**：扩散速率 = (surface area × concentration difference × permeability) / thickness；肺泡每个参数都优化
- **Topic 8 连接**：O₂ 从肺泡扩散入红细胞后，与 Hb 结合，玻尔效应确保在肌肉处释放
- **Topic 13 连接**：叶肉细胞白天 CO₂ 浓度低（光合消耗），维持扩散进入梯度
- **肺表面活性物质缺乏（早产儿）**：肺泡塌陷 → 呼吸困难，需要外源 surfactant 治疗
