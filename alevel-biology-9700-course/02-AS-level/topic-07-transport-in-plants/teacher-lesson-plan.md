# Topic 7 — 植物运输 / Transport in Plants｜教师教案（4 课时）

## 主题概览 / Topic overview
- **对应考纲**：9700 — 7.1 Transport in plants（木质部与韧皮部结构；水分吸收与运输；蒸腾；韧皮部运输）
- **对应试卷**：Paper 1（选择：结构辨认、蒸腾因素）、Paper 2（结构题：定义、解释、描述）、Paper 3（实验技能：potometer 操作与数据分析）
- **对应课时**：Lessons 29–32（100-Hour Master Plan 中第 29–32 课）
- **前置知识**：Topic 4（水势 water potential / osmosis）；Topic 1（植物细胞壁、液泡）；Topic 2（糖类：蔗糖）

- **本主题 A\* 关键**：
  1. **内聚-张力学说（cohesion-tension theory）**：需要解释氢键→内聚力→张力的因果链（最高频 explain 题）
  2. **影响蒸腾因素**：每个因素必须解释**机理**（不只说快/慢），且与气孔开闭挂钩
  3. **韧皮部压力流学说（mass flow hypothesis）**：源端主动装载→水势降低→水分进入→高压→流动→库端卸载，按顺序完整作答
  4. **结构-功能对应**：木质部死细胞/木质化/无端壁，韧皮部筛板/无核/伴胞——考试识图极常见

## 课时 7.1 — 植物运输组织 / Xylem & Phloem Structure（60 min）

### 教学目标 / Learning objectives
学生在本课后能够：
- [ ] **State** 多细胞植物需要运输系统的原因（SA:V 比）
- [ ] **Describe** 木质部导管（xylem vessel）的结构特征（死细胞、木质化、无端壁、中空）
- [ ] **Explain** 木质部结构如何适应其功能（运输水和无机离子；提供支撑）
- [ ] **Describe** 韧皮部筛管分子（sieve tube element）和伴胞（companion cell）的结构特征
- [ ] **Identify** 木质部和韧皮部在根、茎、叶横截面中的分布

| English | 中文 |

| xylem vessel | 木质部导管 |

| dead cell | 死细胞 |

| pits | 纹孔 |

### 授课流程 / Lesson flow
| 时间 | 环节 | 老师做什么 | 学生做什么 |
|---|---|---|---|
| 0–5′ | 导入 | 展示一棵大树与一个单细胞的对比图，问"一棵树高 20 米，营养物质怎么从根传到叶？叶片水分怎么补充？" | 猜想、自由讨论 |
| 5–20′ | 精讲①：为什么需要运输系统 | 讲 SA:V 比——细胞越大，内部距离越长，扩散太慢；多细胞植物需要专门运输系统。类比人体血管 | 计算球形细胞 SA:V 的变化趋势 |
| 20–40′ | 精讲②：木质部与韧皮部结构 | 逐一讲结构特征，边讲边画横截面示意图。木质部：**死细胞→管腔空**（无细胞内容物阻碍水流）；**木质素**（lignin）沉积细胞壁→**壁厚硬**；**无端壁**→水分纵向连通；纹孔（pits）→水分横向移动。韧皮部：**筛管分子**（活细胞，无细胞核，有筛板）；**伴胞**（有核，为筛管提供能量/物质） | 画结构图并标注，完成结构-功能对照表 |
| 40–52′ | 识图练习 | 展示根/茎/叶横截面 TEM 或光镜图，要求学生识别：根中木质部在中央（星状，X形）；茎中维管束散生（单子叶）或环状（双子叶），韧皮部在外；叶中维管束（叶脉）贯穿叶肉 | 标注各组织；真题型：识别并描述结构 |
| 52–60′ | 总结 | 锁定"结构-功能"配对表；强调"**死细胞/空腔/无端壁**"是木质部导管的三大考点 | 默写木质部导管三大结构特征及各自功能意义 |

### 板书/重点 / Board work
- **结构-功能对照表**：

| 结构特征 | 功能意义 |
|---|---|
| 木质部：死细胞、中空无内容物 | 最小流动阻力，高效运输水分 |
| 木质素沉积细胞壁 | 壁硬，提供机械支撑；防止管道塌陷 |
| 无端壁（贯通） | 形成连续水柱，水分可纵向连续流动 |
| 纹孔（pits） | 水分可横向在相邻细胞间移动 |
| 韧皮部：筛管无核、有筛板 | 减少细胞器占据空间；筛板允许溶质流动 |
| 伴胞：有核、多线粒体 | 为筛管提供蛋白质和能量（ATP） |

- **维管束分布图**（简笔画）：根（木质部在内，韧皮部在外，放射状）→ 茎（维管束：韧皮部外，木质部内）→ 叶（叶脉，上方木质部，下方韧皮部）。

### 易错点 / Common errors
- ❌ 说木质部"是活细胞" → ✅ 木质部导管是**成熟后死去的死细胞**，细胞壁经木质素强化，**没有细胞质内容物**。
- ❌ 说韧皮部筛管"无细胞" → ✅ 筛管分子是**活细胞**，只是**无细胞核**（核降解），代谢依赖伴胞。
- ❌ 只描述功能不联系结构 → ✅ 结构题凡问"**how is the structure of xylem adapted…**"，必须先写**结构特征**再写**功能意义**（结构→功能因果）。

### 提问设计 / Key questions
- "为什么木质部导管没有细胞核和细胞质却不算'死亡的废物'？它的'空'和'死'对运输有什么好处？"
- "韧皮部筛管与木质部导管都参与运输，最本质的区别是什么？"（活/死；运输物质；运输机制）

### 作业 / Homework
- `practice.md` Part A 第 1–3 题、Part B 第 1 题。
- 画一张木质部导管横切面与纵切面的手绘图，标注所有结构特征。

## 课时 7.2 — 蒸腾与水的运输 / Transpiration & Water Movement（60 min）

### 教学目标 / Learning objectives
学生在本课后能够：
- [ ] **Describe** 水分从土壤进入根毛细胞、经三条途径穿过根皮质进入木质部的过程
- [ ] **Explain** 凯氏带（Casparian strip）迫使水分进入共质体途径的机制及其意义
- [ ] **Define** transpiration（蒸腾作用）
- [ ] **Explain** 内聚-张力学说（cohesion-tension theory）如何解释水在木质部中向上运输

### 必备术语 / Key terms
| English | 中文 |
|---|---|
| root hair cell | 根毛细胞 |
| apoplast pathway | 质外体途径（经细胞壁，不穿过细胞膜） |
| symplast pathway | 共质体途径（经胞间连丝，穿过细胞膜） |
| vacuolar pathway | 液泡途径（经细胞壁→液泡→细胞壁） |

| transpiration stream | 蒸腾流 |
| cohesion | 内聚力 |
| adhesion | 附着力 |
| tension | 张力（负压） |

### 授课流程 / Lesson flow
| 时间 | 环节 | 老师做什么 | 学生做什么 |
|---|---|---|---|
| 0–5′ | 回顾 | 快问：木质部三大结构特征；韧皮部与木质部各运什么？ | 口头作答 |
| 5–22′ | 精讲①：水分进入根的过程 | 画根横截面示意图，逐一解释：① 根毛细胞**表面积大**（长而细的突起）→有利于水分吸收；② 土壤水分水势 > 根细胞水势 → osmosis 进入；③ 三条途径：**质外体**（最快，不穿膜，细胞壁&细胞间隙）；**共质体**（穿细胞膜、胞间连丝）；**液泡途径**（穿液泡膜）；④ **内皮层凯氏带**（Casparian strip，木栓质浸渍，堵塞质外体）→强迫水分经共质体进入内皮层细胞 → 控制进入木质部的物质 | 画三条途径示意图；标注凯氏带位置 |
| 22–40′ | 精讲②：内聚-张力学说 | 讲蒸腾流的"动力"来源：① **蒸腾（气孔散失水汽）**→ 叶肉细胞水势降低 → 水从木质部渗入；② 木质部水柱顶端水分减少 → 产生**张力（tension/负压）**；③ 水分子间**氢键**产生**内聚力（cohesion）**→ 水柱连续拉升；④ 水分子与管壁之间产生**附着力（adhesion）**→ 辅助支撑水柱。板书公式/逻辑链 | 画"蒸腾-内聚-张力"逻辑链流程图 |
| 40–52′ | 真题型练习 | 出题：Explain how water moves up a xylem vessel from roots to leaves. **[5]** 要求用分点写，老师逐句示范标准答法 | 限时独立作答 |
| 52–60′ | 总结 | 用一张图锁定：蒸腾（叶）→ 拉力 → 水柱 → 内聚力 → 吸水（根），闭环；强调"**凯氏带**"是本题高频遗漏点 | 默写内聚-张力学说 5 步逻辑 |

### 板书/重点 / Board work
- **内聚-张力学说逻辑链**（核心板书）：

```
叶片蒸腾（气孔散失水汽）
    ↓
叶肉细胞水势降低 → 水分从木质部进入叶肉（osmosis）
    ↓
木质部管腔顶端产生 TENSION（张力/负压）
    ↓
水分子间氢键产生 COHESION（内聚力）→ 水柱连续、不断裂
    ↓
拉力向下传递 → 根部从土壤吸水
    ↓
ADHESION（附着力）：水与管壁之间也产生，辅助支撑
```

- **三条水分进入根的途径图**（质外体/共质体/液泡途径对比，凯氏带位置标注）。

### 易错点 / Common errors
- ❌ 说内聚-张力学说"靠压力推水上去" → ✅ 是**拉（张力/负压）**，不是推；动力来自顶端（叶），不是底端（根）。
- ❌ 漏掉"氢键"→ ✅ 必须说水分子间**氢键**产生内聚力，否则考试不得分。
- ❌ 凯氏带作用表述不全 → ✅ 凯氏带**堵塞质外体途径**，迫使水分进入**共质体（穿过细胞膜）**→ 细胞可**选择性控制**进入木质部的离子，阻止有害物质。
- ❌ 混淆质外体与共质体 → ✅ 质外体 = 不穿细胞膜（经细胞壁/间隙）；共质体 = 穿细胞膜（经胞间连丝）。

### 提问设计 / Key questions
- "如果破坏了木质部水柱中段的连续性（如引入气泡），根据内聚-张力学说，会发生什么？为什么？"（考内聚力的重要性）
- "凯氏带被破坏后，植物会受到什么影响？"（考对物质选择性控制的理解）

### 作业 / Homework
- `practice.md` Part A 第 4–5 题、Part B 第 2 题。
- 用自己的语言写出内聚-张力学说的解释（英文，约 5 句话，要有关键词 transpiration / tension / cohesion / hydrogen bond / adhesion）。

## 课时 7.3 — 影响蒸腾的因素与实验 / Factors Affecting Transpiration & Potometer（60 min）

### 教学目标 / Learning objectives
学生在本课后能够：
- [ ] **Explain** 光照、温度、空气湿度、风速如何影响蒸腾速率（必须讲机理）
- [ ] **Describe** potometer 的装置、操作步骤与测量原理
- [ ] **Evaluate** potometer 测得的是吸水速率（water uptake rate）而非蒸腾速率的局限
- [ ] **Outline** 旱生植物（xerophyte）的结构适应（简述4个特征）

### 必备术语 / Key terms
| English | 中文 |
|---|---|
| transpiration rate | 蒸腾速率 |
| light intensity | 光照强度 |
| humidity / water vapour | 空气湿度 / 水汽 |
| diffusion gradient | 扩散梯度（浓度/水势梯度） |
| potometer | 蒸腾计（气泡法测吸水速率） |
| xerophyte | 旱生植物 |
| waxy cuticle | 蜡质角质层 |
| sunken stomata | 下陷气孔 |
| rolled leaf | 卷叶 |
| leaf hairs / trichomes | 表皮毛 |

### 授课流程 / Lesson flow
| 时间 | 环节 | 老师做什么 | 学生做什么 |
|---|---|---|---|
| 0–5′ | 回顾 | 快问：蒸腾流的动力来自哪里？三条水分途径的名字？ | 口头作答 |
| 5–20′ | 精讲①：四大影响因素 | 用"蒸腾 = 气孔散失水汽"作为核心逻辑，逐一讲：① **光照强度↑** → 保卫细胞中光合↑→K⁺泵入保卫细胞→水势降低→水分进入→保卫细胞膨胀→**气孔张开↑**→扩散梯度↑→蒸腾↑；② **温度↑** → 水分子动能↑+饱和水汽压↑→叶内外水汽浓度差↑→扩散速率↑；③ **空气湿度↑** → 叶外水汽浓度↑→**扩散梯度↓**→蒸腾↓；④ **风速↑** → 带走叶片外水汽→**扩散梯度维持↑**→蒸腾↑ | 填写四因素-机理表格；强调每因素须联系"扩散梯度"或"气孔开闭" |
| 20–38′ | 精讲②：potometer 实验 | 画装置图（带气泡的毛细管、储水池、植物枝条）；逐步讲：① **装置步骤**：在水中切断枝条防气泡；装入装置；确保接口密封；② **测量原理**：气泡在毛细管中移动距离 × 毛细管截面积 = 吸水体积；③ **局限性**：实际是测**吸水速率**（water uptake）≈ 蒸腾速率，但有一部分水用于细胞生长，因此 **uptake ≥ transpiration**；④ **控制变量**：每次实验只改变一个因素，其余不变 | 画装置图并标注；回答"为什么要在水中切断枝条" |
| 38–50′ | 精讲③：旱生植物适应 | 简述4个结构：① **厚蜡质角质层**（减少角质蒸腾）；② **下陷气孔**（陷窝积累湿润空气，扩散梯度↓）；③ **卷叶**（减小表面积，陷入湿润微环境）；④ **表皮毛**（滞留湿润空气层，扩散梯度↓）。每个特征与机制连接 | 表格：特征→减少蒸腾的机制 |
| 50–60′ | 总结 | 强调数据题答法：描述趋势→引用数值→解释原因；potometer 局限性常考 | 默写四因素影响蒸腾的机理（英文关键词） |

### 板书/重点 / Board work
- **四因素影响蒸腾速率表**（核心板书）：

| 因素 | 变化 | 影响气孔/扩散梯度 | 蒸腾速率 |
|---|---|---|---|
| 光照强度 | ↑ | 气孔张开，扩散梯度↑ | ↑ |
| 温度 | ↑ | 水分子动能↑，饱和水汽压↑，梯度↑ | ↑ |
| 空气湿度 | ↑ | 叶外水汽浓度↑，扩散梯度↓ | ↓ |
| 风速 | ↑ | 带走叶外水汽，维持/增大扩散梯度 | ↑ |

- **Potometer 装置示意图**（毛细管、气泡、储水池、植物枝条，标注测量方法）。

### 易错点 / Common errors
- ❌ 解释蒸腾因素时只说"温度升高蒸腾加快"而不说原因 → ✅ 必须解释**机理**：动能/饱和水汽压/扩散梯度/气孔开闭，否则 explain 题不给分。
- ❌ 说 potometer "测蒸腾速率" → ✅ potometer **实际测吸水速率（water uptake rate）**；因为有一部分水用于其他过程，吸水速率 ≥ 蒸腾速率。
- ❌ 光照与温度对蒸腾的影响机理混淆 → ✅ 光照通过**气孔开闭**影响（保卫细胞机制）；温度通过**动能/扩散速率**影响（不改变气孔开闭，直接改变扩散）。
- ❌ 漏写旱生植物的"下陷气孔"的机理 → ✅ 下陷气孔在凹槽内**积累湿润空气**→ 扩散梯度（叶内-叶外水汽差）降低 → 蒸腾↓。

### 提问设计 / Key questions
- "一株植物从明亮、干燥、有风的环境移入黑暗、潮湿、无风的地方，蒸腾速率会怎么变化？请从三个因素分别解释。"（综合推断）
- "为什么在切断枝条时要在水面下操作？"（考实验操作原理：防气泡进入木质部）

### 作业 / Homework
- `practice.md` Part A 第 6–8 题、Part C（数据题）第 1 题。
- 画 potometer 装置图并注明每步操作的原因。

| 时间 | 环节 | 老师做什么 | 学生做什么 |
|---|---|---|---|
| 0–5′ | 回顾 | 快问：potometer 测的是什么？四个影响蒸腾的因素？ | 口头作答 |
| 5–22′ | 精讲①：translocation 与源库 | 先问"叶片合成的糖如何到达根细胞？"→ 引入 translocation（韧皮部运输有机同化物）。讲解：**源**（source）= 有机物制造或输出的地方（光合活跃的叶、储存物动员中的种子）；**库**（sink）= 有机物利用或储存的地方（生长中的根、果实、茎、分生区）。运输的物质：主要是**蔗糖（sucrose）**，也含氨基酸等同化物 | 举例填表：哪些是源？哪些是库？ |
| 22–42′ | 精讲②：压力流学说（重点） | 用源（叶）→ 韧皮部 → 库（根）流程图讲解：① **源端（叶）**：伴胞利用 ATP **主动将蔗糖装载（active loading）** 进筛管 → 筛管内溶质↑→ **水势降低**；② 水由邻近木质部/叶肉细胞经**osmosis**进入筛管 → **静水压（膨压）升高**；③ 库端（根）：蔗糖被主动/被动**卸载（unloading）**到细胞 → 筛管水势升高 → 水分渗出 → 静水压降低；④ 因此筛管内产生**从高压（源端）→低压（库端）**的压力差 → 液流（mass flow）从源流向库；⑤ 水分循环：流到库端的水由木质部带走 | 画"源→筛管→库"流程图，标注水势和压力变化 |
| 42–52′ | 精讲③：支持证据 | ① **蚜虫针口法（aphid stylet）**：蚜虫刺入筛管，液体自动流出（说明有压力）；麻醉蚜虫切口后继续流出汁液可分析成分→ 证实筛管含蔗糖；② **环割实验（girding/ringing）**：去除树皮（韧皮部）但保留木质部 → 有机物堆积在切口上方（说明有机物在韧皮部从上向下运输）；③ **放射性示踪**：用 ¹⁴CO₂ 饲叶片，追踪 ¹⁴C-蔗糖在韧皮部中运动 | 记录三个证据；思考每个证据支持学说的哪个方面 |
| 52–60′ | 综合练习与总结 | 出示 2 道综合题；对照 `practice-answers.md` 自评；强调 translocation 考点与 cohesion-tension theory 对比 | 完成限时练习，对答案，归档错题 |

### 板书/重点 / Board work
- **压力流学说流程图**（核心板书）：

```
源端（叶）
  伴胞 ATP → 主动装载蔗糖进筛管
  ↓ 筛管水势降低 → 水分由osmosis进入
  ↓ 静水压（膨压）升高 → HIGH PRESSURE
         ↓  （从高压流向低压）
库端（根/果实）
  蔗糖卸载 → 筛管水势升高 → 水分流出
  ↓ 静水压降低 → LOW PRESSURE
  ↓ 水分经木质部循环
```

- **木质部 vs 韧皮部运输对比表**（总结表）：

| 特征 | 木质部 Xylem | 韧皮部 Phloem |
|---|---|---|
| 运输物质 | 水 + 无机离子 | 蔗糖 + 有机同化物 |
| 运输方向 | 向上（根→茎→叶） | 双向（源→库） |
| 运输细胞 | 死细胞，管腔 | 活细胞（筛管），有筛板 |
| 能量需求 | 不需要（被动，蒸腾拉力） | 需要 ATP（主动装载） |
| 运输学说 | 内聚-张力学说 | 压力流学说 |

### 易错点 / Common errors
- ❌ 说韧皮部运输"不需要能量" → ✅ 源端**主动装载（active loading）蔗糖需要 ATP**，伴胞提供；只有主动装载后才产生压力差，液流本身是被动的。
- ❌ 压力流学说中"水进入筛管"原因说错 → ✅ 是因为主动装载蔗糖使**水势降低**，水由**osmosis**进入（不是主动运输水）。
- ❌ 说 translocation 只向上或只向下 → ✅ 韧皮部运输**方向由源库决定**，可以双向（如同一株植物叶→果实向下，叶→茎尖向上）。
- ❌ 环割实验没有解释为什么有机物堆积在切口上方 → ✅ 有机物**无法通过**被去除的韧皮部继续向下运输，所以堆积在切口上方（证明有机物在韧皮部中运输）。
