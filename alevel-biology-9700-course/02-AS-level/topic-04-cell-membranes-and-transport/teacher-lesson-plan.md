### 教学目标 / Learning objectives

### 板书/重点 / Board work
- **膜成分-功能速查表**（学生讲义中有完整版，此处画简化版）：
  | 成分 | 位置 | 功能关键词 |
  |---|---|---|
  | 磷脂双层 | 主体骨架 | 屏障、选择性通透 |
  | 胆固醇 | 插入磷脂之间 | 调节流动性与稳定性 |
  | 内嵌蛋白 | 贯穿双层 | 通道/载体/受体/酶 |
  | 外周蛋白 | 附于膜表面 | 酶、结构支撑 |
  | 糖蛋白/糖脂 | 外表面（糖链朝外） | 细胞识别、受体、信号 |

### 易错点 / Common errors
- ❌ 说胆固醇"增加流动性" → ✅ 胆固醇**调节（regulate）**流动性：高温时**降低**流动性（稳定磷脂排列），低温时**防止**流动性过低（阻止磷脂固化）。避免笼统说"增加"。
- ❌ 混淆 intrinsic protein 与 extrinsic protein → ✅ intrinsic 贯穿双层（也叫 integral protein）；extrinsic 只附着于一侧表面（也叫 peripheral protein）。
- ❌ 说糖蛋白"在膜两侧均有" → ✅ 糖链只朝向**胞外**（extracellular）一侧，形成糖萼（glycocalyx）。

- "如果把细胞放在热水中，膜会发生什么变化？胆固醇如何应对？"（引出胆固醇调节功能）
- "为什么 O₂ 能直接穿过磷脂双层，而葡萄糖不能？"（铺垫下节被动运输分类）

## 课时 4.2 — 被动运输 / Passive Transport（60 min）

### 教学目标 / Learning objectives
- [ ] **Distinguish** 简单扩散（simple diffusion）、易化扩散（facilitated diffusion）与渗透（osmosis）
- [ ] **Explain** 影响扩散速率的因素（菲克定律定性理解）
- [ ] **Define** osmosis（CIE 官方措辞）
- [ ] **Explain** 通道蛋白与载体蛋白在易化扩散中的不同机制

### 必备术语 / Key terms
| English | 中文 |
|---|---|
| simple diffusion | 简单扩散 |
| facilitated diffusion | 易化扩散 |
| concentration gradient | 浓度梯度 |
| channel protein | 通道蛋白 |
| carrier protein | 载体蛋白 |
| osmosis | 渗透（作用） |
| partially permeable membrane | 半透膜/选择性通透膜 |
| water potential (Ψ) | 水势 |
| Fick's law | 菲克定律（定性） |

### 授课流程 / Lesson flow
| 时间 | 环节 | 内容 |
|---|---|---|
| 0–5′ | 回顾 | 膜成分速问：胆固醇功能？糖蛋白位置？ |
| 5–22′ | 精讲 | 被动运输共同点：**顺浓度梯度（high→low）、不需 ATP**。分支：①**简单扩散**：小分子非极性（O₂、CO₂、乙醇、甘油、水分子本身）直接穿越磷脂双层疏水核心；②**易化扩散**：极性/离子/大分子（葡萄糖、氨基酸、Na⁺、K⁺）需借助蛋白——通道蛋白（形成亲水孔道，水分子和特定离子通过）or 载体蛋白（构象变化，与底物结合→改变构象→释放，具特异性，可饱和） |
| 22–35′ | 精讲 | **渗透 osmosis**：定义（官方措辞）；水分子经半透膜从高水势区域移向低水势区域的净运动；水势概念铺垫（下节深讲） |
| 35–48′ | 精讲 | **影响扩散速率的因素（Fick 定律思路）**：速率 ∝ (表面积 × 浓度差) / (扩散距离 × 膜厚)；各因素逐一解释（表面积大→更多碰撞位点；浓度差大→净运动快；距离短→到达快；温度高→分子动能大→运动更快）；通道蛋白数量也影响易化扩散速率 |
| 48–57′ | 真题型 | 练"explain why glucose enters cells by facilitated diffusion and not simple diffusion""state two factors that affect the rate of diffusion" | 限时作答 |
| 57–60′ | 总结 | 三类被动运输对比表；osmosis 定义默写 |

### 板书/重点 / Board work
- **三类被动运输对比表**：
  | | 简单扩散 | 易化扩散 | 渗透 |
  |---|---|---|---|
  | 物质 | 小/非极性 O₂、CO₂ | 极性/离子/大分子 | 水 |
  | 蛋白协助 | 否 | 是（通道/载体） | 否（通过磷脂/水通道 aquaporin） |
  | 需 ATP | 否 | 否 | 否 |
  | 方向 | 顺梯度 | 顺梯度 | 顺水势梯度 |

- **Fick 定律公式框**（定性）：Rate ∝ Surface area × Concentration difference / Diffusion distance

### 易错点 / Common errors
- ❌ 说载体蛋白"像水管通道" → ✅ 通道蛋白是**孔道（channel）**，直接通过；载体蛋白需**构象变化（conformational change）**——是两种不同机制。
- ❌ 说易化扩散"需要 ATP" → ✅ 易化扩散是**被动**（passive）的，利用浓度梯度，**不需 ATP**；需要 ATP 的是主动运输。
- ❌ 定义 osmosis 只说"水从低浓度到高浓度" → ✅ CIE 要求用**水势（water potential）**语言，且强调是**净运动（net movement）**、通过**半透膜（partially permeable membrane）**。

### 提问设计 / Key questions
- "通道蛋白与载体蛋白都能帮助物质进入细胞，有什么本质区别？各自用什么分子举例？"
- "菲克定律告诉我们，肺泡表面积大、壁薄——这如何影响气体交换速率？"（跨主题 synoptic，Topic 9）

### 作业 / Homework
- `practice.md` Part A 第 4–6 题、Part B 第 2 题。

## 课时 4.3 — 渗透与水势计算 / Osmosis & Water Potential（60 min）

### 教学目标 / Learning objectives
- [ ] **Use** 公式 **Ψ = Ψs + Ψp** 分析细胞水势，并计算溶液/细胞的水势
- [ ] **Predict** 净水流方向（水从高水势→低水势）
- [ ] **Describe** 植物细胞在高张/低张溶液中的变化（膨胀→质壁分离）
- [ ] **Describe** 动物细胞在高张/低张溶液中的变化（皱缩→溶血）
- [ ] **State** 初始质壁分离 / 等渗条件

### 必备术语 / Key terms
| English | 中文 |
|---|---|
| water potential (Ψ) | 水势 |
| solute potential (Ψs) | 溶质势 |
| pressure potential (Ψp) | 压力势 |
| turgid / turgor pressure | 膨胀/膨压 |
| plasmolysis | 质壁分离 |
| incipient plasmolysis | 初始质壁分离 |
| haemolysis | 溶血（动物细胞在低张液中裂解） |
| crenation | 皱缩（动物细胞在高张液中） |
| hypertonic / hypotonic / isotonic | 高张/低张/等张溶液 |

### 授课流程 / Lesson flow
| 时间 | 环节 | 内容 |
|---|---|---|
| 0–5′ | 回顾 | 渗透定义默写；osmosis 与 facilitated diffusion 的区别 |
| 5–20′ | 精讲 | **水势概念建立**：Ψ（psi，水势）= Ψs（溶质势，恒负，因溶质降低水分子自由度）+ Ψp（压力势，可正/负/零）；纯水 Ψ = 0（最高水势）；溶质加入→Ψs 变负→水势降低；板书公式，逐步讲解各项含义和符号约定 |
| 20–38′ | 精讲 + 示范 | **植物细胞变化**：①低张液（外液 Ψ > 细胞 Ψ）→水进入→膨压升高→细胞膨胀(turgid)，当 Ψp 升至 Ψp = −Ψs 时，全细胞 Ψ 等于外液 Ψ，净流量为 0；②高张液（外液 Ψ < 细胞 Ψ）→水外出→膨压下降→细胞变松弛，继续失水→原生质体从壁剥离（plasmolysis），当 Ψp = 0 时即**初始质壁分离（incipient plasmolysis）**；**动物细胞**：无壁无法抵抗→低张液：溶血(haemolysis)；高张液：皱缩(crenation) |
| 38–52′ | 计算练习 | 老师板演 1 题（给出 Ψs 和 Ψp 求 Ψ，判断净流向），再学生做 2 题含水势计算与净流向判断；强调带单位（kPa） | 完成计算并互批 |
| 52–60′ | 总结 | 植物/动物细胞在不同溶液中变化总结表；初始质壁分离的判断标准（Ψp = 0） |

### 板书/重点 / Board work
- **水势公式框**：Ψ = Ψs + Ψp（单位 kPa）
  - 纯水：Ψ = 0；溶液：Ψs < 0，故 Ψ < 0
  - 细胞：膨胀时 Ψp > 0；初始质壁分离时 Ψp = 0；萎蔫时 Ψp 可为负
- **净流向口诀**：水往"势低处"流（从高水势→低水势）
- **植物细胞状态表**：
  | 外液 vs 细胞 Ψ | 水流向 | 细胞状态 |
  |---|---|---|
  | 外液 Ψ **高于**细胞 Ψ | 水进入 | 膨胀(turgid) |
  | 外液 Ψ = 细胞 Ψ | 净流量 = 0 | 平衡(incipient or equilibrium) |
  | 外液 Ψ **低于**细胞 Ψ | 水外出 | 质壁分离(plasmolysis) |

### 易错点 / Common errors
- ❌ 水势公式写成 Ψ = Ψs − Ψp → ✅ 正确写法：**Ψ = Ψs + Ψp**（Ψs 本身已是负值，不要再减）。
- ❌ 说"溶液浓度高 → 水势高" → ✅ 正相反：溶液浓度越高，Ψs 越负，水势越**低**。
- ❌ 说初始质壁分离时"细胞刚刚开始失水" → ✅ 初始质壁分离（incipient plasmolysis）是指原生质体**刚刚与壁分离**，此时 Ψp = 0（膨压降为零）——之前已经失去相当多水了。
- ❌ 动物细胞在低张液中说"膨胀" → ✅ 应说**溶血(haemolysis)**，因为没有壁，细胞破裂。

### 提问设计 / Key questions
- "一个植物细胞 Ψs = −1200 kPa，Ψp = 600 kPa，外液水势 = −800 kPa，水会向哪里流？最终平衡时 Ψp 是多少？"（引出计算推理）
- "为什么植物叶片萎蔫时（plasmolysis），可以通过加水恢复，而溶血的红细胞不能恢复？"（synoptic：细胞壁 vs 无壁）

- `practice.md` Part A 第 7–8 题、Part B 第 3 题、Part C 第 1 题（水势计算）。

## 课时 4.4 — 主动运输与协同运输 / Active Transport & Co-transport（60 min）

### 教学目标 / Learning objectives
- [ ] **Explain** 主动运输（active transport）的机制及其与被动运输的区别
- [ ] **Describe** 钠钾泵（Na⁺/K⁺ pump）的工作原理（逆梯度、ATP 水解、构象变化）
- [ ] **Explain** 小肠上皮细胞中葡萄糖的协同运输（co-transport）机制
- [ ] **Compare** 主动运输与易化扩散（成对）

### 必备术语 / Key terms
| English | 中文 |
|---|---|
| active transport | 主动运输 |
| against the concentration gradient | 逆浓度梯度 |
| carrier protein (pump) | 载体蛋白（泵） |
| ATP hydrolysis | ATP 水解 |
| Na⁺/K⁺ pump | 钠钾泵 |
| co-transport | 协同运输 |
| sodium-glucose co-transporter | 钠-葡萄糖协同运输蛋白 |
| lumen / ileum | 肠腔/回肠 |

### 授课流程 / Lesson flow
| 时间 | 环节 | 内容 |
|---|---|---|
| 0–5′ | 回顾 | 水势计算快测一道；osmosis 定义默写 |
| 5–20′ | 精讲 | **主动运输**：逆浓度梯度（against concentration gradient）→ 热力学上不自发 → 需**ATP 水解**供能；载体蛋白（泵）结合底物+ATP → 磷酸化 → 构象变化 → 将物质运到高浓度侧；举例：Na⁺/K⁺ pump——每消耗 1 ATP，泵出 3 Na⁺、泵入 2 K⁺（维持神经冲动、细胞体积）；植物根细胞从低浓度土壤液吸收矿质离子 |
| 20–38′ | 精讲 | **协同运输（co-transport）**：以小肠葡萄糖吸收为例：①Na⁺/K⁺ pump（主动）在细胞基膜侧泵出 Na⁺，维持肠腔侧 Na⁺ 高浓度；②Na⁺ 顺梯度经 Na⁺-葡萄糖协同运输蛋白（SGLT）**同向**进入细胞，同时驱动葡萄糖**逆梯度**进入细胞（葡萄糖借 Na⁺ 的势能"搭便车"）；③葡萄糖再从细胞基底侧经易化扩散（GLUT 载体）进入血液 |
| 38–52′ | 真题型 | 练"Explain why active transport requires ATP""Outline the mechanism by which glucose is absorbed from the small intestine" | 限时 |
| 52–60′ | 总结 | 被动运输 vs 主动运输成对对比表；协同运输图示解读 |

### 板书/重点 / Board work
- **主动 vs 被动运输成对对比表**：
  | 特征 | 被动运输 | 主动运输 |
  |---|---|---|
  | 方向 | 顺浓度梯度 | 逆浓度梯度 |
  | ATP | 不需 | 需要 |
  | 载体蛋白 | 易化扩散需要；扩散不需 | 需要（泵） |
  | 可饱和 | 易化扩散可 | 可 |

- **协同运输图**（肠腔→细胞→血液三格，标注 Na⁺ 主动泵出、Na⁺+葡萄糖协同进入、葡萄糖易化扩散出去）。

### 易错点 / Common errors
- ❌ 说主动运输"需要蛋白质，所以是主动的" → ✅ 决定"主动"的是**逆浓度梯度 + 需 ATP**，不是蛋白质的存在（易化扩散也需蛋白却是被动）。
- ❌ 说协同运输"葡萄糖是主动运输" → ✅ 葡萄糖进入肠细胞是**由 Na⁺ 梯度驱动的**（间接利用 ATP，也叫 secondary active transport），但葡萄糖本身的运动是借助 Na⁺ 的势能。
- ❌ 漏掉基膜侧的葡萄糖去向 → ✅ 葡萄糖从细胞基底侧**经易化扩散（GLUT）**进入血液，两侧膜的机制不同，完整描述才得分。

### 提问设计 / Key questions
- "如果向细胞提供氰化物（抑制线粒体/ATP 合成），主动运输会停止，而扩散不受影响——为什么？"（迁移：synoptic Topic 3 酶抑制剂、Topic 12 呼吸）
- "协同运输中，葡萄糖进入肠细胞的过程是主动还是被动？为什么这个问题有争议？"（引导批判性思维）

- `practice.md` Part A 第 9–10 题、Part B 第 4 题。

### 授课流程 / Lesson flow
| 时间 | 环节 | 内容 |
|---|---|---|
| 0–5′ | 回顾 | 协同运输步骤快速复述 |
| 5–20′ | 精讲 | **胞吞胞吐（批量运输 bulk transport）**：所有都需**ATP**，以囊泡为基本单元；①**吞噬 phagocytosis**：细胞伸出伪足包裹大颗粒（细菌/异物），形成吞噬体 → 与溶酶体融合消化（Topic 11 吞噬细胞）；②**胞饮 pinocytosis**：胞膜内陷包裹小液滴或溶液；③**胞吐 exocytosis**：分泌囊泡与细胞膜融合，将物质分泌到胞外（如神经末梢释放递质、腺体分泌激素）。与 Topic 1 蛋白质分泌途径（exocytosis 最后一步）串联 |
| 20–38′ | 实验讲解 | **甜菜根（beetroot）膜透性实验**：原理——甜菜根液泡含红色素（betalain），正常情况下膜阻止色素外漏；若膜被破坏（高温 or 有机溶剂）→通透性增大→色素渗出→溶液变红。定量方法：用**比色计(colorimeter)**测溶液**吸光度(absorbance)**，吸光度↑ = 色素渗出越多 = 膜损伤越大。自变量：温度（20–80°C 梯度）or 乙醇浓度；因变量：吸光度；控制变量：甜菜根圆片大小/厚度/数量、浸泡时间、水体积 |

| 52–60′ | 总结讲评 | 对 `practice-answers.md`，标记丢分类型；主题总复盘 | 错题归档 |

### 板书/重点 / Board work
- **胞吞胞吐图示**：画膜包裹 → 囊泡形成 → 与溶酶体/分泌 融合的简化流程。
- **甜菜根实验变量表**：
  | 变量类型 | 内容 |
  |---|---|
  | 自变量 | 温度（或有机溶剂浓度） |
  | 因变量 | 溶液吸光度（比色计量） |
  | 控制变量 | 圆片大小与数量、浸泡时间、水的体积 |
  | 预期结果 | 温度↑ → 吸光度↑（高于约 40°C 后显著上升） |

### 易错点 / Common errors
- ❌ 说吞噬作用"不需要能量" → ✅ 内吞胞吐都是**主动过程**，需要 ATP（膜的变形运动需要能量）。
- ❌ 甜菜根实验说"色素是从膜表面渗出" → ✅ 色素储存在**液泡**中，需穿越**液泡膜 + 细胞质膜**两层膜才能到达外部溶液；两层膜都被破坏才有大量色素渗出。
- ❌ 甜菜根实验控制变量漏"圆片大小/表面积" → ✅ 这是关键控制变量，影响接触面积。
- ❌ 比色计设置说"读红色/绿色" → ✅ 测红色素的吸光度，应选**绿色滤光片（green filter）**，因红色素吸收绿光最强。
