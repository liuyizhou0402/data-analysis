# Topic 10 — 传染病 / Infectious Disease｜教师教案（3 课时）

## 主题概览 / Topic overview
- **对应考纲**：9700 — 10.1 Infectious diseases；10.2 Antibiotics and resistance
- **对应试卷**：Paper 1（选择，含耐药机制/抗生素靶点）、Paper 2（结构题：疾病对照/控制措施/耐药选择压力）
- **前置知识**：Topic 1（原核细胞结构、70S 核糖体、肽聚糖）；Topic 6（基因突变）；A2 Topic 17（自然选择——synoptic 连接）
- **本主题 A\* 关键**：
  1. **四大疾病精准三要素**（病原体种类/传播途径/控制措施）——年年考对照表式结构题
  2. **抗生素靶点逻辑**：靶向原核特有结构 → 对真核（人体）无害（与 Topic 1 强连接）
  3. **耐药性 = 自然选择过程**：突变→选择压力→耐药株扩散，步骤不能颠倒（与 Topic 17 连接）
  4. **数据分析**：发病率/患病率/控制措施成效——AO2 高分必掌握

## 课时 10.1（课时 41）— 病原体与传染病 / Pathogens & Infectious Disease（60 min）

### 教学目标 / Learning objectives
学生在本课后能够：
- [ ] **Define** communicable disease、pathogen，并区分传染病与非传染病
- [ ] **State** 霍乱、疟疾、结核（TB）、HIV/AIDS 的病原体类型（细菌/原生生物/病毒）
- [ ] **Describe** 各疾病的主要传播途径
- [ ] **State** 各疾病的主要控制措施

### 必备术语 / Key terms
| English | 中文 |
|---|---|
| communicable (infectious) disease | 传染病 |
| non-communicable disease | 非传染病 |
| pathogen | 病原体 |
| Vibrio cholerae | 霍乱弧菌（细菌） |
| Plasmodium | 疟原虫（原生生物） |
| Mycobacterium tuberculosis | 结核分枝杆菌（细菌） |
| HIV (Human Immunodeficiency Virus) | 人类免疫缺陷病毒 |
| vector | 传播媒介 |
| Anopheles | 按蚊 |
| oral rehydration therapy (ORT) | 口服补液疗法 |
| BCG vaccine | 卡介苗 |
| antiretroviral drug | 抗逆转录病毒药物 |

### 授课流程 / Lesson flow
| 时间 | 环节 | 老师做什么 | 学生做什么 |
|---|---|---|---|
| 0–5′ | 导入 | 展示全球疾病死亡数据（霍乱/TB/疟疾/AIDS 对比），问"这些病的共同点是什么？如何与心脏病区分？" | 讨论、猜想 |
| 5–20′ | 精讲 | 讲 communicable vs non-communicable disease；定义 pathogen（能导致疾病的微生物/病原体，包括细菌、病毒、原生生物、真菌）；强调"传染病 = 可在人之间/通过媒介传播" | 记笔记、补充术语表 |
| 20–40′ | 精讲（重点） | 逐一讲四大疾病，每个用"病原体→传播→控制"三要素框架；用疾病速记对照表引导学生填写（边讲边填，不要求课前背好）；重点讲疟疾媒介传播（vector transmission）和 TB 空气飞沫途径 | 在学生讲义上填写疾病对照表 |
| 40–52′ | 真题型 | 出示"describe the transmission of cholera"和"suggest two measures to control malaria"两题，学生限时作答 | 作答（3 min/题），展示优秀作答 |
| 52–60′ | 总结 | 锁定四大疾病的英文病原体名称（记忆提示）；强调"病原体类型是必考的第一个采分点" | 默写对照表首列（病原体） |

### 板书/重点 / Board work
**疾病速记对照表**（黑板展示骨架，学生填写）：
| 疾病 | 病原体（学名+类型） | 传播途径 | 控制措施 |
|---|---|---|---|
| 霍乱 Cholera | *Vibrio cholerae*（细菌） | 污染水/食物 | 清洁水源；口服补液盐 |
| 疟疾 Malaria | *Plasmodium*（原生生物） | 按蚊（*Anopheles*）叮咬媒介 | 灭蚊/蚊帐/驱虫剂 |
| 结核 TB | *Mycobacterium tuberculosis*（细菌） | 飞沫/空气吸入 | BCG 疫苗；抗生素完整疗程 |
| HIV/AIDS | HIV（病毒） | 体液/性接触/血液/母婴 | 安全性行为；抗逆转录病毒药；教育 |

### 易错点 / Common errors
- ❌ 把 *Plasmodium*（疟原虫）说成"细菌" → ✅ 是**原生生物（protozoan / protist）**。
- ❌ 霍乱传播说"直接接触感染者" → ✅ 主要是**摄入受污染的水或食物**（faecal-oral route）。
- ❌ HIV/AIDS 混淆：HIV 是病原体（病毒），AIDS 是疾病状态 → ✅ *HIV causes AIDS*，两者不能互换。
- ❌ 控制措施只写"打疫苗"用于霍乱 → ✅ 霍乱首要措施是**清洁水源 + 口服补液盐**（疫苗不是考纲核心）。

### 提问设计 / Key questions
- "为什么疟疾难以通过隔离病人来控制？"（引出媒介传播的特殊性——要控制媒介，不是控制病人）
- "HIV 目前为什么无法治愈？"（病毒整合到宿主 DNA；为下节免疫/病毒复制铺垫）

### 作业 / Homework
- `practice.md` Part A 第 1–4 题、Part B 第 1 题（四大疾病结构题）。

## 课时 10.2（课时 42）— 抗生素与耐药性 / Antibiotics & Resistance（60 min）

### 教学目标 / Learning objectives
学生在本课后能够：
- [ ] **Explain** 抗生素如何通过靶向细菌特有结构来杀死细菌
- [ ] **Explain** 为什么抗生素对病毒无效
- [ ] **Explain** 抗生素耐药性如何通过自然选择产生（突变→选择→扩散）
- [ ] **Suggest** 减少耐药性传播的措施

### 必备术语 / Key terms
| English | 中文 |
|---|---|
| antibiotic | 抗生素 |
| cell wall synthesis inhibitor | 细胞壁合成抑制剂 |
| 70S ribosome | 70S 核糖体（原核特有） |
| peptidoglycan (murein) | 肽聚糖（细菌细胞壁成分） |
| antibiotic resistance | 抗生素耐药性 |
| mutation | （基因）突变 |
| natural selection | 自然选择 |
| selection pressure | 选择压力 |
| MRSA (methicillin-resistant *Staphylococcus aureus*) | 耐甲氧西林金黄色葡萄球菌 |
| antibiotic resistance gene | 抗药性基因 |

### 授课流程 / Lesson flow
| 时间 | 环节 | 老师做什么 | 学生做什么 |
|---|---|---|---|
| 0–5′ | 回顾 | 快问四大疾病病原体（4 人各答 1 种） | 抢答 |
| 5–22′ | 精讲 | 抗生素作用机制：①靶向**细菌细胞壁合成**（如青霉素——阻止肽聚糖交联，细菌无法维持细胞壁完整性，裂解）；②靶向**70S 核糖体**（抑制蛋白质合成，如链霉素）；人体细胞**无肽聚糖壁、有 80S 核糖体**→ 不受影响；与 Topic 1 原核结构强连接 | 在讲义"结构-靶点-效果"图上标注 |
| 22–34′ | 精讲 | 抗生素为何对病毒无效：病毒**无细胞壁（无肽聚糖）、无核糖体、无细胞代谢**——没有抗生素可以靶向的结构；病毒依靠宿主细胞复制，用抗生素会损伤宿主 | 回答"如果给病毒性感冒开抗生素，会发生什么？" |
| 34–52′ | 精讲（重点） | **耐药性的产生机制**（按步骤逐一讲清）：① 细菌种群中**随机发生突变** → 少数细菌产生耐药性（如能产生 β-内酰胺酶分解青霉素）；② 使用抗生素 = **选择压力**（不耐药菌被杀死）；③ 耐药菌**存活并大量繁殖**（无竞争）→ 种群中耐药基因频率升高；④ 耐药菌扩散，**水平基因转移**（质粒）加速传播；MRSA 作为典型案例讲解 | 画"耐药性产生流程图" |
| 52–60′ | 总结 | 减少耐药性的措施：①**完成全程抗生素疗程**（不中途停药）；②避免滥用（不为病毒性感染开抗生素）；③农业/畜牧业减少预防性使用；④开发新抗生素 | 讨论：为什么不完成疗程会更危险？ |

### 板书/重点 / Board work
**抗生素靶点对比**：
| 靶点 | 细菌 | 人体细胞 |
|---|---|---|
| 细胞壁（肽聚糖） | **有** → 抗生素可抑制其合成 | **无** → 不受影响 |
| 核糖体 | **70S** → 某些抗生素靶向 70S | **80S** → 结构不同，不受靶向 |
| 细胞膜 | 有（但与真核相似，常不作靶点） | 有 |

**耐药性产生流程**：
```
随机突变 → [少数个体携带耐药基因]
    ↓ 抗生素使用（选择压力）
敏感菌被杀死 / 耐药菌存活
    ↓ 耐药菌大量繁殖
耐药菌成为种群主体 → 抗生素失效
    ↓ 水平基因转移（质粒）
耐药性向其他细菌传播（如 MRSA）
```

### 易错点 / Common errors
- ❌ "抗生素使细菌产生了突变（引起耐药性）" → ✅ 突变是**随机发生**的（早于抗生素使用）；抗生素是**选择压力**，不是突变的原因。
- ❌ "耐药是细菌主动学会的" → ✅ 是**自然选择**，是基因频率在种群中的改变，不是个体"学会"。
- ❌ 说抗生素"没有靶点"来解释对病毒无效 → ✅ 要说清楚病毒**缺乏抗生素靶向的特定结构**（无细胞壁、无 70S 核糖体）。
- ❌ 不完成疗程"没关系，症状消了就好" → ✅ 残留的少数**耐药菌存活→繁殖→未来感染更难治**。

### 提问设计 / Key questions
- "MRSA 患者不应使用普通青霉素——为什么？这反映了什么自然选择原理？"（巩固耐药机制）
- "农场给健康动物喂抗生素促进生长，这与人类健康有什么关系？"（跨情境 suggest 练习）

### 作业 / Homework
- `practice.md` Part A 第 5–8 题、Part B 第 2–3 题（抗生素机制与耐药性）。

## 课时 10.3（课时 43）— 传染病综合与数据分析 / Review & Data Analysis（60 min）

### 教学目标 / Learning objectives
学生在本课后能够：
- [ ] **Describe** 并 **Explain** 发病率/患病率数据的趋势（AO2 数据三步法）
- [ ] **Evaluate** 控制措施的有效性（引用数据支持）
- [ ] **Suggest** 为什么某地区某疾病发病率高/低
- [ ] 完成本主题综合练习并自评

### 必备术语 / Key terms
| English | 中文 |
|---|---|
| incidence rate | 发病率（单位时间新病例数） |
| prevalence | 患病率（某时刻现有病例数） |
| mortality rate | 死亡率 |
| eradication | 消灭（根除） |
| vector control | 媒介控制 |
| herd immunity | 群体免疫 |
| antiretroviral therapy (ART) | 抗逆转录病毒治疗 |

### 授课流程 / Lesson flow
| 时间 | 环节 | 老师做什么 | 学生做什么 |
|---|---|---|---|
| 0–5′ | 回顾 | 快问：1. 耐药性产生的四个步骤；2. 青霉素靶向什么结构？ | 口述 |
| 5–20′ | 精讲（数据技能） | 讲 incidence vs prevalence 区别；示范"数据三步法（Describe–Quote–Explain）"用一道 TB 发病率折线图题：①趋势+②引用数值+③解释（BCG 推广/抗生素使用） | 记三步法模板 |
| 20–40′ | 当堂练 | 发放 Part C、Part D 数据题（疟疾发病率 + 控制措施表），学生限时 20 min 作答 | 完成 Part C、Part D |
| 40–52′ | 讲评 | 对照答案逐题讲评；重点强调：①describe 趋势必须引用数据值；②evaluate 题要给出"为什么措施有效/无效"的生物学原因，不能仅说"数据下降了" | 对照自评、标注丢分类型 |
| 52–60′ | 综合复习 | 全主题 5 min 复盘：①四大疾病对照表；②抗生素靶点；③耐药性三个关键词（突变/选择压力/繁殖） | 默写关键词表，归档错题 |

### 板书/重点 / Board work
**数据题三步法提示框**：
```
Step 1 — Describe 趋势
  "The incidence of X [increases/decreases/peaks at…] between [year A] and [year B]."

Step 2 — Quote 数据（必须！）
  "…from [数值 + 单位] to [数值 + 单位], a [change quantity] [increase/decrease]."

Step 3 — Explain 原因（若题目要求）
  "This is likely because [控制措施/病原体/宿主因素], which [生物学机制]."
```

**Evaluate 控制措施框架**：
- 措施**有效**：说明机制（e.g. 蚊帐减少按蚊叮咬机会 → 减少 *Plasmodium* 传播）+ 引用数据支持
- 措施**局限**：指出原因（资源/依从性/耐药/媒介适应等）

### 易错点 / Common errors
- ❌ 数据题只写"发病率下降" → ✅ 必须引用**具体数值**（如"from 250 per 100 000 to 40 per 100 000"）。
- ❌ evaluate 只说"有效"或"无效"不给原因 → ✅ 用生物学机制解释，再引用数据支持。
- ❌ 混淆 incidence（新病例）和 prevalence（现有病例总数） → ✅ 题目会交替用，注意单位提示（每年/某时刻）。

### 提问设计 / Key questions
- "同一地区 HIV 患病率上升，但每年新增病例（incidence）下降——可能的原因是什么？"（考查 incidence vs prevalence 理解 + ART 延长寿命）
- "如何判断一项控制措施'确实有效'而非只是巧合？"（引出控制变量/实验设计思维，铺垫 P3/P5）

### 作业 / Homework
- 整理本主题错题本；预习 Topic 11 免疫"必背定义"（lymphocyte/antibody/antigen）。
