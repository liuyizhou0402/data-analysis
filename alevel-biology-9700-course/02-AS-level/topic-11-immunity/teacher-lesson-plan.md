# Topic 11 — 免疫 / Immunity｜教学设计

## 基本信息 / Overview
| 项目 | 内容 |
|---|---|
| **主题** | Topic 11: Immunity |
| **课时** | 4 × 60 min（约 240 min） |
| **考纲对应** | CIE 9700 Syllabus Section 11 |
| **前置知识** | Topic 1（细胞器）、Topic 10（感染性疾病）|
| **考试权重** | P1（5–7 题）+ P2（20–25 分）|

## 学习目标 / Learning Objectives
完成本主题后，学生能够：
1. 描述非特异性免疫（phagocytosis）的过程
2. 区分体液免疫（humoral immunity）与细胞免疫（cell-mediated immunity）
3. 描述 B 细胞、T 细胞的活化和效应细胞的功能
4. 解释克隆选择（clonal selection）和克隆扩增（clonal expansion）
5. 解释主动免疫与被动免疫的区别，以及记忆细胞的作用
6. 描述单克隆抗体（monoclonal antibodies）的制备原理及应用

## 课程安排 / Lesson Breakdown

### 第 1 课（60 min）—— 非特异性免疫与吞噬作用 Non-specific & Phagocytosis

**5 min 导入**：皮肤割破后为什么会红肿？炎症反应（inflammation）是第一道防线。

**15 min 讲授**：
第一道防线（物理/化学屏障）：
- 皮肤（不可穿透的角质层）
- 黏液（traps pathogens）+ 纤毛（cilia sweeping）
- 盐酸（胃，低 pH 杀菌）

非特异性免疫——吞噬作用（phagocytosis）：
1. 病原体进入 → 血液中**中性粒细胞（neutrophils）**和组织中**巨噬细胞（macrophages）**被招募
2. 吞噬细胞识别病原体表面抗原（非特异性）
3. 包裹病原体形成**吞噬体（phagosome）**
4. 溶酶体（lysosome）与吞噬体融合 → **水解酶消化**病原体
5. 消化碎片（抗原片段）呈递到细胞表面（MHC II）→ 成为**抗原呈递细胞（APC）**

**20 min 活动**：
- 绘制吞噬过程步骤图
- 讨论：为什么需要特异性免疫？（非特异性免疫速度快但不具记忆性）

**15 min 抗原定义**：
- 抗原（antigen）：激发免疫反应的分子（通常是蛋白质/糖蛋白）
- 自身抗原 vs 外来抗原（self vs non-self 识别）
- 淋巴细胞来源：骨髓（B 细胞成熟于骨髓，T 细胞成熟于胸腺）

**5 min 总结**

---

### 第 2 课（60 min）—— 体液免疫 Humoral Immunity

**5 min 复习**：抗原呈递细胞（APC）已呈递抗原片段，引出特异性免疫。

**25 min 讲授**：
**B 细胞激活过程**（体液免疫，针对胞外抗原）：
1. B 细胞膜上有**特异性抗体（B cell receptor）**——与特定抗原互补
2. **克隆选择（clonal selection）**：特定 B 细胞被抗原选中
3. **T helper 细胞（Th）释放淋巴因子（lymphokines）**激活被选中的 B 细胞
4. **克隆扩增（clonal expansion）**：B 细胞快速有丝分裂
5. 分化为：
   - **浆细胞（plasma cells）**：大量分泌抗体（5000 个/min）
   - **记忆细胞（memory B cells）**：长期存活，再次感染时快速应答

**抗体结构**：Y 形糖蛋白，两条重链 + 两条轻链，**可变区（variable region）**与抗原结合，**恒定区（constant region）**激活免疫效应。

**抗体作用**：
- **Agglutination（凝集）**：多价抗体使病原体团聚，易被吞噬
- **Opsonisation（调理）**：包被病原体，促进吞噬
- **Neutralisation（中和）**：阻断病毒/毒素与宿主细胞结合
- **Complement activation（补体激活）**：裂解细菌细胞膜

**15 min 活动**：
- 学生画出 B 细胞激活→浆细胞→抗体分泌的流程图
- 分析：初次免疫应答 vs 二次免疫应答图（抗体浓度 vs 时间）

**15 min 记忆细胞**：
- 初次接触抗原：潜伏期长，抗体浓度低
- 再次接触：**记忆细胞迅速激活**→更短潜伏期、更高抗体浓度、更长持续时间
- 疫苗原理：使身体产生记忆细胞，无需真实感染

**5 min 总结**

---

### 第 3 课（60 min）—— 细胞免疫与免疫记忆 Cell-mediated Immunity & Memory

**5 min 复习**：体液免疫针对胞外病原体。

**20 min 讲授**：
**T 细胞激活过程**（细胞免疫，针对胞内病原体/肿瘤/移植细胞）：
1. APC 呈递抗原 → 特异性 **T 细胞（T helper/Tc）**被激活
2. **T helper 细胞（Th）**：释放细胞因子激活 B 细胞和 Tc 细胞
3. **细胞毒性 T 细胞（Tc，cytotoxic T cells）**：
   - 识别被病毒感染的细胞（或肿瘤细胞）表面 MHC I + 病毒抗原
   - 释放**穿孔素（perforin）**→ 打穿靶细胞膜 → 靶细胞凋亡（apoptosis）
4. **记忆 T 细胞**：长期存活

**HIV 与细胞免疫的关联**（复习 Topic 10）：HIV 摧毁 T helper 细胞 → 体液免疫和细胞免疫均受损。

**20 min 主动 vs 被动免疫**：
| | 主动免疫 Active | 被动免疫 Passive |
|---|---|---|
| 来源 | 自身免疫系统产生抗体 | 外部提供抗体 |
| 持续时间 | **长期**（有记忆细胞）| **短期**（无记忆细胞，抗体降解）|
| 保护时机 | 慢（需数天/周建立）| **快**（立即有效）|
| 举例 | 疫苗（灭活/减毒/亚单位）| 破伤风血清、母乳（IgA）|

**自然主动**（感染后恢复）vs **人工主动**（疫苗）
**自然被动**（母乳/胎盘抗体）vs **人工被动**（注射抗血清）

**15 min 往年题分析**：主动/被动免疫对比题

**5 min 总结**

---

### 第 4 课（60 min）—— 单克隆抗体及应用 Monoclonal Antibodies

**5 min 导入**：抗体高度特异性 → 如果能大量制备特定抗体，有什么应用？

**20 min 讲授**：
单克隆抗体（monoclonal antibodies, mAbs）制备步骤：
1. 向小鼠注射目标抗原 → 小鼠产生特异性 B 细胞
2. 取出脾脏中的 **B 细胞**
3. 与**骨髓瘤细胞（myeloma cells，肿瘤细胞，可无限分裂）**融合
4. 融合形成**杂交瘤细胞（hybridoma cells）**
5. 选择性培养（HAT 培养基）→ 只有杂交瘤细胞存活
6. **克隆化**：筛选产生目标抗体的克隆
7. 大量培养 → 收获单克隆抗体

**应用**：
- **妊娠检测**（hCG 抗体）
- **癌症诊断和治疗**（herceptin，靶向 HER2 乳腺癌）
- **COVID-19 检测**（侧流免疫层析，lateral flow test）
- 科研（标记特定蛋白）

**20 min 活动 + 往年题**：
- 分析侧流检测原理（两条线的含义）
- 单克隆抗体制备步骤的顺序题

**15 min 综合复习**：
- 免疫系统总结图（吞噬细胞→APC→T/B 细胞激活→效应）
- 体液 vs 细胞免疫对比

**5 min 总结 + 自测清单检查**

## 评分考点 / Key Exam Points
1. 克隆选择 = 特定 B/T 细胞被对应抗原选中；克隆扩增 = 该细胞快速有丝分裂
2. 浆细胞 = 分泌抗体（不是记忆细胞）；记忆细胞 = 长期存活
3. 主动免疫记忆细胞存在 → 持久；被动免疫无记忆细胞 → 短暂
4. 杂交瘤细胞：B 细胞（特异性）× 骨髓瘤细胞（无限增殖）
5. 抗体 Y 形：两个抗原结合位点（variable region 互补于抗原）

---

## 常见误区 / Common Misconceptions
- "浆细胞也能变成记忆细胞"——**错**，两者是 B 细胞克隆扩增后的不同分化路径
- "抗体杀死病原体"——抗体标记/凝集/中和，**吞噬细胞**才负责消灭
- "主动免疫立即有效"——**错**，主动免疫需要数天/周建立（建立记忆细胞需要时间）
- "疫苗包含活的病原体"——减毒（attenuated）疫苗含活但弱化的，灭活（killed）疫苗含死的——不同类型
