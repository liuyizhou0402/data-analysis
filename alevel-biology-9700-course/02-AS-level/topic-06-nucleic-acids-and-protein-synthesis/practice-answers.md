## Part A — 选择题答案

| 题号 | 答案 | 为何对 / 干扰项为何错 |
|:--:|:--:|---|
| 1 | **C** | DNA 含**脱氧核糖（deoxyribose）**；RNA 含**核糖（ribose）**。A 反了（DNA 含 T，RNA 含 U）；B 错（有些 RNA 如 tRNA 有部分双链区）；D 错。 |
| 2 | **C** | **解旋酶（Helicase）**解开双螺旋/断 H 键。A DNA 聚合酶合成新链；B RNA 聚合酶转录；D 连接酶连接片段。 |
| 3 | **C** | 模板链（3'→5'）=TACGGC，则 mRNA（5'→3'）= **AUGCCG**（U 代替 T）。注意：转录按模板链互补，方向 5'→3'。 |
| 4 | **B** | 遗传密码：**不重叠（non-overlapping）+ 简并（degenerate）**（多个密码子可编码同一氨基酸）。A 错（一个密码子对应一个 aa）；C 错（密码子通用）；D 错（简并性）。 |
| 5 | **C** | 反密码子 3'–UAC–5' → 密码子 5'–AUG–3' → **甲硫氨酸（Methionine）**。 |
| 6 | **C** | UUU→UUC 均编码苯丙氨酸（phenylalanine），蛋白不变 = **沉默突变（silent）**。 |
| 7 | **B** | 碱基插入→移码（frameshift）→之后所有密码子**全部改变**→蛋白质从该点开始大量改变。 |
| 8 | **B** | **翻译（translation）**：mRNA → 多肽链，在核糖体上，tRNA 参与。A 是转录；C 是逆转录；D 错。 |
| 9 | **C** | GAG→GUG，碱基替换→氨基酸改变（Glu→Val）→**错义突变（missense）**。Silent 无变化；frameshift 是插入/缺失；nonsense 产生终止密码。 |
| 10 | **B** | 3 代后 8 个 DNA 分子，半保留复制：只有最外两条 = 原始链各 1 条 → **2 个**分子含原始链。 |

## Part B — 结构题（按采分点 ✓ 拆解）

### B1
**(a) Structure of DNA nucleotide. [3]** — *Describe*
- ✓ A **deoxyribose** (pentose) sugar. **[1]**
- ✓ A **phosphate** group attached to the 5' carbon. **[1]**
- ✓ One of four **nitrogenous bases**: A, T, G, or C (accept drawing/naming). **[1]**

**(b) Compare DNA and mRNA. [3]** — *Compare（三组成对）*
- ✓ DNA has **deoxyribose**; mRNA has **ribose**.
- ✓ DNA contains **thymine (T)**; mRNA contains **uracil (U)**.
- ✓ DNA is **double-stranded**; mRNA is **single-stranded**.

### B2
**(a) Describe semi-conservative replication. [4]** — *Describe（步骤）*
- ✓ **Helicase** unwinds the double helix and breaks **hydrogen bonds** between base pairs. **[1]**
- ✓ Each strand acts as a **template**; free DNA nucleotides align by **complementary base pairing**. **[1]**
- ✓ **DNA polymerase** joins (condenses) nucleotides in the 5'→3' direction. **[1]**
- ✓ Each new DNA molecule has **one original and one new strand** (hence "semi-conservative"). **[1]**

**(b) Meselson-Stahl result supports semi-conservative. [3]** — *Explain*
- ✓ After 1st replication: all DNA **intermediate density** → every molecule has one ¹⁵N strand and one ¹⁴N strand → consistent with each original strand being a template. **[1]**
- ✓ After 2nd replication: 50% intermediate + 50% light → the ¹⁴N templates from generation 1 are now being used, producing all-¹⁴N molecules. **[1]**
- ✓ This pattern **could only occur** if original strands are conserved and used as templates (rules out conservative and dispersive models). **[1]**

### B3
**(a) Number of amino acids coded. [2]** — *State and Explain*
- ✓ **4** amino acids. **[1]**
- ✓ UAG is a **stop codon** — it does not code for an amino acid; 4 codons (AUG, CAG, UGU, AAA) code for amino acids. **[1]**

**(b) Effect of AAA→AAG. [2]** — *Suggest and Explain*
- ✓ Both AAA and AAG code for **lysine** (this is a silent/synonymous mutation). **[1]**
- ✓ **No change** to the amino acid sequence / protein is unchanged (degenerate code). **[1]**

**(c) Effect of deleting one base from codon 2. [3]** — *Explain*
- ✓ Deletion of one base causes a **frameshift mutation**. **[1]**
- ✓ All codons **after the deletion** are shifted → all amino acids from position 2 onwards are different. **[1]**
- ✓ Likely to produce a **premature stop codon** / truncated / non-functional protein. **[1]**

### B4
**(a) Transcription. [4]** — *Describe（步骤）*
- ✓ **RNA polymerase** binds to the promoter region of the gene on the **template strand**. **[1]**
- ✓ Hydrogen bonds between DNA strands **break** locally; template strand is exposed. **[1]**
- ✓ Free **RNA nucleotides** align by complementary base pairing (A→U, T→A, G→C, C→G). **[1]**
- ✓ RNA polymerase joins nucleotides by **phosphodiester bonds** to form mRNA; mRNA is synthesised **5'→3'**. **[1]**

**(b) Structure and function of tRNA. [3]** — *Describe*
- ✓ **Clover-leaf** shape with hydrogen-bonded stem-loops; about 80 nucleotides. **[1]**
- ✓ Has an **anticodon loop** (three bases that are complementary to the mRNA codon). **[1]**
- ✓ Has a **3' CCA attachment site** for the specific amino acid; carries amino acid to ribosome for translation. **[1]**

## Part C — 数据题

### C1
**(a) Calculate number of amino acids. [2]** — *Calculate*
- ✓ Number of codons = 900 / 3 = 300. **[1]**
- ✓ 300 − 1 (stop codon) = **299 amino acids**. **[1]**

**(b) Two ways protein may be affected. [2]** — *Suggest*
- ✓ Glycine is **non-polar**; arginine is **positively charged** → different **R group interactions** in tertiary structure (loss of hydrophobic interaction / gain of ionic bond with charged residue). **[1]**
- ✓ Change in **tertiary/quaternary structure** → change in shape of active site (if enzyme) → reduced / altered function. **[1]**

**(c) Effect of nonsense mutation. [2]** — *Explain*
- ✓ A stop codon is introduced **at codon 101**. **[1]**
- ✓ Translation terminates prematurely → a **truncated polypeptide** (shorter than normal) is produced; likely **non-functional**. **[1]**

### C2
**(a) Identify organism that cannot have double-stranded DNA. [2]** — *Identify and Explain*
- ✓ **Organism R** (% A = 30, %T = 20 → A ≠ T). **[1]**
- ✓ In double-stranded DNA, A pairs with T and G pairs with C → %A must equal %T and %G must equal %C (Chargaff's rules). R violates this → must be single-stranded DNA (or RNA). **[1]**

**(b) Ratio of purines to pyrimidines for P; what it shows. [2]** — *Calculate and Explain*
- ✓ Purines (A+G) = 28+22 = 50; Pyrimidines (T+C) = 28+22 = 50; ratio = **1:1**. **[1]**
- ✓ This confirms that **one purine is always paired with one pyrimidine** in double-stranded DNA → the two strands are complementary / antiparallel structure. **[1]**

**(c) Suggest why %G = %C. [2]** — *Suggest*
- ✓ G on one strand **always pairs with C** on the complementary strand (three hydrogen bonds). **[1]**
- ✓ Therefore the total number of G bases in both strands = total number of C bases → **%G = %C** throughout the molecule. **[1]**

## Part D — 计算题

### D1
**(a) Total nucleotides. [1]**
- ✓ 600 base pairs × 2 = **1200 nucleotides**. **[1]**

**(b) Molecules with at least one original strand after 2 rounds. [2]**
- ✓ Semi-conservative: after 1st round → 2 molecules, each with 1 original strand → 2 original strands distributed. **[1]**
- ✓ After 2nd round → 4 molecules; original strands still in 2 of them (each original strand in separate molecule) → **2 molecules**. **[1]**

**(c) Calculate number of G nucleotides. [2]**
- ✓ If A = 32%, then T = 32% (complementary), G = C = (100 − 64)/2 = **18%**. **[1]**
- ✓ Total nucleotides = 1200; G = 18% × 1200 = **216 guanine nucleotides**. **[1]**

## 老师讲评建议 / Marking tips
1. **codon / anticodon 混淆**：许多学生将 codon（mRNA 上）与 anticodon（tRNA 上）混淆，或将 DNA 模板链方向弄反。每节课结束默写"DNA 模板链→mRNA 密码子→tRNA 反密码子"三对方向。
2. **Explain 题只 Describe**：B2b 的 Meselson-Stahl 题要求 explain 为什么结果支持半保留模型，而非只描述结果——必须说"rules out conservative model"或"original strand conserved"。
3. **移码计算错误**：删除第 N 个碱基后，第 N 个密码子及之后所有密码子全部改变（不只是"前后一个"）。用具体例子（给序列对比）帮学生建立感觉。
