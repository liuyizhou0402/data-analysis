**对应 Lesson 24**

### 教学目标 / Learning objectives
学生在本课后能够：
- [ ] **Describe** 核苷酸的三个组成成分，并说明 DNA 与 RNA 核苷酸的差异
- [ ] **Draw / Label** DNA 双链结构：糖-磷酸骨架、碱基配对、氢键数目
- [ ] **Compare** DNA 与 RNA 的结构（成对作答）
- [ ] **State** ATP 是核苷酸衍生物（简述连接即可）

### 授课流程 / Lesson flow（5–15–20–15–5）
| 时间 | 环节 | 老师做什么 | 学生做什么 |
|---|---|---|---|
| 0–5′ | 导入 | 展示 DNA 双螺旋图片，问"你知道这条'梯子'的每一步是由什么组成的？" → 引出核苷酸 | 猜想，激活前知 |
| 5–20′ | 精讲（核苷酸） | 讲核苷酸三组成：① 五碳糖（DNA→脱氧核糖，RNA→核糖）② 磷酸基团 ③ 含氮碱基；画单个核苷酸结构图。强调 DNA 碱基 A/T/G/C，RNA 碱基 A/U/G/C（T 换成 U） | 跟随画图、完成术语表 |
| 20–40′ | 精讲（DNA 双螺旋） | ① 两条链**反向平行 antiparallel**（5'→3' 与 3'→5' 对向）② **糖-磷酸骨架**在外侧 ③ 碱基在内侧互补配对：A-T（**2** 氢键）、G-C（**3** 氢键）④ 磷酸二酯键连接相邻核苷酸（共价键）⑤ Watson-Crick 1953 年模型（DNA 拍 X 衍射 + Chargaff 规律）。结合 DNA vs RNA 对比表 | 在空白模板上标注双链图 |

| 55–60′ | 总结 | 锁定：碱基对+氢键数；antiparallel；ATP 也含腺嘌呤核苷酸（简述一句即可）| 默写 DNA/RNA 对比表 |

### 板书/重点 / Board work
- **核苷酸结构图**（三组件：磷酸—五碳糖—碱基，画出连接方式）
- **DNA vs RNA 对比表**（见学生讲义）
- **碱基对口诀**：A-T 两个氢键（英文"AT = 2 tiers"），G-C 三个氢键（"GC = 3 tiers"）

### 易错点 / Common errors
- ❌ 说"DNA 有尿嘧啶" → ✅ DNA 含 **T（胸腺嘧啶）**，RNA 含 **U（尿嘧啶）**。
- ❌ 说"氢键连接两条链的糖-磷酸骨架" → ✅ 氢键连接的是**互补碱基对**之间。
- ❌ 忘记 antiparallel：两链走向相反——一条 5'→3'，另一条 3'→5' → ✅ 画图时标方向。
- ❌ G-C 说"两氢键"、A-T 说"三氢键" → ✅ **A-T = 2 氢键，G-C = 3 氢键**（G-C 更稳定）。

### 提问设计 / Key questions
- "在 DNA 双链中，已知一条链的碱基序列，你能写出另一条链的序列吗？" → 引出互补配对规则实际操作。
- "G-C 含量越高的 DNA 分子越稳定，为什么？" → 引出 3 氢键。

## 课时 6.2 — DNA 复制 / DNA Replication（60 min）
**对应 Lesson 25**

### 教学目标
- [ ] **Explain** 半保留复制（semi-conservative replication）的含义与机制
- [ ] **Describe** 解旋酶（helicase）与 DNA 聚合酶（DNA polymerase）的作用
- [ ] **Outline** Meselson-Stahl 实验及其证明半保留复制的逻辑
- [ ] **State** DNA 聚合酶只能沿 5'→3' 方向合成新链

### 必备术语
| English | 中文 |
|---|---|
| semi-conservative replication | 半保留复制 |
| helicase | 解旋酶 |
| DNA polymerase | DNA 聚合酶 |
| template strand | 模板链 |
| complementary strand | 互补链 |
| free DNA nucleotides | 游离脱氧核苷酸 |
| phosphodiester bond | 磷酸二酯键 |
| 5'→3' direction | 5' 到 3' 方向 |
| Meselson-Stahl experiment | 梅塞尔森-斯塔尔实验 |
| ¹⁵N / ¹⁴N | 重氮/轻氮（密度标记） |

### 授课流程
| 时间 | 环节 | 内容 |
|---|---|---|
| 0–5′ | 回顾 | DNA 结构快问：碱基对、氢键数、antiparallel |
| 5–20′ | 精讲（复制机制） | 1. **Helicase（解旋酶）**：沿 DNA 移动，**断氢键**，解开双链 → 两条模板链暴露。2. 游离脱氧核苷酸按**碱基互补配对**原则与模板链配对。3. **DNA polymerase（DNA 聚合酶）**：将核苷酸依次连接（形成磷酸二酯键），只沿 **5'→3'** 方向合成。4. 结果：每个新 DNA 分子含**一条旧链**（模板）+**一条新链** → 半保留。 |
| 20–40′ | 精讲（Meselson-Stahl） | 实验逻辑：① ¹⁵N 培养细菌（全重 ¹⁵N-DNA）→ 转入 ¹⁴N 培养基复制一代 → 离心：**所有 DNA 在中间位置**（一条 ¹⁵N + 一条 ¹⁴N）= 只有"半保留"才能解释（全保留应有一半纯重、一半纯轻；弥散复制应全在中间但再复制一代结果不同）。② 复制两代 → 出现一半中间、一半轻带 → 再次验证。 |
| 40–55′ | 真题型 | 练"describe how DNA replicates""explain the result of Meselson-Stahl""why polymerase needs a template"类题 |
| 55–60′ | 总结 | 半保留口诀：旧链当模板、新链互补生，每个子代含一旧一新 |

### 板书/重点
- **复制流程箭头图**（helicase → 解链 → DNA pol → 5'→3' 合成两条新链）。
- **Meselson-Stahl 结果图**（密度梯度离心的条带位置：0代/1代/2代）。

### 易错点 / Common errors
- ❌ 说"解旋酶破坏磷酸二酯键" → ✅ helicase 断的是**氢键**（碱基对间），不是骨架的磷酸二酯键。
- ❌ 说"两条新链同方向合成" → ✅ 两条模板链方向相反，聚合酶始终 5'→3'，所以一条连续、一条分段（考纲只要求知道 5'→3'方向）。
- ❌ 说 Meselson-Stahl 证明了保守复制 → ✅ 证明了**半保留（semi-conservative）**复制——若是保守复制，第一代应该有两条不同密度带。
- ❌ 复制时说"RNA 聚合酶" → ✅ 复制用的是 **DNA 聚合酶**（转录才用 RNA 聚合酶）。

### 提问设计
- "若复制为'全保留（conservative）'，离心结果会如何？" → 强化 Meselson-Stahl 鉴别逻辑。
- "DNA 聚合酶为什么不能'从头起始'合成？它需要什么？" → 引入引物概念（了解即可，考纲不深入）。

### 作业
- `practice.md` Part A 第 4–6 题、Part B 第 2 题、Part D 第 1 题（复制计算）。

## 课时 6.3 — 遗传密码与转录 / Genetic Code & Transcription（60 min）
**对应 Lesson 26**

### 教学目标
- [ ] **State** 遗传密码的四个特征：三联体/不重叠/简并/（近乎）通用
- [ ] **Define** codon（密码子）与 anticodon（反密码子）
- [ ] **Describe** 转录的过程：场所、酶、模板、产物
- [ ] **Apply** 遗传密码表读取氨基酸

### 必备术语
| English | 中文 |
|---|---|
| genetic code | 遗传密码 |
| codon | 密码子（mRNA 上的三联体） |
| triplet code | 三联体密码 |
| non-overlapping | 不重叠 |
| degenerate | 简并（多个密码子编码同一氨基酸） |
| (near) universal | （近乎）通用 |
| transcription | 转录 |
| RNA polymerase | RNA 聚合酶 |
| template strand | 模板链（又称 antisense strand） |
| coding strand | 编码链（又称 sense strand） |
| messenger RNA (mRNA) | 信使 RNA |
| start codon (AUG) | 起始密码子 |
| stop codon (UAA/UAG/UGA) | 终止密码子 |

### 授课流程
| 时间 | 环节 | 内容 |
|---|---|---|
| 0–5′ | 回顾 | DNA 复制步骤复述（helicase→解链→DNA pol→5'→3'） |
| 5–20′ | 精讲（遗传密码） | 四特征逐条讲：① **Triplet**：每3个碱基编码1个氨基酸（4³=64种组合，够编 20 种氨基酸+终止信号）② **Non-overlapping**：相邻三联体不共用碱基 ③ **Degenerate**：大多氨基酸由多个密码子编码（如 Leu 有 6 个密码子）→ 某些突变不改变氨基酸（silent mutation 埋下伏笔）④ **(Near) Universal**：几乎所有生物用同一套密码——支持共同祖先（synoptic！） |
| 20–38′ | 精讲（转录） | ① 场所：**细胞核内**（真核）② RNA 聚合酶结合到 DNA → 解开局部双链 ③ 以 **template strand（模板链）** 为模板（3'→5'） ④ 游离**核糖核苷酸**按互补配对（U 代 T）依次连接 ⑤ 产生**mRNA**（5'→3'）⑥ mRNA 离开核，通过**核孔**进入细胞质 |
| 38–52′ | 真题型 | 给一段 DNA 模板链，让学生写 mRNA 序列并用密码表找氨基酸；练"describe transcription" |
| 52–60′ | 总结 | 密码表使用技巧（先找第1碱基行，再第2碱基列，再第3碱基）；转录产物 vs 翻译产物 |

### 板书/重点
- **遗传密码四特征一览**（画四格：triplet/non-overlapping/degenerate/universal）。
- **转录流程简图**（DNA → RNA pol → mRNA → 核孔 → 细胞质）。
- **T/U 互换规则**：转录时模板链 A 对应 mRNA 的 U（不是 T）。

### 易错点 / Common errors
- ❌ 混淆 template strand 与 coding strand → ✅ mRNA 序列与 **coding strand** 一致（只是 T→U），转录用的模板是 **template strand**（互补反向）。
- ❌ 说遗传密码是"codon" 在 DNA 上 → ✅ **codon 在 mRNA 上**（DNA 上对应的三联体叫 triplet，mRNA 上叫 codon）。
- ❌ 说"简并意味着每个密码子对应多个氨基酸" → ✅ 简并是"**多个密码子**对应**同一氨基酸**"，方向相反。
- ❌ 转录说"酶是 DNA 聚合酶" → ✅ 转录用 **RNA 聚合酶**。

### 提问设计
- "如果遗传密码是两联体，能编多少种氨基酸？为什么要三联体？"（4²=16 不够，4³=64 足够）
- "为什么'近乎'通用而不是完全通用？"（极少数例外，如线粒体密码）

### 作业
- `practice.md` Part A 第 7–8 题、Part B 第 3 题、Part C 第 1 题。

## 课时 6.4 — 翻译 / Translation（60 min）
**对应 Lesson 27**

### 教学目标
- [ ] **Describe** 翻译的全过程：mRNA 到核糖体、tRNA 搬运氨基酸、codon-anticodon 配对、肽键形成
- [ ] **Define** anticodon 并解释 codon-anticodon 互补配对
- [ ] **Explain** 为什么翻译需要 tRNA 的中介作用（RNA 不直接识别氨基酸）
- [ ] **Outline** 核糖体沿 mRNA 移动产生多肽的过程

### 必备术语
| English | 中文 |
|---|---|
| translation | 翻译 |
| ribosome (80S) | 核糖体（80S；由大亚基 + 小亚基） |
| transfer RNA (tRNA) | 转运 RNA |
| anticodon | 反密码子（tRNA 上的三联体碱基环） |
| aminoacyl-tRNA | 氨酰-tRNA（携带特定氨基酸的 tRNA） |
| peptide bond | 肽键 |
| polypeptide | 多肽 |
| start codon (AUG) | 起始密码子（编码蛋氨酸 Met） |
| stop codon | 终止密码子（UAA/UAG/UGA，不对应氨基酸） |
| P site / A site | P 位（肽酰位） / A 位（氨酰位）（了解即可） |

### 授课流程
| 时间 | 环节 | 内容 |
|---|---|---|
| 0–5′ | 回顾 | 转录步骤：RNA pol/模板链/mRNA/核孔；密码子在哪里？ |
| 5–22′ | 精讲（翻译完整步骤） | ① mRNA 从核孔出来，结合到**核糖体（细胞质）**② 核糖体识别**起始密码子 AUG**（编码 Met）③ 每个 **tRNA** 携带**特定氨基酸**，其**反密码子（anticodon）**与 mRNA 上**密码子（codon）互补配对** ④ 核糖体催化：相邻两个氨基酸之间形成**肽键（缩合反应）** ⑤ 核糖体沿 mRNA **5'→3' 移动**，每次移动一个密码子 ⑥ 遇**终止密码子**（UAA/UAG/UGA）→ 多肽链释放 ⑦ 核糖体解离 |
| 22–38′ | 结构-功能联系 | tRNA 为何是翻译的关键中介？→ 氨基酸无法直接识别密码子；tRNA 一端有 anticodon 识别 mRNA，另一端携带对应氨基酸（由氨酰-tRNA 合成酶特异识别）。**"适配器"作用**。连接 Topic 2（蛋白质结构）：多肽→折叠→三维构象 |
| 38–52′ | 真题型 | 给 mRNA 序列 + 遗传密码表，写氨基酸序列；练"describe translation from AUG to stop codon" |
| 52–60′ | 总结 | 转录 vs 翻译完整对比（场所/酶/模板/产物/核糖体/方向）；polysome（多聚核糖体）——多个核糖体同时翻译一条 mRNA（简述，理解即可） |

### 板书/重点
- **翻译流程动画式板书**（mRNA + 核糖体 + tRNA 携带氨基酸 + 肽链逐步延长）。
- **codon–anticodon 互补示例**：mRNA codon AUG（5'→3'）对应 tRNA anticodon UAC（3'→5'）。
- **转录 vs 翻译对比表**（见学生讲义）。

### 易错点 / Common errors
- ❌ 说"tRNA 直接合成蛋白质" → ✅ tRNA 是**搬运**氨基酸的；**核糖体**催化肽键形成。
- ❌ 混淆 codon 和 anticodon → ✅ **codon 在 mRNA**，**anticodon 在 tRNA**（两者互补配对）。
- ❌ 说终止密码子"编码一个氨基酸，终止翻译" → ✅ 终止密码子**不对应任何氨基酸**，终止释放因子识别它让链释放。
- ❌ 说翻译在细胞核内 → ✅ 翻译在**细胞质中（核糖体上）**；转录在核内。

### 提问设计
- "一个细胞在旺盛合成某种蛋白质时，一条 mRNA 上会有多少核糖体？这有什么意义？"（→ polysome，效率高）
- "为什么终止密码子有三种而不是一种？"（冗余/更可靠——迁移 suggest）

- `practice.md` Part A 第 9–10 题、Part B 第 4 题、Part C 第 2 题。

## 课时 6.5 — 基因突变与综合复习 / Gene Mutation & Review（60 min）
**对应 Lesson 28**

### 教学目标
- [ ] **Describe** 三类点突变（替换/插入/缺失）及其对蛋白质的影响
- [ ] **Distinguish** silent / missense / nonsense 替换突变
- [ ] **Explain** 插入/缺失导致 frameshift（移码）的机制及严重后果
- [ ] **Analyse** 镰刀型贫血：碱基→密码子→氨基酸（Glu→Val）→蛋白质结构→功能改变

### 必备术语
| English | 中文 |
|---|---|
| gene mutation | 基因突变 |
| point mutation | 点突变（单碱基变化） |
| substitution | 碱基替换 |
| insertion / deletion | 插入 / 缺失 |
| frameshift mutation | 移码突变 |
| silent mutation | 沉默突变（氨基酸不变） |
| missense mutation | 错义突变（氨基酸改变） |
| nonsense mutation | 无义突变（产生终止密码子） |
| sickle cell anaemia | 镰刀型（细胞）贫血 |
| glutamic acid (Glu) → valine (Val) | 谷氨酸 → 缬氨酸 |
| haemoglobin S (HbS) | 血红蛋白 S 型 |

### 授课流程
| 时间 | 环节 | 内容 |
|---|---|---|
| 0–5′ | 回顾 | 翻译步骤口头复述；密码表抽问 |
| 5–22′ | 精讲（替换突变三类型） | **①Silent（沉默）**：碱基改变 → 密码子改变 → 但编码**同一氨基酸**（简并性的结果）→ 蛋白质不变。**②Missense（错义）**：碱基改变 → 密码子改变 → 编码**不同氨基酸**→ 蛋白质结构/功能改变（可轻微到严重）。**③Nonsense（无义）**：碱基改变 → 密码子变成**终止密码子**→ 翻译提前终止 → 多肽链**缩短**，功能大概率丧失。配合密码表逐一演示（如 GAA Glu → GUA Val = missense；GAA → UAA = nonsense） |
| 22–40′ | 精讲（插入/缺失 = 移码） | 插入或缺失**一个（或非3的倍数个）碱基** → 后续所有三联体阅读框**全部移动** → frameshift → 从突变点之后的**所有氨基酸都改变** → 多肽链常早期终止（遇到提前出现的终止密码子）→ 功能严重丧失。对比：若缺失 3 个碱基（1个密码子）→ 仅缺失一个氨基酸，其余不变（相对较轻）。**镰刀型贫血**：第 6 位 GAG(Glu) → GTG(Val) — **单碱基替换（missense）**；Val 是非极性氨基酸，Glu 是极性带负电 → HbS 在低氧下聚合 → 红细胞变镰刀形 → 破裂/堵塞 → 贫血/疼痛 |
| 40–55′ | 综合练 | 完成 `practice.md` 混合题（涵盖复制、遗传密码、转录、翻译、突变），限时作答 |
| 55–60′ | 讲评+总结 | 对照 `practice-answers.md`，按四类归档丢分（知识/表达/审题/没引数据） |

### 板书/重点
- **突变类型汇总表**：
  | 突变类型 | 碱基变化 | 密码子变化 | 氨基酸变化 | 对蛋白影响 |
  |---|---|---|---|---|
  | Silent（沉默） | 1个替换 | 改变但同义 | **不变** | 无 |
  | Missense（错义） | 1个替换 | 改变且不同义 | **改变** | 可有/有 |
  | Nonsense（无义） | 1个替换 | 变为终止密码子 | 翻译**提前终止** | 严重 |
  | Frameshift 移码 | 插入/缺失（非3n） | 全部后续改变 | **全改变** | 极严重 |

### 易错点 / Common errors
- ❌ 说"silent 突变是 DNA 未发生改变" → ✅ DNA **有改变**（碱基替换了），只是氨基酸和蛋白质不变（因为密码子简并）。
- ❌ 说"插入3个碱基导致 frameshift" → ✅ 插入 3 个碱基**不会移码**（阅读框增加一个密码子），**插入1或2个**才会。
- ❌ 镰刀型贫血只说"HbS 和正常 HbA 碱基不同" → ✅ 要精确到：**第 6 位氨基酸 Glu（GAG）→ Val（GTG），单碱基 A→T 替换**。
- ❌ 说突变"一定有害" → ✅ 突变是**随机的**，可以有害、中性（silent）或极少情况有利。

### 提问设计
- "鉴于遗传密码的简并性，哪些密码子第三位碱基改变最不影响氨基酸？"（→ 观察密码表，第三碱基常有更多自由度）
- "为什么镰刀型贫血是单碱基替换却有如此严重影响？"（→ 关键位置的关键氨基酸，极性变化导致蛋白质聚合）
