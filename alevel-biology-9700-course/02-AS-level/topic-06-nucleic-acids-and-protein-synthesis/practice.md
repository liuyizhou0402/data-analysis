## Part A — 选择题 Multiple Choice（仿 Paper 1）

**1.** Which correctly describes a difference between DNA and RNA?
- A. DNA contains uracil; RNA contains thymine
- B. DNA is double-stranded; RNA is always single-stranded
- C. DNA contains deoxyribose; RNA contains ribose
- D. DNA has adenine; RNA does not

**2.** During semi-conservative DNA replication, which enzyme unwinds the double helix by breaking hydrogen bonds?
- A. DNA polymerase
- B. RNA polymerase
- C. Helicase
- D. Ligase

**3.** A section of DNA has the base sequence: 5'–ATGCCG–3'. What is the mRNA sequence transcribed from the template (antisense) strand?
- A. 5'–ATGCCG–3'
- B. 5'–UACGGC–3'
- C. 5'–AUGCCG–3'
- D. 5'–TACGGC–3'

**4.** Which statement about the genetic code is correct?
- A. Each codon codes for more than one amino acid
- B. The code is non-overlapping and degenerate
- C. Different organisms use different genetic codes
- D. Each amino acid is coded for by exactly one codon

**5.** A tRNA molecule has the anticodon 3'–UAC–5'. Which amino acid does it carry? (Use the codon: AUG = methionine)
- A. Tyrosine
- B. Valine
- C. Methionine
- D. Isoleucine

**6.** A mutation changes a codon from UUU (phenylalanine) to UUC. The protein is unchanged. What type of mutation is this?
- A. Missense mutation
- B. Nonsense mutation
- C. Silent mutation
- D. Frameshift mutation

**7.** A single base insertion occurs in the middle of a gene. Which best describes the effect on the protein?
- A. One amino acid is changed
- B. All amino acids after the insertion point are likely changed (frameshift)
- C. Only the amino acid at the insertion site is affected
- D. No effect — the protein is unchanged

**8.** Which correctly describes the process of **translation**?
- A. DNA → mRNA, using RNA polymerase
- B. mRNA → polypeptide, at the ribosome, using tRNA
- C. mRNA → DNA, using reverse transcriptase
- D. Protein → mRNA, using ribosomes

**9.** Sickle-cell anaemia results from a single base substitution. The codon GAG (glutamic acid) is changed to GUG (valine). What type of mutation is this?
- A. Silent
- B. Frameshift
- C. Missense
- D. Nonsense

**10.** A DNA molecule is allowed to replicate for 3 generations. How many of the resulting 8 DNA molecules contain an original (parental) strand?
- A. 1
- B. 2
- C. 4
- D. 8
