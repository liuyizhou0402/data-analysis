## 1. 显微镜与细胞研究 / Microscopy

### 1.1 放大率 vs 分辨率（核心辨析）
- **Magnification（放大率）**：图像比实物**放大了多少倍**。
- **Resolution（分辨率）**：能把**两个靠得很近的点分辨为两个点**的能力。

> 关键理解：放大率高 ≠ 看得清。如果分辨率不够，把图像放再大也只是"模糊地变大"（**empty magnification**）。**看得清靠的是 resolution。**
>
> 类比：把一张低像素照片不断放大，只会越来越糊——放大率上去了，分辨率没上去。

**分辨率由什么决定？** 由所用"照明波长"决定：**波长越短，分辨率越高**。
- 光（可见光）波长长（约 400–700 nm）→ 光镜分辨率上限约 **200 nm**。
- 电子波长极短 → 电镜分辨率可达 **~0.5 nm**（高数百倍）。

### 1.2 三种显微镜对比
| | 光镜 Light | TEM 透射电镜 | SEM 扫描电镜 |
|---|---|---|---|
| 照明 | 光（长波长） | 电子束（短波长） | 电子束 |
| 分辨率 | ~200 nm | ~0.5 nm（最高） | ~3–10 nm |
| 图像 | 彩色、2D | 黑白、2D、看**内部超微结构** | 黑白、**3D 表面** |
| 活体 | 可观察活的 | 否（需真空、需切片/染色） | 否 |
| 典型用途 | 看细胞、组织整体 | 看细胞器内部细节 | 看表面立体形貌 |

> ⚠️ 电镜**看不了活细胞**（样品要脱水、置于真空、可能产生**人工假象 artefact**）。

## 2. 放大率计算与单位 / Magnification Calculations

### 2.1 核心公式
$$ \text{Magnification (M)} = \frac{\text{Image size (I)}}{\text{Actual size (A)}} $$

变形：**I = M × A**，**A = I / M**。

> 公式三角：把要求的量盖住即可。**放大率 M 没有单位**（它是倍数）。

### 2.2 单位换算链（必背！丢分重灾区）
$$ 1\ \text{mm} = 1000\ \mu\text{m} = 1\,000\,000\ \text{nm} $$
- mm → μm：×1000
- μm → nm：×1000

### 2.3 解题四步模板
1. **写公式**（M = I/A 或其变形）
2. **统一单位**（把 I 和 A 换成同一单位——这步不做必错）
3. **代入计算**，结果**带单位**（A 有单位，M 无单位）
4. **检查有效数字**（与题目数据一致）

**例题**：一个细胞实际长 **50 μm**，在图上量得 **100 mm**，求放大率。
- 统一单位：100 mm = 100 × 1000 = 100 000 μm
- M = I/A = 100 000 / 50 = **×2000**（无单位）

### 2.4 scale bar（比例尺）题
- 用尺量出图中 **scale bar 的实际图上长度**（= I）。
- bar 标注的值（如 "10 μm"）= A。
- M = I / A。
- 再用 M 去求图中其他结构的真实大小。

## 3. 真核细胞器：结构与功能 / Eukaryotic Organelles

> 核心思维：**Structure relates to function（结构决定功能）**。背的时候永远连着"它的结构如何支持功能"。

| 细胞器 Organelle | 结构 Structure | 功能 Function |
|---|---|---|
| **Nucleus 细胞核** | 双层核膜(nuclear envelope)、含核孔、内有染色质(DNA) | 储存遗传信息、控制细胞活动（转录在此） |
| **Nucleolus 核仁** | 核内致密区 | 制造 rRNA、组装核糖体亚基 |
| **Rough ER 粗面内质网** | 膜囊网络、**表面附核糖体** | 合成、折叠并运输**蛋白质** |
| **Smooth ER 滑面内质网** | 膜囊网络、**无核糖体** | 合成**脂质**与类固醇 |
| **Golgi apparatus 高尔基体** | 扁平膜囊堆叠 | **修饰、包装**蛋白质，形成分泌小泡（如糖蛋白） |
| **Ribosome 核糖体(80S)** | 由 rRNA+蛋白组成、**无膜** | **蛋白质合成**（翻译） |
| **Mitochondrion 线粒体** | 双层膜、内膜折叠成**嵴(cristae)**、内含基质(matrix) | **有氧呼吸**、合成 ATP（嵴增大表面积→更多酶→更多 ATP） |
| **Lysosome 溶酶体** | 单层膜小泡、含**水解酶** | 分解废物、衰老细胞器、病原体 |
| **Centriole 中心粒** | 两个垂直排列的微管筒（动物细胞） | 参与**纺锤体**形成（细胞分裂） |

> ⚠️ 易错：
> - 核糖体**无膜**；线粒体说"合成 ATP"而非笼统"产生能量"。
> - nucleus（核）≠ nucleolus（核仁）。

## 4. 植物细胞 vs 动物细胞 / Plant vs Animal Cell

植物细胞**比动物细胞多出**：
| 结构 | 功能 |
|---|---|
| **Cell wall（纤维素）** | 提供支撑与形状、防止过度吸水涨破 |
| **Chloroplast 叶绿体** | 进行光合作用（含类囊体、基质、色素） |
| **Permanent vacuole 大液泡 + tonoplast 液泡膜** | 维持膨压(turgor)、储存物质 |
| **Plasmodesmata 胞间连丝** | 贯穿细胞壁的通道，连通相邻细胞 |

动物细胞特征：有 **centriole**；**无**细胞壁、叶绿体、大液泡。

> 🎯 **Compare 题必须成对**：
> *"Plant cells have a cellulose cell wall **whereas** animal cells do not; plant cells contain chloroplasts **whereas** animal cells do not…"*

## 5. 蛋白质合成与分泌途径 / Protein Secretion Pathway（高频结构题）

把细胞器**串起来**理解（以分泌蛋白如酶/激素为例）：

```
① 细胞核 DNA  →（转录 mRNA 经核孔出核）
② 核糖体（附于 RER）合成多肽
③ RER 折叠蛋白 → 出芽形成囊泡(vesicle)
④ 囊泡运至 高尔基体，修饰/包装（如加糖基）
⑤ 高尔基体出芽 → 分泌小泡(secretory vesicle)
⑥ 分泌小泡与细胞膜融合 → 胞吐(exocytosis)出胞
   （全程由 线粒体 供 ATP）
```

> 🎯 这就是为什么**分泌旺盛的细胞**（如胰腺腺泡细胞）有特别多的 **RER、高尔基体、线粒体和核糖体**——典型 explain 题。

**细胞骨架 Cytoskeleton（microtubules）**：由微管等组成，作用为①支撑细胞形状②作为"轨道"运输囊泡③形成纺锤体。

## 6. 原核细胞 / Prokaryotic Cells（细菌）

原核 = **无核膜、无膜包细胞器**的细胞（细菌）。

| 结构 | 说明 |
|---|---|
| 环状 DNA | 裸露于细胞质，**无核膜、无组蛋白** |
| **70S 核糖体** | 比真核 80S 小 |
| 细胞壁 | 成分为**肽聚糖(peptidoglycan/murein)**（非纤维素） |
| 质粒 plasmid | 小的环状 DNA，常带额外基因（如抗药性） |
| 可有：鞭毛 flagellum、荚膜 capsule | 运动 / 保护 |

---

## 7. 原核 vs 真核 对比表 / Prokaryote vs Eukaryote（必背成对）

| 特征 | 原核 Prokaryote | 真核 Eukaryote |
|---|---|---|
| 核膜 | **无**（DNA 裸露、环状） | **有**（DNA 在核内、线状、缠组蛋白） |
| 膜包细胞器 | **无**（无线粒体、ER 等） | **有** |
| 核糖体 | **70S**（小） | **80S**（大） |
| 细胞壁 | 肽聚糖 | 纤维素（植物）/无（动物） |
| 大小 | 小（~1–5 μm） | 大（~10–100 μm） |
| 质粒 | 常有 | 一般无 |

---

## 8. 病毒 / Viruses

- **非细胞结构（acellular）**：不是细胞，无细胞器、无代谢。
- 组成：**核酸（DNA 或 RNA）+ 蛋白质衣壳(capsid)**；部分有**包膜(envelope)**。
- 只能在**宿主活细胞内**借用其机器复制。
- 是否"活着"有争议（无独立代谢）。
