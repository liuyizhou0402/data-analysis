## 课时 1.1 — 显微镜与细胞研究 / Microscopy（60 min）

### 教学目标 / Learning objectives
学生在本课后能够：
- [ ] **Define** magnification 与 resolution，并说清两者区别
- [ ] **Explain** 为什么电镜分辨率远高于光镜（电子波长更短）
- [ ] **Compare** 光镜（light microscope）、TEM、SEM 的特点与用途

### 必备术语 / Key terms
| English | 中文 |
|---|---|
| magnification | 放大率 |
| resolution / resolving power | 分辨率/分辨本领 |
| light microscope | 光学显微镜 |
| transmission electron microscope (TEM) | 透射电镜 |
| scanning electron microscope (SEM) | 扫描电镜 |
| wavelength | 波长 |

### 授课流程 / Lesson flow
| 时间 | 环节 | 老师做什么 | 学生做什么 |
|---|---|---|---|
| 0–5′ | 导入 | 展示同一细胞的光镜 vs 电镜图，问"为什么电镜看得更清楚？" | 猜想 |
| 5–22′ | 精讲 | 讲 magnification（放大多少倍）vs resolution（能分清多近的两点）；用"把模糊照片放大还是模糊"类比说明放大≠看清 | 记笔记 |
| 22–38′ | 精讲 | 讲电镜原理：电子波长 ≪ 可见光 → resolution 高；对比 TEM（内部超微结构、2D）vs SEM（表面、3D） | 完成对比表 |
| 38–52′ | 真题型 | 练"区分放大率与分辨率""为什么电镜分辨率高"类结构题 | 限时作答 |
| 52–60′ | 总结 | 锁定两个定义的英文措辞 | 默写定义 |

### 板书/重点 / Board work
- **核心对比表**：
  | | 光镜 | TEM | SEM |
  |---|---|---|---|
  | 照明源 | 光（长波长） | 电子（短波长） | 电子 |
  | 分辨率 | ~200 nm | ~0.5 nm | ~3–10 nm |
  | 看活体？ | 可 | 否（真空、需处理） | 否 |
  | 成像 | 彩色、2D | 黑白、2D 内部 | 黑白、3D 表面 |

### 易错点 / Common errors
- ❌ "电镜放大率高所以清楚" → ✅ 看得清是因为**resolution（分辨率）高**，不是放大率。放大率再高、分辨率不够仍模糊（empty magnification）。
- ❌ 说电镜"用光" → ✅ 用**电子束**。

### 提问设计 / Key questions
- "把光镜图像无限放大，能看清核糖体吗？为什么不能？"（引出 resolution 受波长限制）

- `practice.md` Part A 第 1–2 题、Part B 第 1 题。

## 课时 1.2 — 放大率计算与单位 / Magnification & Scale（60 min）

### 教学目标
- [ ] 熟练运用 **M = I / A**（放大率 = 图像大小 ÷ 实际大小）求三者中任一
- [ ] 正确进行单位换算：**1 mm = 1000 μm = 1 000 000 nm**
- [ ] 由 **scale bar** 求放大率与实际大小

### 必备术语
| English | 中文 |
|---|---|
| image size (I) | 图像大小 |
| actual / real size (A) | 实际大小 |
| scale bar | 比例尺 |
| micrometre (μm) | 微米 |

### 授课流程
| 时间 | 环节 | 内容 |
|---|---|---|
| 0–5′ | 回顾 | 上节定义快问快答 |
| 5–18′ | 精讲 | 公式三角 **I = M × A**；强调**先统一单位再代入** |
| 18–35′ | 示范 | 老师板演 2 题：①已知 I、M 求 A；②由 scale bar 求 M。每步写公式、换单位、带单位 |
| 35–52′ | 当堂练 | 学生做 4 道计算（含 scale bar），同桌互批 |
| 52–60′ | 总结 | 强调"**单位不统一 = 必错**"，归纳换算链 |

### 板书/重点
- **公式三角**：盖住要求的量 → I=M×A、M=I/A、A=I/M
- **换算链**：mm ×1000→ μm ×1000→ nm
- **解题模板**：
  1. 写公式
  2. 把 I、A 换成同一单位
  3. 代入、算、**带单位**
  4. 检查有效数字

### 易错点 / Common errors
- ❌ I 用 mm、A 用 μm 直接相除 → ✅ 先统一单位。
- ❌ 放大率带单位（放大率**无单位**！）→ ✅ M 是倍数，写 ×1500 或 1500。
- ❌ scale bar 读数不量"bar 本身长度" → ✅ 用尺量 scale bar 的图上长度作为 I。

### 提问设计
- "一个细胞实际 50 μm，图上 10 cm，放大率是多少？"（练换算：10 cm = 100 mm = 100000 μm → M = 100000/50 = ×2000）

### 作业
- `practice.md` Part A 第 3 题、Part B 第 2 题（计算专练）。

## 课时 1.3 — 真核细胞结构（动物）/ Eukaryotic Organelles（60 min）

### 教学目标
- [ ] **State** 动物真核细胞各细胞器的结构与功能
- [ ] **Identify** 电镜图中的细胞器
- [ ] 理解 **80S 核糖体**与膜系统的关系

### 必备术语
| English | 中文 |
|---|---|
| nucleus / nuclear envelope / nucleolus | 细胞核/核膜/核仁 |
| rough & smooth endoplasmic reticulum (RER/SER) | 粗面/滑面内质网 |
| Golgi apparatus (body) | 高尔基体 |
| ribosome (80S) | 核糖体 |
| mitochondrion (cristae, matrix) | 线粒体（嵴、基质） |
| lysosome | 溶酶体 |
| centriole | 中心粒 |

### 授课流程
| 时间 | 环节 | 内容 |
|---|---|---|
| 0–5′ | 回顾 | 计算题快测一道 |
| 5–25′ | 精讲 | 按"**结构→功能**"逐个讲细胞器；重点：核（控制中心、含 DNA）、RER（结合核糖体、合成/运输蛋白）、SER（合成脂质）、高尔基（修饰/包装/分泌）、线粒体（有氧呼吸、ATP；嵴增大表面积）、核糖体（蛋白质合成、80S）、溶酶体（含水解酶、分解） |
| 25–40′ | 识图 | 给 TEM 动物细胞图，学生标注各细胞器并说功能 |
| 40–55′ | 真题型 | 练"描述某细胞器功能""为何分泌细胞 RER/高尔基/线粒体多"类题 |
| 55–60′ | 总结 | 必背"结构-功能"对照表 |

### 板书/重点
- **结构-功能对照表**（见学生讲义），强调每个结构如何**支持其功能**（如线粒体嵴 → 大表面积 → 多酶 → 多 ATP）。

### 易错点 / Common errors
- ❌ 混淆 nucleus 与 nucleolus → ✅ nucleolus 在核内，制造 rRNA/核糖体亚基。
- ❌ 说核糖体"是膜包的细胞器" → ✅ 核糖体**无膜**。
- ❌ 笼统说线粒体"产生能量" → ✅ 准确说"**通过有氧呼吸合成 ATP**"（能量不被'产生'）。

### 提问设计
- "为什么分泌大量蛋白质的细胞（如胰腺细胞）会有特别多的 RER、高尔基体和线粒体？"（铺垫下节分泌途径 + 结构功能 + ATP）

### 作业
- `practice.md` Part A 第 4–6 题、Part B 第 3 题。

## 课时 1.4 — 植物细胞与蛋白质分泌途径 / Plant Cell & Secretory Pathway（60 min）

### 教学目标
- [ ] **Compare** 动物细胞与植物细胞（成对作答）
- [ ] **Outline** 蛋白质从合成到分泌的完整途径（细胞器协作）
- [ ] **State** 细胞骨架（microtubules）的作用

### 必备术语
| English | 中文 |
|---|---|
| cell wall (cellulose) | 细胞壁（纤维素） |
| chloroplast | 叶绿体 |
| permanent vacuole / tonoplast | 大液泡/液泡膜 |
| plasmodesmata | 胞间连丝 |
| cytoskeleton / microtubule | 细胞骨架/微管 |

### 授课流程
| 时间 | 环节 | 内容 |
|---|---|---|
| 0–5′ | 回顾 | 细胞器功能快问 |
| 5–18′ | 精讲 | 植物细胞**多出**：细胞壁、叶绿体、大液泡(+tonoplast)、胞间连丝；动物**多出/有**：中心粒、（无壁无叶绿体）。强调"compare 要成对" |
| 18–38′ | 精讲（重点） | **蛋白质分泌途径**：① 核内 DNA 指令 → ② 核糖体(RER 上)合成多肽 → ③ RER 出芽小泡运至 → ④ 高尔基体修饰/包装 → ⑤ 分泌小泡 → ⑥ 与质膜融合、胞吐(exocytosis)出胞。线粒体供 ATP | 画流程图 |
| 38–52′ | 真题型 | 练"按顺序描述分泌途径""对比动植物细胞"题 | 限时 |
| 52–60′ | 总结 | 分泌途径 6 步口诀 + 细胞骨架作用（支撑、运输、纺锤体） |

### 板书/重点
- **分泌途径流程图**（核 → RER/核糖体 → 高尔基 → 分泌小泡 → 胞吐）。
- **动植物对比表**（成对）。

### 易错点 / Common errors
- ❌ compare 只写植物有什么，不写动物对应情况 → ✅ **成对**："Plant cells have a cellulose cell wall **whereas** animal cells do not."
- ❌ 分泌途径漏掉"小泡运输"步骤 → ✅ 强调 RER→高尔基、高尔基→膜都是**囊泡(vesicle)**运输。
- ❌ 说"所有植物细胞都有叶绿体" → ✅ 仅含叶绿体的细胞（如根细胞没有）。

### 提问设计
- "如果高尔基体被抑制，分泌的蛋白质会出现什么问题？"（迁移/suggest 练习）

- `practice.md` Part B 第 4 题、Part C 第 1 题（分泌途径应用）。

## 课时 1.5 — 原核细胞、病毒与综合复习 / Prokaryotes, Viruses & Review（60 min）

### 教学目标
- [ ] **Compare** 原核细胞与真核细胞（结构差异、成对）
- [ ] **Describe** 细菌细胞与病毒的结构
- [ ] 完成本主题综合练习并自评

### 必备术语
| English | 中文 |
|---|---|
| prokaryote / eukaryote | 原核生物/真核生物 |
| peptidoglycan (murein) | 肽聚糖 |
| plasmid | 质粒 |
| 70S ribosome | 70S 核糖体 |
| capsid / envelope | 衣壳/包膜 |

### 授课流程
| 时间 | 环节 | 内容 |
|---|---|---|
| 0–5′ | 回顾 | 分泌途径复述 |
| 5–22′ | 精讲 | 原核特征：**无核膜**（DNA 裸露呈环状、无组蛋白）、**无膜包细胞器**、**70S 核糖体**、肽聚糖细胞壁、有质粒/可有鞭毛荚膜；与真核成对对比 | 完成对比表 |
| 22–34′ | 精讲 | 病毒：**非细胞**、核酸(DNA 或 RNA)+衣壳，部分有包膜；只能在宿主细胞内复制 | 记笔记 |
| 34–55′ | 综合练 | 完成 `practice.md` 整套混合题，限时 | 作答 |
| 55–60′ | 讲评 | 对 `practice-answers.md`，标记三类丢分 | 错题归档 |

### 板书/重点
- **原核 vs 真核 成对对比表**（核膜、细胞器、核糖体 80S/70S、DNA 形态、细胞壁成分、大小）。

### 易错点 / Common errors
- ❌ 说原核"没有 DNA" → ✅ 有 DNA，只是**无核膜包被、呈环状、无组蛋白**。
- ❌ 说原核"没有核糖体" → ✅ 有，是 **70S**（比真核 80S 小）。
- ❌ 把病毒当细胞 → ✅ 病毒**无细胞结构、无代谢**，是否"活"有争议。

### 提问设计
- "为什么某些抗生素能杀细菌却不伤人体细胞？"（铺垫 Topic 10/11：靶向 70S 核糖体或肽聚糖壁）
