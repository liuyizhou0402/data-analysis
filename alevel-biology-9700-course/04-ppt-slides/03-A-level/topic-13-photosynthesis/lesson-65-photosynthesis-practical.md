## Hill Reaction DCPIP 实验

**背景**：Robin Hill（1939）发现叶绿体在光照下能还原电子受体

**实验**：
- 提取叶绿体悬浮液 + **DCPIP（蓝色染料）**
- 光照下 DCPIP 被还原 → **蓝色→无色**

**说明**：光反应产生还原力（NADPH 或替代电子受体被还原）
→ 证明光反应产生还原性物质，**与 CO₂ 固定无关**

---

## Aquatic Plant Method 水草测量法

**实验装置**（Elodea / 金鱼草）：

1. 水草置于 NaHCO₃ 溶液中（提供稳定[CO₂]）
2. 改变光强（调节光源距离）
3. 计数单位时间气泡数量（O₂）→ 光合速率

**局限性**：
- 气泡大小不一致
- 部分 O₂ 溶于水
- 实际测量 apparent（表观）速率（= 总光合 - 呼吸）

---

## Leaf Disc Method 叶圆片法（另一实验）

1. 叶圆片用注射器抽走气体（沉底）
2. 放入 NaHCO₃ 溶液
3. 光照下光合产生 O₂ → 叶片**浮起**
4. 记录 50% 叶片浮起的时间 → 时间越短 = 速率越快

---

## Photosynthesis vs Respiration 光合 vs 呼吸对比

| 特征 | Photosynthesis | Respiration |
|---|---|---|
| 能量 | 光能→化学能 | 化学能→ATP |
| 气体 | 吸收 CO₂，释放 O₂ | 吸收 O₂，释放 CO₂ |
| 场所 | 叶绿体 | 细胞质 + 线粒体 |
| 产物 | 葡萄糖 | ATP（+ CO₂ + H₂O） |
| 发生时间 | 有光 | 所有时间 |

---

## Compensation Point Revisited 补偿点回顾

**在光补偿点时**：
- 光合速率 = 呼吸速率
- 净 O₂ = 0，净 CO₂ = 0
- **实际光合 > 表观光合**（真实光合包含供呼吸的那部分）

**关系**：
$$\text{True photosynthesis} = \text{Apparent photosynthesis} + \text{Respiration rate}$$

## Topic 13 Key Equations

**光合总方程**：
$$6\text{CO}_2 + 6\text{H}_2\text{O} \xrightarrow{\text{light}} \text{C}_6\text{H}_{12}\text{O}_6 + 6\text{O}_2$$

**光解水**：
$$2\text{H}_2\text{O} \rightarrow 4\text{H}^+ + 4\text{e}^- + \text{O}_2$$

**碳固定**：
$$\text{CO}_2 + \text{RuBP}(5\text{C}) \rightarrow 2\times\text{GP}(3\text{C})$$

## Key Terms 必背术语

| Term | 含义 |
|---|---|
| DCPIP | 光合氧化还原指示剂（蓝→无色=还原） |
| Hill reaction | 希尔反应（离体叶绿体光还原） |
| Apparent photosynthesis | 表观光合速率 |
| True photosynthesis | 真实光合 = 表观 + 呼吸速率 |

## Topic 13 Summary 主题综合小结

✅ 叶绿体：基粒（光反应）+ 基质（暗反应）
✅ 多色素：吸收更宽光谱
✅ 光反应：PSII（光解水/O₂）→传递链（ATP）→PSI→NADPH
✅ Calvin cycle：CO₂+RuBP→GP→TP→RuBP再生+有机物
✅ 限制因素：光、CO₂、温度（Blackman定律）
✅ 真实光合 = 表观光合 + 呼吸速率
