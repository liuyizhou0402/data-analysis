# Lesson 66: Homeostasis & Negative Feedback
## 稳态与负反馈机制
Cambridge CIE 9700 | A-Level Biology | Topic 14

---

## Learning Objectives 学习目标

完成本课后，你能够：

- 定义稳态（homeostasis）及其生理意义
- 描述负反馈（negative feedback）的工作原理
- 区分效应器（effector）与感受器（receptor）

---

## What is Homeostasis? 什么是稳态？

**定义**：维持体内环境（internal environment）在**最适范围**内的过程

**内环境**（组织液/血浆）需维持：
- 体温（temperature）
- 血糖浓度（blood glucose）
- 水势/渗透压（water potential）
- pH 值

**意义**：酶和细胞功能依赖稳定的内环境

---

## Components of Homeostasis 稳态组成要素

| 组成 | 功能 | 示例 |
|---|---|---|
| **Receptor（感受器）** | 检测偏离正常值的变化 | 化学感受器、热感受器 |
| **Coordinator（协调中枢）** | 处理信息，发出指令 | 下丘脑、胰岛 |
| **Effector（效应器）** | 执行纠正响应 | 肌肉、腺体 |

---

## Negative Feedback 负反馈

**原理**：偏离 → 检测 → 响应 → 恢复到正常值

```
正常值 (set point)
    ↓
偏高/偏低 → Receptor 检测
    ↓
Coordinator 整合信号
    ↓
Effector 反向纠正
    ↓
恢复正常值 → 关闭响应
```

**关键**：响应方向与偏离方向**相反**（负反馈）

---

## Why Negative Feedback? 为什么要负反馈？

**优势**：
- 自动纠正偏差
- 维持动态平衡（dynamic equilibrium）
- 不需要持续主动监控

**正反馈（positive feedback）**：
- 响应方向与偏离方向相同 → 偏差越来越大
- 通常有害（发烧螺旋式升高）
- 少数例外：分娩时宫缩、血液凝固

---

## Body Temperature Regulation 体温调节（示例）

**正常体温**：约 37°C

**体温升高时**：
- 汗腺分泌汗液（蒸发散热）
- 皮肤血管舒张（vasodilation）→ 散热
- 骨骼肌减少产热

**体温降低时**：
- 竖毛肌收缩（立毛）→ 减少对流
- 皮肤血管收缩（vasoconstriction）→ 减少散热
- 骨骼肌颤抖（shivering）→ 产热

## Key Terms 必背术语

| Term | 含义 |
|---|---|
| Homeostasis | 稳态（维持内环境稳定） |
| Negative feedback | 负反馈（偏离→反向纠正→恢复） |
| Set point | 正常值/设定点 |
| Effector | 效应器（执行纠正） |
| Receptor | 感受器（检测偏离） |

## Practice Question

> Explain how **negative feedback** maintains a constant internal environment. **[4]**

**A\* 答法**：
1. A **receptor** detects a change/deviation from the set point.
2. A **coordinator** (e.g. hypothalamus) processes the signal.
3. An **effector** produces a response that is **opposite** to the original change.
4. The variable returns to the set point, which **switches off** the response.

## Summary 本课小结

✅ 稳态：维持内环境（体温/血糖/渗透压）在最适范围
✅ 三要素：感受器→协调中枢→效应器
✅ 负反馈：偏离→反向响应→恢复正常值→关闭响应
✅ 动态平衡：允许小幅波动，而非绝对恒定
✅ 正反馈少见，是扩大偏差的过程（如分娩）
