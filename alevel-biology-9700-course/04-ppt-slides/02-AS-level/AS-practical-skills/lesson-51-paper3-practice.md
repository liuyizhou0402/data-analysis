## Question Type 1: Following Instructions 按指示操作

**题目特征**：给出步骤，要求记录结果/计算/绘图

**答题重点**：
- 按格式记录（表格、有效数字）
- 图按标准绘制
- 计算写出过程（show working）

---

## Question Type 2: Planning 规划题（高分重点）

**常见结构**：
> "Plan an experiment to investigate the effect of X on Y"

**必须包含（得分点）**：
1. **Hypothesis（假设）**：预期结果方向
2. **IV（独立变量）+ 操作方法**：如何改变
3. **DV（因变量）+ 测量方法**：如何测量
4. **CV（控制变量）+ 每个原因**：至少 2 个
5. **对照组（control）**
6. **重复（repeats）**：减少误差

---

## Planning Template 规划题模板

```
Hypothesis: As [IV] increases, [DV] will [increase/decrease] because...

Method:
- Prepare [solutions/concentrations] of [variable] at [values]
- Measure/record [DV] using [instrument] after [time/conditions]
- Control variables: keep [CV1] constant by [how]; [CV2] by [how]

Repeat each measurement [n] times; calculate mean to reduce random error.

Control: Use [zero/standard] concentration of [IV] as control.
```

---

## Question Type 3: Data Analysis 数据分析题

**步骤**：
1. **描述趋势**（不解释）：用数字 + 正确方向
2. **引用最大/最小值**或关键转折点
3. 计算（速率、百分比变化等）
4. **解释**（联系生物学）

---

## Question Type 4: Evaluation 评价题

**"Evaluate the experimental design"要回答**：
- **Limitation（局限性）**：说出具体限制
- **Improvement（改进）**：针对每个限制提出具体改进

**常见改进方向**：
| 局限 | 改进 |
|---|---|
| 样本量小 | 增加重复次数（>3） |
| 主观判断颜色 | 用比色计量化 |
| 单一物种/来源 | 使用多个来源 |
| 环境温度未控制 | 在恒温水浴中进行 |

---

## Key Terms 必背术语

| Term | 含义 |
|---|---|
| Hypothesis | 假设（可检验的预测） |
| Control experiment | 对照实验 |
| Limitation | 局限性 |
| Improvement | 改进建议 |
| Conclusion | 结论（数据支持的陈述） |

## Practice Question (Planning)

> Plan an experiment to investigate how **light intensity** affects the rate of **transpiration** in a plant shoot. **[6]**

**得分点（示例）**：
1. IV: 改变光强（距光源距离）
2. DV: 气孔计（potometer）中气泡移动距离/时间 = 吸水速率
3. CV: 温度、风速、湿度保持恒定
4. 对照：黑暗条件（0 光强）
5. 重复：每个光强测量3次，取平均
6. 范围：至少5个光强梯度
