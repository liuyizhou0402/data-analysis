# Lesson 5: Prokaryotes & Viruses

## Learning Objectives 学习目标

完成本课后，你能够：

- 描述原核细胞（细菌）的结构特征
- 比较原核细胞与真核细胞的异同
- 描述病毒的结构（非细胞生命体）

---

## Prokaryotic Cell Structure 原核细胞结构

| 结构 | 特点 |
|---|---|
| **Cell wall** | 肽聚糖（peptidoglycan）构成，不同于植物 |
| **Plasma membrane** | 与真核相似 |
| **Circular DNA** | 裸露，不与组蛋白结合，位于**拟核（nucleoid）** |
| **Plasmid** | 额外的小型环状 DNA |
| **70S ribosomes** | 比真核（80S）小 |
| **Capsule** | 黏液层（部分细菌），抗吞噬 |
| **Pili/Flagella** | 菌毛（附着）/ 鞭毛（运动） |

---

## No Membrane-Bound Organelles 无膜结合细胞器

**原核细胞缺乏**：
- ✗ Nucleus（无核被膜）
- ✗ Mitochondria
- ✗ ER（内质网）
- ✗ Golgi apparatus
- ✗ Lysosomes

只有核糖体（但大小为 70S）

---

## Prokaryote vs Eukaryote 原核 vs 真核

| 特征 | Prokaryote | Eukaryote |
|---|---|---|
| 核被膜 | ✗ | ✓ |
| DNA 形态 | 环状，裸露 | 线状，与组蛋白结合 |
| 核糖体 | 70S | 80S |
| 膜结合细胞器 | ✗ | ✓ |
| 细胞壁 | 肽聚糖（多数） | 纤维素（植物）/ 无（动物） |
| 大小 | 1–10 μm | 10–100 μm |
| 质粒 | 通常有 | 通常无 |

---

## Virus Structure 病毒结构

病毒 = **非细胞生命体（non-cellular）**，必须寄生于宿主才能复制

**基本组成**：
- **Capsid（衣壳）**：蛋白质外壳，保护遗传物质
- **Nucleic acid**：DNA 或 RNA（非两者兼有）
- **Envelope（包膜）**：部分病毒有（来自宿主膜，如流感病毒）

**注意**：病毒无核糖体、无代谢酶 → 不能独立复制

---

## HIV Structure (Example) HIV 结构示例

- **Envelope**：双层脂质 + 糖蛋白（gp120 用于识别 T 细胞）
- **Capsid**：蛋白质核心
- **RNA**：两条正链 RNA（逆转录病毒）
- **Reverse transcriptase（逆转录酶）**：RNA → DNA

## Key Terms 必背术语

| Term | 含义 |
|---|---|
| Nucleoid | 拟核（原核 DNA 所在区域） |
| Plasmid | 质粒（额外环状 DNA） |
| Peptidoglycan | 肽聚糖（细菌细胞壁成分） |
| 70S ribosome | 原核核糖体 |
| Capsid | 病毒衣壳（蛋白质外壳） |

## Practice Question

> State **three** ways in which a prokaryotic cell differs from a eukaryotic cell. **[3]**

**A\* 答法**：
1. Prokaryotes have **no nuclear envelope** / no membrane-bound nucleus.
2. Prokaryotes have **70S ribosomes** (eukaryotes: 80S).
3. Prokaryotes have **circular / naked DNA** (no histone proteins associated).
