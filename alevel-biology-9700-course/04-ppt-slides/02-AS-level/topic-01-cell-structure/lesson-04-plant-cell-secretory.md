## Plant-Specific Structures 植物细胞特有结构

| 结构 | 特征 | 功能 |
|---|---|---|
| **Cell wall** | 纤维素（cellulose）构成 | 支撑，防止过度膨胀 |
| **Large central vacuole** | 含细胞液，液泡膜（tonoplast） | 维持膨压（turgor pressure） |
| **Chloroplasts** | 双层膜，类囊体，基质 | 光合作用 |
| **Plasmodesmata** | 穿越细胞壁的胞间连丝 | 相邻细胞间通讯与物质运输 |

---

## Chloroplast Structure 叶绿体结构

**双层膜**：outer membrane + inner membrane

**Thylakoid（类囊体）**：
- 内膜折叠形成 → 堆叠成 **granum（基粒）**
- 光反应场所（光系统 I/II、ATP 合酶）

**Stroma（基质）**：
- 含酶、DNA、70S 核糖体
- 暗反应（Calvin cycle）场所

## Animal vs Plant Cell 动植物细胞比较

| 特征 | Animal cell | Plant cell |
|---|---|---|
| Cell wall | ✗ | ✓ (cellulose) |
| Large vacuole | ✗ (小，临时) | ✓ (永久，大) |
| Chloroplasts | ✗ | ✓ (有叶的细胞) |
| Centrioles | ✓ | ✗ |
| Plasmodesmata | ✗ | ✓ |
| Mitochondria | ✓ | ✓ |
| Nucleus, RER, Golgi | ✓ | ✓ |

## Secretory Pathway 蛋白质分泌途径

蛋白质（如抗体、消化酶、胰岛素）的分泌路径：

```
核糖体（Ribosome）
    ↓ 合成多肽
粗面内质网（RER）
    ↓ 折叠、修饰，装入囊泡
高尔基体（Golgi apparatus）
    ↓ 进一步修饰、分选，打包
囊泡（Secretory vesicle）
    ↓ 移向细胞膜
胞吐（Exocytosis）→ 分泌到细胞外
```

---

## Why Each Step Matters 每步的关键作用

| 步骤 | 细胞器 | 关键事件 |
|---|---|---|
| 1 | Ribosome on RER | 多肽合成，进入 RER 腔 |
| 2 | RER | 蛋白质折叠，二硫键形成 |
| 3 | Golgi | 糖基化（glycosylation），分选 |
| 4 | 分泌囊泡 | 运输至细胞膜 |
| 5 | Exocytosis | 囊泡与膜融合，释放蛋白质 |

## Key Terms 必背术语

| Term | 含义 |
|---|---|
| Tonoplast | 液泡膜 |
| Plasmodesmata | 胞间连丝 |
| Granum/Grana | 基粒（类囊体堆叠） |
| Secretory pathway | 蛋白质→RER→Golgi→胞吐 |
| Exocytosis | 胞吐（囊泡与膜融合释放物质） |

## Practice Question

> Describe the **route taken by a protein** from its synthesis to its secretion from the cell. **[4]**

**A\* 答法（每句 = 1 分）**：
1. Synthesised by **ribosomes on RER**
2. Enters lumen of RER; **folded/modified**
3. Transported in vesicles to **Golgi apparatus**; further modified
4. Packaged into **secretory vesicles** → **exocytosis** at plasma membrane

## Summary 本课小结

✅ 植物特有：cell wall（纤维素）、大液泡、叶绿体、胞间连丝
✅ 叶绿体：双膜 + 类囊体（光反应）+ 基质（Calvin cycle）
✅ 动植物最大区别：cell wall / large vacuole / chloroplasts / centrioles
✅ 分泌途径：Ribosome → RER → Golgi → vesicle → exocytosis
