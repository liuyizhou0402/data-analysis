## Antibody Structure 抗体结构

**Y形蛋白质**：

- **2条重链（heavy chains）+ 2条轻链（light chains）**，由二硫键连接
- **Variable region（可变区）**：两臂末端，与**特定抗原**结合（高度特异性）
- **Constant region（恒定区）**：茎部，与免疫细胞或补体结合

**抗原结合位点（antigen binding sites）**= 2个（每条臂一个）

---

## Antigen & Antibody 抗原与抗体

**Antigen（抗原）**：刺激免疫反应的物质（通常是蛋白质/多糖）
- 每个抗原有特定**表位（epitope）**

**Antibody（抗体）**= **Immunoglobulin（免疫球蛋白）**
- 由 B 细胞（→ 浆细胞）产生
- 可变区形状与特定表位**互补**

---

## Humoral Immunity 体液免疫（步骤）

1. **抗原呈递（Antigen presentation）**：巨噬细胞呈递抗原 → T helper 细胞激活
2. **T helper 细胞激活 B 细胞**：T helper 细胞分泌细胞因子（cytokines）→ 辅助 B 细胞激活
3. **Clonal selection（克隆选择）**：识别特定抗原的 B 细胞被选中
4. **Clonal expansion（克隆扩增）**：选中的 B 细胞**迅速增殖**
5. **分化（Differentiation）**：→ 浆细胞（plasma cells）+ 记忆细胞（memory cells）

---

## Plasma Cells vs Memory Cells 浆细胞 vs 记忆细胞

| 特征 | Plasma cells（浆细胞） | Memory cells（记忆细胞） |
|---|---|---|
| 功能 | 大量**分泌抗体**（每秒数千个） | 长期存活，快速反应 |
| 寿命 | 短（数天） | 长（数年/终生） |
| 再感染时 | — | 快速增殖，大量产抗体 |
| 含 RER | 大量（抗体合成） | 少量 |

---

## Antibody Functions 抗体的功能

| 功能 | 机制 |
|---|---|
| **中和（Neutralisation）** | 结合毒素/病毒，阻止其与靶细胞结合 |
| **凝集（Agglutination）** | 多个抗体交联病原体 → 团块易被吞噬 |
| **调理（Opsonisation）** | 抗体包裹病原体，促进吞噬 |
| **补体激活** | 抗体-抗原复合物激活补体级联 → 溶菌 |

## Key Terms 必背术语

| Term | 含义 |
|---|---|
| Antibody (immunoglobulin) | 抗体（Y型蛋白） |
| Variable region | 可变区（结合特定抗原） |
| Epitope | 表位（抗原上被识别的部位） |
| Clonal expansion | 克隆扩增（B细胞大量增殖） |
| Plasma cell | 浆细胞（大量分泌抗体） |
| Memory B cell | 记忆B细胞（快速再次应答） |

## Practice Question

> Explain the role of **B lymphocytes** in the immune response to infection. **[4]**

**A\* 答法**：
1. B lymphocytes with **complementary surface receptors** bind to specific antigens (with T helper cell assistance).
2. Stimulated B cells undergo **clonal expansion** (rapid division).
3. Differentiate into **plasma cells** → secrete large amounts of **specific antibodies**.
4. Some differentiate into **memory B cells** → long-lived, enable rapid secondary response.

## Summary 本课小结

✅ 抗体：Y形，可变区（特异性）+ 恒定区
✅ 体液免疫：抗原呈递→T helper激活→B细胞克隆扩增→浆细胞+记忆细胞
✅ 浆细胞：大量分泌抗体（短命）
✅ 记忆细胞：长寿，二次免疫时快速大量产抗体
✅ 抗体功能：中和、凝集、调理、补体激活
