# Lesson 8: Carbohydrates II — Polysaccharides
## 多糖：淀粉、糖原与纤维素
Cambridge CIE 9700 | A-Level Biology | Topic 2

---

## Learning Objectives 学习目标

完成本课后，你能够：

- 描述淀粉（starch）、糖原（glycogen）、纤维素（cellulose）的结构
- 解释结构如何决定功能（storage vs structural）
- 比较三种多糖的异同

---

## Starch 淀粉

**组成**：α-glucose → α-1,4 糖苷键（直链）+ α-1,6 分支

**两种成分**：
- **Amylose（直链淀粉）**：无分支，螺旋结构
- **Amylopectin（支链淀粉）**：有 α-1,6 分支

**功能**：植物**储能**多糖（叶绿体/淀粉粒中）

**为什么适合储能**：
- 不溶于水 → 不影响渗透压
- 紧密螺旋 → 节省空间
- 可水解 → 快速释放葡萄糖

---

## Glycogen 糖原

**组成**：α-glucose → α-1,4 糖苷键 + **更多 α-1,6 分支**

**功能**：动物（肝脏/肌肉）与真菌的**储能**多糖

**与淀粉比较**：
- 分支更多 → 更紧凑
- 末端更多 → 更快速水解释放葡萄糖
- 同样不溶于水

---

## Cellulose 纤维素

**组成**：β-glucose → β-1,4 糖苷键

**关键**：相邻 β-glucose 翻转 180° → 链为直线
→ 多条链通过**氢键**平行排列 → **微纤维（microfibril）**

**功能**：植物细胞壁的**结构**多糖

**为什么适合结构**：
- 氢键交联 → 高强度、不可拉伸
- 不溶于水
- 人类无法消化（无 β-1,4 水解酶）

---

## Comparison Table 三种多糖对比

| 特征 | Starch | Glycogen | Cellulose |
|---|---|---|---|
| 单体 | α-glucose | α-glucose | β-glucose |
| 糖苷键 | α-1,4 (+α-1,6 分支) | α-1,4 (+更多分支) | β-1,4 |
| 分支 | 少 | 多 | 无（氢键横向连接） |
| 功能 | 植物储能 | 动物/真菌储能 | 植物结构 |
| 溶解性 | 不溶 | 不溶 | 不溶 |

---

## Iodine Test 碘液检测淀粉

**检测淀粉**（非糖）：

1. 加入 **iodine in potassium iodide solution（碘液）**
2. **阳性**：蓝黑色（blue-black）
3. **阴性**：棕黄色（yellow-brown，原色）

原理：I₂ 进入淀粉螺旋结构 → 形成蓝黑色复合物

## Key Terms 必背术语

| Term | 含义 |
|---|---|
| α-1,4 glycosidic bond | 淀粉/糖原直链键 |
| α-1,6 glycosidic bond | 淀粉/糖原分支键 |
| β-1,4 glycosidic bond | 纤维素键 |
| Microfibril | 微纤维（纤维素平行链束） |
| Amylose / Amylopectin | 直链 / 支链淀粉 |

## Practice Question

> Compare the structure of **starch** and **cellulose** and explain how their structures relate to their functions. **[4]**

**A\* 答法**：
1. Starch contains **α-glucose**; cellulose contains **β-glucose**.
2. Starch has **α-1,4 (and α-1,6) bonds**; cellulose has **β-1,4 bonds**.
3. Cellulose chains are **straight** → parallel → **hydrogen bonds** → microfibrils → **structural strength**.
4. Starch **coiled/branched** → **compact** storage; **insoluble** → no osmotic effect.

## Summary 本课小结

✅ 淀粉：α-glucose + α-1,4/α-1,6 → 螺旋/分支 → 植物储能
✅ 糖原：分支更多 → 更快水解 → 动物储能
✅ 纤维素：β-glucose + β-1,4 → 直链 → 氢键 → 微纤维 → 结构
✅ 结构—功能关系是考试核心：同是葡萄糖多聚体，功能完全不同
✅ 碘液检测淀粉 → 蓝黑色
