## What Is a Gene Mutation? 基因突变

**定义**：DNA 碱基序列的**永久改变（permanent change）**

**发生时间**：S期 DNA 复制时
**类型（点突变 point mutation）**：
- **Substitution（替换）**：一个碱基替换为另一个
- **Insertion（插入）**：额外碱基插入
- **Deletion（缺失）**：碱基丢失

---

## Substitution 替换突变

一个碱基对被另一个碱基对替换

**对蛋白质的影响**：
| 结果类型 | 说明 |
|---|---|
| **Silent（沉默）** | 密码子变化但编码相同氨基酸（遗传密码简并性） |
| **Missense（错义）** | 密码子变化 → 不同氨基酸 → 蛋白质功能改变 |
| **Nonsense（无义）** | 密码子变化 → 终止密码子 → 多肽链提前终止 |

---

## Sickle-Cell Anaemia 镰刀型细胞贫血症（例子）

**原因**：β-血红蛋白基因第6位密码子
- 正常：GAG → **谷氨酸（Glu）**
- 突变：GTG → **缬氨酸（Val）**（替换突变）

**后果**：
- HbS 代替 HbA → 低氧时聚集 → 红细胞变镰刀形
- 镰刀形红细胞：阻塞毛细血管，携氧能力下降

---

## Insertion & Deletion 插入与缺失突变

**Frameshift mutation（移码突变）**：

- 插入或缺失**1个或2个**碱基 → 后续所有密码子**全部错位**
- 影响远大于替换突变 → 通常产生无功能蛋白

**插入3个碱基**：
- 仅增加一个氨基酸，后续读码框不变（影响较小）

## Mutagenic Agents 诱变剂

| 类型 | 例子 |
|---|---|
| 物理 | UV（紫外线，引起嘧啶二聚体），X射线 |
| 化学 | 亚硝酸（碱基类似物），苯并芘（香烟） |
| 生物 | 某些病毒（整合 DNA，引起突变） |

## Key Terms 必背术语

| Term | 含义 |
|---|---|
| Point mutation | 点突变（单碱基改变） |
| Substitution | 碱基替换 |
| Insertion / Deletion | 插入 / 缺失 |
| Frameshift | 移码突变 |
| Silent mutation | 沉默突变（无氨基酸改变） |
| Mutagen | 诱变剂 |

## Practice Question

> Explain why an **insertion mutation** usually has a greater effect on the protein produced than a **substitution mutation**. **[3]**

**A\* 答法**：
1. Insertion causes a **frameshift** — the reading frame shifts for all codons after the insertion point.
2. This changes **all subsequent amino acids** in the polypeptide.
3. Substitution only changes **one amino acid** / may be silent due to degeneracy of genetic code.

## Topic 6 Summary 主题综合小结

✅ 核苷酸：五碳糖 + 磷酸 + 碱基（DNA:T；RNA:U）
✅ DNA 双螺旋：反平行，A-T(2H)，G-C(3H)
✅ 复制：半保留，解旋酶解链，DNA 聚合酶合成
✅ 转录：RNA 聚合酶→mRNA（5'→3'）；真核细胞剪接
✅ 翻译：核糖体 + tRNA + mRNA → 多肽
✅ 突变：替换（单个氨基酸改变/沉默）；插入/缺失（移码，影响大）
