## Command Words 指令词（必背）

| 指令词 | 要求 |
|---|---|
| **Define** | 给出精确定义（通常 1–2 分，不需要例子） |
| **State** | 简短陈述，无需解释 |
| **Describe** | 说出特征/过程（不解释原因） |
| **Explain** | 说出原因/机制（why/how） |
| **Compare** | 说出异同（两者都要提，用 whereas） |
| **Suggest** | 基于知识推断（题目可能无标准答案） |
| **Calculate** | 写出过程 + 结果 + 单位 |
| **Outline** | 简要描述主要步骤 |

---

## Scoring by Mark Points 踩分点意识

**每个 ✓ = 1 分**

**原则**：
- 4 分题 = 4 个独立得分点
- 不能用同一个知识点重复得分
- 答"因为…所以…"的因果链结构

**错误示范**（2分题）：
- ❌ "The enzyme is denatured" → 只有 1 个点，不完整

**正确示范**：
- ✅ "The enzyme is denatured" ← 1分
- ✅ "The active site changes shape so the substrate cannot bind" ← 1分

---

## The "Explain" Formula Explain 题公式

**高温使酶失活（例）**：

```
高温 → 断裂氢键/离子键 → 活性位点形状改变 →
底物不能结合（不互补）→ 不形成 E-S 复合物 →
反应速率降低
```

**每个箭头"→"= 1 个得分点**

## Compare Template Compare 题模板

**必须用 "whereas" 对比两者**：

> "Arteries have thick muscular walls, **whereas** veins have thin muscular walls."
> "DNA contains thymine, **whereas** RNA contains uracil."

⚠️ 只写一方 → 通常丢一半分

---

## Data Question 3-Step Framework 数据题三步法

**对于图表/数据分析题**：

**步骤 1 — Describe（描述趋势）**
> "As X increases, Y increases..."

**步骤 2 — Quote（引用数值）**
> "...from A to B (at X = __ to X = __)..."

**步骤 3 — Explain（联系生物学）**
> "This is because..."

---

## Common Mistakes 常见失分模式

| 错误 | 改正 |
|---|---|
| Describe 时只说方向没有数值 | 引用具体数值 |
| Explain 时只给结论没有机制 | 写出因果链 |
| Compare 只写一方 | 两方都写，用 whereas |
| Calculate 没写单位 | 每个计算结果都加单位 |
| Define 只是重复题目词 | 写出核心概念（不能用同词定义自身） |

## Practice: Command Word Recognition

判断以下问题需要用哪种答法：

1. "**State** two functions of the Golgi apparatus" → **State：简短列出，不解释**
2. "**Explain** why the oxygen dissociation curve is S-shaped" → **Explain：说出协同效应的机制**
3. "**Compare** the structure of DNA and RNA" → **Compare：两者都写，用 whereas**
4. "**Calculate** the magnification" → **Calculate：写过程+单位**
